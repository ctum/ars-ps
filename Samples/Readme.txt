The "ADSI Provider" folder contains examples of VBScript and PowerShell scripts demonstrating how to
manage directory objects and computer resources by using the Active Roles ADSI provider. 

The "ARServer Snap-in" folder contains examples of the C++ and .NET projects demonstrating how to
create property page extensions and menu extensions in the Active Roles console.

The "RestrictGroupScope" folder contains a policy script allowing you to configure a policy that
would prevent Active Roles from creating universal groups in certain folders.

The "WI" folder contains a C# code sample for customizing the Active Roles Web Interface.
