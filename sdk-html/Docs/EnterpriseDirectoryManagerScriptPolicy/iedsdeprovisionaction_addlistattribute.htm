IEDSDeprovisionAction::AddListAttribute

The method AddListAttribute adds a new detailed record to the Deprovisioning Results report or the Results of Undo Deprovisioning report for the processed user. The record describes changes made to the specified attribute of the user. The record contains the following fields: attribute name, attribute old value, attribute new value.

Sub  AddListAttribute(byVal AttrName, byVal oldValue, byVal newValue)

Arguments

AttrName
[in] Required. String. Contains ldapDisplayName of the processed user's attribute to be changed by the deprovision/un-deprovision operation. 
oldValue
[in] Required. Contains an old value of the user's attribute.
newValue
[in] Required. Contains a new value of the user's attribute.
