Specifying Parameters Using ADSI Scripts

To add/remove parameters for your policy scripts, you can use the CreatePolicySetting or RemovePolicySettingbyName method of the IEDMPolicyEntry interface, respectively. To modify values of existing parameters, use the Item property of the IEDMPolicyEntry interface.

Adding Parameters

The following code snippet illustrates how to add the parameter Par1 to be used by the script policy PolicyScript1 defined for the Policy Object MyPO, and set that parameter to 100. 

[VBScript]
Const strPOName = "MyPO"
Const strAPEName = "PolicyScript1"
Const strParamName = "Par1"
Const strParamValue = "100"
On Error Resume Next
Set objPO = GetObject("EDMS://CN=" & strPOName & ",CN=Administration,CN=Policies,CN=Configuration")
Flag = 0
For Each APE In objPO
   IF (inStrRev(APE.Name, strAPEName)) Then
       Set APS1=APE.CreatePolicySetting
       APS1.Name = strParamName
       APS1.Value= strParamValue
       APE.AddPolicySetting APS1
       objPO.SetInfo
       Flag = 1
       Exit For
   END IF
Next
If (Flag = 1) Then
    If (err.number = 0) Then
          MsgBox "The parameter " & strParamName & " has been added.",vbOK + vbInformation
                         else
          MsgBox "Failed to add the parameter "  & strParamName & ". Error code: " & CStr(Err.number),vbOK + vbCritical
    End If 
             Else
          MsgBox "Cannot find the policy " & strAPEName & " in the Policy Object " & strPOName,vbOK + vbCritical
End If




[PowerShell]
$strAPOName = "Test APO"
$strAPEName = "Test APE"
$strParamName = "customParam"
$strParamValue = "100"
$objAPO = [adsi]"EDMS://CN=$strAPOName,CN=Administration,CN=Policies,CN=Configuration"
$flag = 0
try {
    foreach($objAPE in $objAPO._NewEnum) {
        if ($objAPE.Name -eq $strAPEName) {
            $objAPESetting = $objAPE.CreatePolicySetting()
            $objAPESetting.Name = $strParamName
            $objAPESetting.Value = $strParamValue
            $objAPE.AddPolicySetting($objAPESetting)
            $objAPO.SetInfo()
            $flag = 1
            
            break
        }
    }
    if ($flag -eq 1) { 
        "The parameter '$strParamName' has been added." 
    } else { 
        "Cannot find the policy '$strAPEName' in the Policy Object '$strAPOName'." 
    }
}
catch {
    "Failed to add the parameter '$strParamName'`nError: $error"
}

Removing Parameters

The following code snippet illustrates how to remove the parameter Par1 defined for the script policy PolicyScript1.

[VBScript]
Const strPOName = "MyPO"
Const strAPEName = "PolicyScript1"
Const strParamName = "Par1"
On Error Resume Next
Set objPO = GetObject("EDMS://CN=" & strPOName & ",CN=Administration,CN=Policies,CN=Configuration")
Flag = 0
For Each APE In objPO
   IF (inStrRev(APE.Name, strAPEName)) Then
       APE.RemovePolicySettingByName strParamName
       objPO.SetInfo
       Flag = 1
       Exit For
   END IF
Next
If (Flag = 1) Then
    If (err.number = 0) Then
           MsgBox "The parameter " & strParamName & " has been removed.",vbOK + vbInformation
                   else
           MsgBox "Failed to remove the parameter "  & strParamName & ". Error code: " & CStr(Err.number),vbOK + vbCritical
    End If
              Else
	                MsgBox "Cannot find the policy " & strAPEName & " in the Policy Object " & strPOName,vbOK + vbCritical
End If




[PowerShell]
$strAPOName = "Test APO"
$strAPEName = "Test APE"
$strParamName = "customParam"
$objAPO = [adsi]"EDMS://CN=$strAPOName,CN=Administration,CN=Policies,CN=Configuration"
$flag = 0
try {
    foreach($objAPE in $objAPO._NewEnum) {
        if ($objAPE.Name -eq $strAPEName) {
            $objAPE.RemovePolicySettingByName($strParamName)
            $objAPO.SetInfo()
            $flag = 1
            break
        }
    }
    if ($flag -eq 1) { 
        "The parameter '$strParamName' has been removed." 
    } else { 
        "Cannot find the policy '$strAPEName' in the Policy Object '$strAPOName'." 
    }
}
catch {
    "Failed to remove the parameter '$strParamName'`nError: $error"
}
