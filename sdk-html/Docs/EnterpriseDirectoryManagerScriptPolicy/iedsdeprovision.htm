IEDSDeprovision

The IEDSDeprovision interface enables policy scripts to add records to the Deprovisioning Results or Results of Undo Deprovisioning report. The interface also allows the policy script to retrieve values of attributes the processed user had before deprovisioning.

Note:

The Deprovisioning Results report or the Results of Undo Deprovisioning report for a user is displayed when you select the user in the details pane of the Active Roles snap-in, and then click Deprovisioning Results or Results of Undo Deprovisioning, respectively, on the shortcut menu.

Properties

The IEDSDeprovision interface has the following properties.

Property

Description

OriginalUserAccount

Returns a reference to a user-type object that contains values the processed user had before the deprovision operation.

Note This property can be used only within the onPreDeprovision and  onDeprovision event handlers.

DeprovisionedProperties

Returns a list of all properties of the user account that were changed in accord with the standard Deprovisioning policies when the user account was deprovisioned.

Note This property can be used only within the onPreUnDeprovision and  onUnDeprovision event handlers.

Methods

The IEDSDeprovision interface defines the following methods.

Method

Description

AddDeprovisioningReportRecord

Allows you to add records to the Deprovision Results report.

Note This method can be used only within the following event handlers: onPreDeprovision, onDeprovision, onPreUnDeprovision, onUnDeprovision.

AddUnDeprovisioningReportRecord

Allows you to add records to the Results of Undo Deprovisioning report.

Note This method can be used only within the following event handlers: onPreUnDeprovision, onUnDeprovision, onPreUnDeprovision, onUnDeprovision.

Example Code

The following sample policy script illustrates the use of the IEDSDeprovision interface.

[VBScript]
Sub onPreDeprovision(Request)
Dim cityValue
cityValue = CStr(Request.OriginalUserAccount.Get("l"))
EventLog.ReportEvent 3, "Pre deprovisioning info:", "This user is from '" + cityValue + "' city."

End Sub

[PowerShell]
function onPreDeprovision($Request)
{
$cityValue = [string]$Request.OriginalUserAccount.Get("l")
$EventLog.ReportEvent(3, "Pre deprovisioning info:", ("This user is from '" + $cityValue + "' city."))
}
 

This policy script adds a record to the EDM Server event log. The record contains the value of the "l" attribute the user had before the deprovision operation.
 
   

To use this policy script

  1. Prepare the corresponding script module and create a deprovisioning Policy Object that uses this script module. For related procedures, refer to Applying Policy Scripts.

  2. Link the created deprovisioning Policy Object to a container, such as an Organizational Unit.

  3. In the console tree, click that container, and then in the details pane, select any user that resides in the container.

  4. On the Action menu, click Deprovision.
