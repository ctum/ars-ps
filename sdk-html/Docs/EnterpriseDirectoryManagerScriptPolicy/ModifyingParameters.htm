Modifying Parameters for Policy Scripts

The following code snippet illustrates how to modify the value of the "MyParameter" parameter defined for the script policy "MyScriptPolicy" in the "MyPO" Policy Object. The script sets this parameter to "300."

Note:

You can use this sample code to set or modify a value of any string-type parameter regardless of a method of its definition. For information about methods used to define the policy script parameters, refer to "Specifying Parameters with ADSI Scripts" and "Declaring Parameters in Policy Scripts" in Related Topics. 

[VBScript]
Const strPOName = "MyPO"
Const strAPEName = "MyScriptPolicy"
Const strParamName = "MyParameter"
Const strParamNewValue = "300"
On Error Resume Next
Set objPO = GetObject("EDMS://CN=" & strPOName & ",CN=Administration,CN=Policies,CN=Configuration")
Flag = 0
For Each APE In objPO
   IF (inStrRev(APE.Name, strAPEName)) Then
       Set objParam = APE.Item(strParamName)
       objParam.Value = strParamNewValue
       objPO.SetInfo
       Flag = 1
       Exit For
   END IF
Next
If (Flag = 1) Then
    If (err.number = 0) Then
           MsgBox "The parameter " & strParamName & " has been modified.",vbOK + vbInformation
                   else
           MsgBox "Failed to modify the parameter "  & strParamName & ". Error code: " & CStr(Err.number),vbOK + vbCritical
    End If
              Else
	     MsgBox "Cannot find the policy " & strAPEName & " in the Policy Object " & strPOName,vbOK + vbCritical
End If




[PowerShell]
$strAPOName = "Test APO"
$strAPEName = "Test APE"
$strParamName = "customParam"
$strParamValue = "2011"
$objAPO = [adsi]"EDMS://CN=$strAPOName,CN=Administration,CN=Policies,CN=Configuration"
$flag = 0
try {
    foreach($objAPE in $objAPO._NewEnum) {
       if ($objAPE.Name -eq $strAPEName) {
           foreach($objParam in $objAPE) {
               if ($objParam.Name -eq $strParamName) {
                   $objParam.Value = $strParamValue 
                   $objAPO.SetInfo()
                   $flag = 1
                   break
               }
           }
       }
   }
   if ($flag -eq 1) { 
       "The parameter '$strParamName' has been modified." 
   } else { 
       "Cannot find the policy '$strAPEName' in the Policy Object '$strAPOName'." 
   }
}
catch {
   "Failed to modify the parameter '$strParamName'`nError: $error"
}

Note:

The script policy name (MyScriptPolicy) is displayed on the Policies tab of the Properties dialog box for the "MyPO" Policy Object.
