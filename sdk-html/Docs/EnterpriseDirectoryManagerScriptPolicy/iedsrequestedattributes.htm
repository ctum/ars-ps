IEDSRequestedAttributes

The IEDSRequestedAttributes interface enables policy scripts to control a list of the processed object's properties requested to be retrieved. Active Roles clients send a request to retrieve the object's properties in the following cases:

  • When using the Get, GetEx, GetInfo, GetInfoEx methods of the IADs interface

  • When searching for directory objects (retrieving the specified properties of the objects found) 

Note:

All methods and properties of IEDSRequestedAttributes can be used only within the onPreGet, onPostGet, and onPreSearch event handlers.

Properties

The IEDSRequestedAttributes interface has the following properties.

Property

Description

RequestedAttributes

Returns an array of the LDAP display names of the processed object's properties requested to be retrieved.

Methods

The IEDSRequestedAttributes interface defines the following methods.

Method

Description

AddRequestedAttribute

Adds the specified property of a processed object to a list of properties requested to be retrieved.

RemoveRequestedAttribute

Removes the specified property of a processed object from a list of properties requested to be retrieved.

IsAttributeRequested

Indicates whether the specified property of a processed object is in a list of properties requested to be retrieved.

Example Code

The following code snippet illustrates the use of the IEDSRequestedAttributes interface. In this example, the onPreGet event handler raises an error when an Active Roles client tries to retrieve the user's phone number. 

[VBScript]
Sub onPreGet(Request)
 Dim PropertyList, Prop
 If Request.Class <> "user" Then Exit Sub
 ' Get a list of the requested properties of User
 PropertyList=Request.RequestedAttributes
 '
 ' Your code using PropertyList goes here
 '
 If Request.IsAttributeRequested("telephoneNumber") Then 
    Err.Raise 1, "Administrative Policy violation",_
                      "You cannot retrieve the user's phone number"
 End If
End Sub

[PowerShell]
function onPreGet($Request)
{
    if($Request.Class -ne "user"){return}
    # PropertyList=Request.RequestedAttributes
    #
    # Your code using PropertyList goes here
    #
    if($Request.IsAttributeRequested("telephoneNumber"))
    {
        throw ("Administrative Policy violation" + [System.Environment]::NewLine + "You cannot retrieve the user's phone number")
    }
}

Note:

In the onPreGet and onPreSearch event handlers, the Active Roles Administration Service directs error messages to EDM Server Log.
