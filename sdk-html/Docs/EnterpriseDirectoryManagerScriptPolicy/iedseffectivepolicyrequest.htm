IEDSEffectivePolicyRequest

The IEDSEffectivePolicyRequest interface enables policy scripts to retrieve and modify information about the policy constraints and definitions for the directory object whose processing has been requested.

Note:

The methods of this interface can be used only within the onGetEffectivePolicy event handler.

Methods

The IEDSRequestedAttributes interface defines the following methods.

Method

Description

SetEffectivePolicyInfo

Allows you to set and modify the policy constraints defined in the object Request for the specified attribute of the processed object.

GetEffectivePolicyInfo

Returns information about the policy constraints defined in the object Request for the specified attribute of the processed object.

ClearEffectivePolicyInfo

Allows you to clear the policy constraints defined in the object Request for the specified attribute of the processed object.
