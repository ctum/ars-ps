Developing Custom Entitlement Specifier

ActiverRoles Server allows you to extend the set of out-of-the-box entitlement specifiers by creating your own entitlement specifiers using the ActiverRoles Server UI. Also, you can create custom entitlement specifiers using PowerShell script.

The following sample creates a container named MyEntitlements in Active Roles container for entitlement profile specifiers, and then creates entitlement profile specifier MySpecifier with the following settings:

  • Entitlement profile specifier tyle: Personal resource entitlement

  • Scope: Active Directory

  • Icon file: C:\Temp\ icon_Computer.gif

This entitlement profile specifier extends entitlement profile with an item including the icon and the contents of the edsOneViewSpecifier virtual attribute of the user, if this attribute is set. For more information about creating and using virtual attributes in ActiverRoles Server, see Using Virtual Attributes.

 

[PowerShell]
#This function generates separator value in the GUID format
function GetSeparator()
{
return [System.String]::Format("<{0}>", [System.Guid]::NewGuid().ToString("N"));
}


# The following function returns the string referencing the icon to show in entitlement profile.

function GetIconString($filePath)
{
[System.Drawing.Image] $img = [System.Drawing.Image]::FromFile($filePath);
[System.IO.MemoryStream] $ms = New-Object System.IO.MemoryStream;
  $img.Save($ms, $img.RawFormat);
[System.string] $result;
$result = [System.string]::Format("data:image/{0};base64,{1}", [System.String](GetImageFormat($img)), [System.Convert]::ToBase64String($ms.GetBuffer()));
$ms.Dispose();
$ms = $null;  $img.Dispose();
$img = $null;
return $result;
}


# The following function returns image format extension for the icon.
function GetImageFormat([System.Drawing.Image]$img)
{
if ($img -eq $null)
{    return [System.Drawing.Imaging.ImageFormat]::Jpeg.ToString();  }
if ($img.RawFormat.Guid -eq [System.Drawing.Imaging.ImageFormat]::Tiff.Guid)  {
  return [System.Drawing.Imaging.ImageFormat]::Tiff.ToString();
}
if ($img.RawFormat.Guid -eq [System.Drawing.Imaging.ImageFormat]::Gif.Guid)
{
  return [System.Drawing.Imaging.ImageFormat]::Gif.ToString();
}

 if ($img.RawFormat.Guid -eq [System.Drawing.Imaging.ImageFormat]::Png.Guid)

 {

   return [System.Drawing.Imaging.ImageFormat]::Png.ToString();

 }

 return [System.Drawing.Imaging.ImageFormat]::Jpeg.ToString();

}

[System.String]$arsHost = "localhost";

[System.String]$userName = "";

[System.String]$userPwd = "";

[System.String]$containerName = "MyEntitlements";

[System.String]$specifierName = "MySpecifier";

[System.String]$resourceDisplayName = "Is Service account";

[System.String]$iconPath = "C:\Temp\icon_Computer.gif";

$root = [adsi]"EDMS:";

$Cont = $root.OpenDSObject("EDMS://$arsHost/CN=Entitlement Profile Specifiers,CN=Server Configuration,CN=Configuration", $userName, $userPwd, 32768);

$Cont = $Cont.Create("edsOneViewSpecifiersContainer", "CN=$containerName");

$Cont.SetInfo();

$isServiceAttr = "edsvaIsServiceAccount";

$mySpec = $Cont.Create("edsOneViewSpecifier", "CN=$specifierName");

$mySpec.Put("edsaEntitlementType", 1);# 1 - Self, 2 - Membership, 3 - ManagedBy

$scopeGuid = "80a8172f-2d21-4035-b5d0-2675f24b66a1";#guid of 'CN=Active Directory' object

$mySpec.Put("edsaEntitlementRule", "[0x1;$scopeGuid;(&(&(&(objectCategory=person)(objectSid=*)(!(sAMAccountType:1.2.840.113556.1.4.804:=3)))(!(userAccountControl:1.2.840.113556.1.4.804:=2048)))(&($isServiceAttr=*)(objectClass=user)))]");

$mySpec.Put("edsaResourceTypeDisplayName", $resourceDisplayName);

$mySpec.Put("edsaResourceNamingAttribute", "name");

$resAttrs = @("name", "description", [System.String](GetSeparator), "edsvaParentCanonicalName");

$mySpec.Put("edsaResourceAttributes", $resAttrs);

$mySpec.Put("edsaResourceTypeIcon", [System.String](GetIconString($iconPath)));

$mySpec.SetInfo();
