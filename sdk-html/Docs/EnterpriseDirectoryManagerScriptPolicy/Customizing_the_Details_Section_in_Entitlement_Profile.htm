Customizing Details Section in Entitlement Profile

To include custom HTML content into the Details section of an entitlement profile you can use the edsaHTMLDetailsAttribute attribute of the edsOneViewSpecifier object. You need to set the edsaHTMLDetailsAttribute attribute with the name of the virtual attribute that includes custom HTML code which will be displayed in the Details section of an entitlement profile.

The following code example of a script module implementing the scenario performs the following steps:

  1. Retrieves the list of the groups to which the user belongs  by enumerating the memberOf attribute of the user object.

  2. Generates the list of the groups as a HTML code snippet.

  3. Populates the edsvaHtmlMemberOf virtual attribute with the HTML code generated in step 2.

Note

Before running this script, you need create the edsvaHtmlMemberOf virtual attribute.

 

[PowerShell]
function onPreGet($Request)
{
# Here goes the event handler code
if ($Request.Class -eq "user")
{
      if ($Request.IsAttributeRequested("edsvaHtmlMemberOf") -eq $true)
      {
          $Request.AddRequestedAttribute("memberOf");
      }
  }
}
function onPostGet($Request)
{
  if ($Request.Class -ne "user")
  {
      return;
  }
  # Here goes the event handler code    
  if ($Request.IsAttributeRequested("edsvaHtmlMemberOf") -eq $true)
  {
      $inData = $Request.GetEx("memberOf")
      $htmlData = BuildHtmlString $inData
      if ($htmlData.Length -gt 0)
      {
          $Request.PutEx(2, "edsvaHtmlMemberOf", $htmlData)
      }
  }
}
function BuildHtmlString($strValues)
{
  if ($strValues -eq $null -or $strValues.Length -eq 0)
  {
      return ""
  }
  [System.Text.StringBuilder] $sb = New-Object System.Text.StringBuilder;
  $sb.Append("<b>Member Of:</b><table border=`"2`">") | Out-Null
  foreach($strValue in $strValues)
  {
      $sb.Append("<tr><td>$strValue</td></tr>") | Out-Null
  }
  $sb.Append("</table>") | Out-Null
  return $sb.ToString()
}
