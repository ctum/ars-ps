IEDSRequestedAttributes::RemoveRequestedAttribute

The RemoveRequestedAttribute method allows you to remove the specified property of a processed object from a list of properties requested to be retrieved. 

Sub  RemoveRequestedAttribute(ByVal AttrName)

Arguments

AttrName

[in] Required. String. LDAP display name of the object's property you want to remove from a list of properties to be retrieved.
