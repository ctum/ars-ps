Task Object

The Task object allows the scheduled task script to access properties of the scheduled task object (the object class edsScheduledTask) that executes that script. The Task object also allows you to get information about the task's schedule, such as the time the schedule task will start next time.

Properties and Methods

The Task object supports the following properties and methods:

  • ADsPath Property

  • DirObj Property

  • NextRunTime Property

  • StartTime Property

  • Terminate Method 

ADsPath Property

Read-only. Returns the ADsPath to the scheduled task object (the object class edsScheduledTask) that executes the script. 

Task.AdsPath

DirObj Property

Read-only. Returns a reference to the scheduled task object (the object class edsScheduledTask) that executes the script. 

Task.DirObj

Remarks

Task.DirObj refers to a scheduled task object, i.e. it implements the IADs interface and all interfaces available for the Active Roles-specific objects. For a list of those interfaces as well as properties and methods the interfaces provide, see Active Roles-specific Interfaces.

NextRunTime Property

Read-only. Returns a Variant of subtype Date indicating the date and time the schedule task will start next time. This property can return an empty value.

Task.NextRunTime

StartTime Property

Read-only. Returns a Variant of subtype Date indicating the date and time the schedule task started.

Note:

If you start the task with the All Tasks>Execute command on the shortcut menu the Active Roles snap-in provides for that scheduled task, the StartTime property returns the date and time of this operation. Otherwise, it returns the date and time the schedule task must start according to the schedule.

Task.StartTime

Remarks

To clarify the format of data the NextRunTime and StartTime properties return, consider the following example.

If you insert in your scheduled task script the following line:

[VBScript]
EventLog.ReportEvent EDS_EVENTLOG_INFORMATION_TYPE, "Start Time: " & CStr(Task.StartTime),_
            "Next Run Time: " & CStr(Task.NextRunTime)




[PowerShell]
$EDS_EVENTLOG_INFORMATION_TYPE = 4
$StartTime = $Task.StartTime
$NextRunTime = $Task.NextRunTime
$EventLog.ReportEvent($EDS_EVENTLOG_INFORMATION_TYPE, "Start Time: $StartTime `nNext Run Time: $NextRunTime")  

after the scheduled task executes, entries similar to the following example will be added to the EDM Server Log:

Start Time: 9/24/2009 1:47:00 PM
Next Run Time: 9/28/2009 1:47:00 PM

Terminate Method

Terminates the execution of the scheduled task.

Task.Terminate

Remarks

This method does not influence the task's schedule. It only terminates the execution of task when you call this method in your scheduled task script.
