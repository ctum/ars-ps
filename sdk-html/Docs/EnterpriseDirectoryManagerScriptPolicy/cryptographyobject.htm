Cryptography Object

The Cryptography object allows you to encrypt or decrypt strings or arrays of bytes. Use this object when you want to protect any information from unauthorized viewing or use.

Before using the Cryptography object in your scripts, obtain the reference to this object using the Cryptography property of the object Security as shown the following code snippet:

Set objCrypt = Security.Cryptography

Methods

The Cryptography  object supports the following methods:

  • Encrypt Method

  • Decrypt Method

  • EncryptToString Method

  • DecryptFromString Method

Encrypt Method

The Function procedure that encrypts the specified array of bytes and returns an array of the encrypted bytes. 

Object.Encrypt(ByVal arrByte)

Arguments

Object
Required. Always the Security.Cryptography object.
arrByte
[in] Required. Contains the array of bytes to encrypt.

Decrypt Method

The Function procedure that decrypts the array of bytes encrypted with the Encrypt method. Returns the decrypted array of bytes. 

Object.Decrypt(ByVal EncryptedBytes)

Arguments

Object
Required. Always the Security.Cryptography object.
EncryptedBytes
[in] Required. Contains the encrypted array of bytes you want to decrypt.

EncryptToString Method

The Function procedure that encrypts the specified string and returns the encrypted string. 

Object.EncryptToString(ByVal Str)

Arguments

Object
Required. Always the Security.Cryptography object.
Str
[in] Required. String. Contains the string you want to encrypt.

DecryptFromString Method

The Function procedure that decrypts the string encrypted with the EncryptToString method. Returns the decrypted string. 

Object.DecryptFromString(ByVal EncryptedStr)

Arguments

Object
Required. Always the Security.Cryptography object.
EncryptedStr
[in] Required. Contains the encrypted string you want to decrypt.
