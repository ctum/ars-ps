Specifying Parameters for Scheduled Tasks

Parameters for scheduled task scripts are stored into the edsaParameters property of the scheduled task object (the object class edsScheduledTask).

You can add, remove or modify parameters for schedules tasks in one of the following ways:

  • With the Active Roles MMC snap-in. For more information, see "Specifying Parameters with MMC Snap-in" in Related Topics

  • With your custom scripts (see the sample code below)

To add, remove or modify parameters used by scheduled task scripts, use the property Value and the method Remove provided by the IEDMParameters interface.

The following code snippet illustrates how to add (modify) the parameter Par1 for the scheduled task MyTask and set that parameter to 100. 

Set objSchT = GetObject("EDMS://CN=MyTask,CN=Scheduled Tasks,CN=Server Configuration,cn=Configuration")
objSchT.Parameters("Par1") = "100"
objSchT.SetInfo

The following code snippet illustrates how to remove the parameter Par1 from the scheduled task object MyTask.

Set objSchT = GetObject("EDMS://CN=MyTask,CN=Scheduled Tasks,CN=Server Configuration,cn=Configuration")
objSchT.Parameters.Remove "Par1"
objSchT.SetInfo

Since the property Value is declared as the default property for the IEDMParameters interface, you can make your source code more compact by omitting the property name Value, as shown in the code snippet above.

For example, the following code fragments use both standard and default property syntax:  

' Standard syntax
objSchT.Parameters.Value("Par1") = "100"


' Default property syntax
objSchT.Parameters("Par1") = "100"
