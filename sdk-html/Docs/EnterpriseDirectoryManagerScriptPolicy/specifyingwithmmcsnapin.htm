Specifying Parameters with MMC Snap-in

This topic explains how to use the Active Roles MMC snap-in to add, modify and remove parameters for scheduled task scripts.

To add, modify or remove a parameter

  1. Choose the Advanced mode for Active Roles snap-in display:

    • On the View menu, click Mode, and then in the View Mode dialog box, select the Advanced Mode option and click OK.

  2. In the console tree, expand the Configuration node, expand the Server Configuration node, click Scheduled Tasks, and in the details pane, select the scheduled task in question.

  3. On the Action menu, click Properties. and then go to the Parameters tab in the Properties dialog box.

  4. On the Parameters tab, do one of the following:

    • To add a parameter, click Add, and then in the Add Parameter dialog box, specify the name and value of the parameter.

    • To modify a parameter, select the parameter from the Values to assign to script parameters at execution time list, click Edit, and then use the Edit Parameter dialog box to modify the parameter settings.

    • To remove a parameter, select the parameter from the Values to assign to script parameters at execution time list, and then click Remove.
