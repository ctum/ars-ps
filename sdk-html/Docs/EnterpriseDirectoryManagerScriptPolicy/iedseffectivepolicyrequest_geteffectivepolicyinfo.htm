IEDSEffectivePolicyRequest::GetEffectivePolicyInfo

The GetEffectivePolicyInfo method returns information about the policy constraints defined in the object Request for the specified attribute of the processed object.

Function GetEffectivePolicyInfo(byVal AttrName, byVal Type)

Arguments

AttrName
[in] Required. Ldap display name of the attribute about which you want to get the information.
Type
[in] Required. Identifies the type of the policy definitions for the AttrName attribute. The method returns information about applied the policy constraints and definitions of the specified type, if any.

Settings

The possible values for the Type argument are listed in the EDS_EFFECTIVE_POLICY_ENUM enumeration.

Remarks

If no the policy constraints and definitions of the specified type apply to the AttrName attribute, the method returns Empty.
