IEDsRequestParameters::GetInControl

The GetInControl method returns the value of the specified InControl. 

Function  GetInControl(ControlID)

Arguments

ControlID

[in] Required. String. Contains the control ID.

Settings

For built-in controls, the ControlID argument can take any value from the Constant or Control ID column of this table. For your custom InControl, set this argument to the control name (any unique string).

Remarks

If the control in question is not placed in a request, the call of this method will result in an error. The following code snippet illustrates how you can handle the error using the Err object (VBScript) or the $Error object (PowerShell):

[VBScript]
On Error Resume Next
controlVal = objectRequest.GetInControl("MyControl")
If Err.Number <> 0 Then
    ' Your code that handles this error goes here
End If




[PowerShell]
$errcount = $Error.count
trap {continue;}
$controlValue=$Request.GetInControl("MyControl")
if ($Error.count -ne $errcount)
{
   # --- Your code that handles this error goes here ---
}
