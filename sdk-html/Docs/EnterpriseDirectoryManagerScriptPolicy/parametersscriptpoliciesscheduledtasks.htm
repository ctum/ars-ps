Parameterizing Script Policies and Scheduled Tasks

Active Roles allows you to define custom parameters that enable you to control the execution of policy scripts and scheduled tasks. The use of parameters lets you create scripts based on information not available to you at design time, such as information obtained from your user or calculated at run time.

This section explains how to implement script policies and scheduled tasks that can use parameters. The following subjects are covered:

  • Specifying parameters for policy scripts

  • Specifying parameters for scheduled tasks

  • Using parameters in policy scripts

  • Using parameters in scheduled task scripts
