Skeleton Script

You can use the following skeleton script to develop your own policy scripts.

To use the skeleton script

  1. Copy the script source to a file.

  2. In the file, only retain the event handlers you want to use.

  3. Follow the procedure outlined in Applying Policy Scripts.

  4. When creating a new script module, copy the script code from this file and paste it in the script editor.
