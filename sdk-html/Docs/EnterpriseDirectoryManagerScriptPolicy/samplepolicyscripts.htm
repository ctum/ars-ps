Sample Policy Scripts

Active Roles SDK comes with a number of sample policy scripts. This section describes each sample script and provides additional information to help you create your own policy scripts.

Some samples of policy scripts are located in the folder '\SDK\Samples'. 

To use a sample script

  1. Follow the procedure outlined in Applying Policy Scripts.

  2. When creating a new script module, copy the script code from this file and paste it in the script editor.
