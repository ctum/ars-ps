Using Request Parameters

This topic provides sample code snippets that illustrate how to retrieve values of the server-defined variables using the Parameter property method of the IEDSRequestParametrs interface.

The following example shows how to bind to the destination OU using the LdapServer and TargetContainer server-defined variables, when handling a request to move an object. 

[VBScript]
Sub onPreMove (Request)
 ' Binding to the destination OU
  Dim strADsPath
  strADsPath = "EDMS://" & Request.Parameter("LdapServer")
  strADsPath = strADsPath & "/" & Request.Parameter("TargetContainer")
  Dim ou
  Set ou = GetObject(strADsPath)
 '
 ' Here goes your code that uses the OU object
 '
End Sub




[PowerShell]
function onPreMove($Request)
{
 # Binding to the destination OU
  $ADsPath = $Request.Parameter("LdapServer") + "/" + $Request.Parameter("TargetContainer")
  $ou = Get-QADObject $ADsPath
 #
 # Here goes your code that uses the OU object
 #
}

The following example shows how to bind to the original object when handling a request to copy that object.

[VBScript] 
Sub onPreCreate (Request)
 ' Verify the type of the request
  If CInt(Request.Parameter("Type")) = EDST_REQ_COPY Then
 ' Object copying requested
 ' Binding to the original object
    Dim strADsPath
    strADsPath = "EDMS://" & Request.Parameter("LdapServer")
    strADsPath = strADsPath & "/" & Request.GetInControl(EDS_CONTROL_SOURCE_OBJECT_DN)
    Dim obj
    Set obj = GetObject(strADsPath)
 '
 ' Here goes your code that uses the original object
 '
  End If
End Sub




[PowerShell]
function onPreCreate($Request)
{
	# Verify the type of the request
	If([int]Request.Parameter("Type") -eq $Constants.EDST_REQ_COPY)
	{
		# Object copying requested
		# Binding to the original object
		$ADsPath = $Request.Parameter("LdapServer") + "/" + $Request.GetInControl($Constants.EDS_CONTROL_SOURCE_OBJECT_DN)
		$obj = Get-QADObject $ADsPath
		#
		# Here goes your code that uses the original object
		#
	}
}
