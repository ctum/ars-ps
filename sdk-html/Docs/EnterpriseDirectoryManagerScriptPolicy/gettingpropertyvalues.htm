Getting In-Process Property Values

The IADs interface of the object Request allows you to get and modify the in-process property values of the directory object being processed (in-process object data). Here the object Request acts as a property bag used to store the property values that are specified by the client or a policy script. Unless a property value is set in the object Request, the value cannot be accessed.

The object Request is created by Administration Service based on the data the service retrieves from the client's request. For example, the client may request to modify certain properties of a directory object. When a client sends a request, it specifies the property values to be set. The object Request contains all those values, allowing your script to access and modify them.

Note:

The object Request can only be used to access the in-process attribute values. If you need to get or modify other attribute values of the object being processed, you can use the system ADSI provider or the DirObj object to access the object instance stored in Active Directory (stored object data).

You can retrieve in-process object data using the Get or GetEx method of the object Request. Each of these methods allows you to retrieve a specific property by name. After you have retrieved a property, you read its value.

When creating the object Request, Administration Service places all the in-process data into that object. You do not need to use the GetInfo method before retrieving object properties.

Since the object Request is ADSI-compliant, the syntax of its Get and GetEx methods conforms to the ADSI specification.

  • Single-value properties can be retrieved using the Get method of the object Request; for information about how to use the Get method of any ADSI-compliant object, please refer to the MSDN Library.

  • Multi-value properties can be retrieved using the GetEx method of the object Request; for information about how to use the GetEx method of any ADSI-compliant object, please refer to the MSDN Library.

Sample Code

The following code snippetы show how to retrieve in-process property values for a user object by using the Get and GetEx methods of the object Request.

[VBScript] 
Sub onPreCreate (Request)
    Dim strLocality, strHomePhone, OtherHomePhones,homeNum
    ' Get the user's locality specified in the request
    strLocality = CStr(Request.Get("l"))
        '
        ' Here goes your code that uses the locality property value.
        '
    ' Get the user telephone numbers specified in the request
    strHomePhone = CStr(Request.Get("homePhone"))
    OtherHomephones = Request.GetEx("otherHomePhone")
    For Each homeNum In OtherHomephones
       '
       ' Here goes your code that uses the retrieved telephone numbers homeNum
       '
    Next
End Sub
