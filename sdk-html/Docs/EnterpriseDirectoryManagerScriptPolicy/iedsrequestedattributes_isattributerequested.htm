IEDSRequestedAttributes::IsAttributeRequested

The IsAttributeRequested method returns TRUE, if the specified property of a processed object is in a list of properties requested to be retrieved. Otherwise, it returns FALSE. 

Function  IsAttributeRequested(ByVal AttrName)

Arguments

AttrName

[in] Required. String. LDAP display name of the object's property.
