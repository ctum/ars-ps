Active Roles Administration Service Intrinsic Objects

Active Roles provides a set of intrinsic (built-in) COM objects that are passed to the appropriate scripting engine to be available in the policy script or scheduled task script namespace. Each intrinsic object has associated methods and properties, which are described in detail in this section. Policy scripts don't need to call the function CreateObject or GetObject to create or get the intrinsic object.

This section discusses the following objects:

  • Request Object

  • Constants Object

  • DirObj Object

  • DomainOverrideCredentials Object

  • EventLog Object

  • Policy Object

  • PolicyEntry Object

  • Security Object

  • Cryptography Object

  • Sessions Object

  • Session Object

  • Task Object
