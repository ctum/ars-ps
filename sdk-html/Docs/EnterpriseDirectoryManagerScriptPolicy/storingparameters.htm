Specifying Parameters for Policy Scripts

You can specify custom parameters for policy scripts in one of the following ways:

  • Declare custom parameters and specify their values using your custom ADSI scripts. These parameters can be only of the String type.

  • Declare custom parameters in policy scripts. These parameters can be of any built-in or custom type.

The information about all parameters defined for all policy entries a Policy Object includes is stored into the edsaAPEListXML property of the Policy Object. This section covers the following subjects:

  • Specifying Parameters with ADSI Scripts

  • Declaring Parameters in Policy Scripts

  • Modifying Policy Script Parameters
