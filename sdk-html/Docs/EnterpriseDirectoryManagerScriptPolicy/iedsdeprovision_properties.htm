IEDSDeprovision Properties

The following table describes properties provided by the IEDSDeprovision interface.

Property

Description

OriginalUserAccount

Access: Read-only

This property can be used only in a request to deprovision a user account.

Returns a reference to a user-type object that contains values of the in-process object attributes the processed user had before the deprovision operation. The returned object implements the IADs interface and other interfaces used for User objects.

DeprovisionedProperties

Access: Read-only

This property can be used only in a request to un-deprovision a deprovisioned user account.

Returns a list of all properties of the user account that were changed in accord with the standard Deprovisioning policies when the user account was deprovisioned. The properties that were changed by a custom, script-based deprovisioning policy (if any) are not included in the list. The returned object is a collection of property entries, with each entry containing the name of a certain property along with the value or values of that property as of the time the deprovisioning of the user account was started.
