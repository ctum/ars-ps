IEDSPolicyComplianceRequest::SetPolicyComplianceInfo

The SetPolicyComplianceInfo method allows you to send a message of the specified type to the Active Roles client. 

Sub SetPolicyComplianceInfo(byVal AttrName,byVal Type,byVal Message,[byVal IsAdjustable])

Arguments

AttrName
[in] Required. A string expression containing an additional description of the message, such as a name of the object's attribute about which you want to report. 
Type
[in] Required. Identifies the type of message the method sends to the Active Roles client.
Message
[in] Required. A string expression containing the message the method sends to the Active Roles client.
IsAdjustable
[in] Optional. Boolean. The default value is FALSE. When TRUE, the Policy Check Results report for objects that violate enforced administrative policies will contain the Set data link on the User Interface. Clicking this link lets you adjust the objects properties in compliance with administrative policies.

Settings

The Type argument can take any value from the EDS_POLICY_COMPLIANCE_ENUMenumeration.

Remarks

The current version of the Active Roles SDK allows you to use the SetPolicyComplianceInfo method only within the onCheckPropertyValues event handler.

When the script calls this method with the Type argument set to EDS_POLICY_COMPLIANCE_ERROR, for example:

[VBScript]
Request.SetPolicyComplianceInfo "Department",EDS_POLICY_COMPLIANCE_ERROR,_
"The value of the department property does not meet the corporate policy."




[PowerShell]
$Request.SetPolicyComplianceInfo("Department", `
  $Constants.EDS_POLICY_COMPLIANCE_ERROR, "The value of the department property does not meet the corporate policy.", $false)

Active Roles displays the dialog box similar to the following screen.

 

Enlarge figure

Note:

Unlike the Raise method of the object Err, the SetPolicyComplianceInfo method does not result in the script termination. You can change the property value that does not conform to the corporate policy.

For more examples illustrating the use of the SetPolicyComplianceInfo interface, refer to Checking Property Values.
