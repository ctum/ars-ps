IEDsRequestParameters::PutOutControl

The PutOutControl method allows you to set the value of the specified OutControl.

Sub PutOutControl(byVal ControlID,byVal ControlSynt,byVal ControlValue)

Arguments

ControlID
[in] Required. String. Contains the control ID..
ControlSynt
[in] Required. Integer. Identifies the data type of the control value. This argument can take any value from the ADSTYPEENUM enumeration (see the MSDN Library).
ControlValue
[in] Required. Contains the control value you want to set. It can be both a single value and an array of values.
 

Settings

For built-in controls, the ControlID argument can take any value from the Constant or Control ID column of this table. For your custom OutControl, set this argument to the control name (any unique string).

 

Sample Code

This sample policy script illustrates how you can use the onPostModify method to move an object when any of its attributes is changed. After the move, the EDS_CONTROL_OBJECT_DN built-in control contains the new valid DN for the object.

 

[VBScript]
Sub onPostModify(Request)
Set objContainerObject=GetObject("EDMS://OU=recycle bin,DC=lab,DC=company,DC=com")
Set objMovedObject=objContainerObject.MoveHere(DirObj.AdsPath, vbNullString)
Request.PutOutControl EDS_CONTROL_OBJECT_DN, EDSTYPE_DN_STRING, objMovedObject.Get ("distinguishedName")
End Sub

[PowerShell]
function onPostModify($Request)
{
$obj = Move-QADObject $Request.Get("distinguishedName") -NewParentContainer "OU=recycle bin,DC=lab,DC=company,DC=com"
$Request.PutOutControl($Constants.EDS_CONTROL_OBJECT_DN, $Constants.EDSTYPE_DN_STRING, $obj.DN )
}

If you apply the Policy Object that includes this script policy to an object, the Administration Service treats the request to move the object as if it were initiated by the client that requested the operation that caused the script to execute. As a result, the requested changes are properly recorded in the management history trail (change history), and are submitted for approval if this is required by the approval rules that are in effect.
