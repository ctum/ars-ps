Security Object

The object Security provides properties that allow you to access the objects Sessions and Cryptography. You can use this object in both the policy scripts and the scheduled task scripts.

Properties

The object Security supports the following properties:

  • Cryptography Property

  • Sessions Property

Cryptography Property

Read-only. Returns a reference to the object Cryptography. 

Security.Cryptography

Sessions Property

Read-only. Returns a reference to the object Sessions. 

Security.Sessions

Example

For examples of the code snippets that illustrate how to use the Security object, refer to Cryptography Object and Sessions Object.
