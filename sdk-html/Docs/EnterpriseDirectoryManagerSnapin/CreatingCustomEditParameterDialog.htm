Customizing Edit Parameter Dialog Box

This topic explains how you can customize the Edit Parameter dialog box designed to specify the policy script parameters of a custom type.

To create a custom Edit Parameter dialog box

  1. Create a class that implements the IARSCustomPicker interface.

  2. In this class, define the ShowPicker method used to display the customized Edit Parameter dialog box.

  3. Using Microsoft Visual Studio, create an assembly (.dll file) that uses this class.

  4. Save the assembly to the folder < Active Roles installation folder>/MMC/Extensions/.

  5. Copy the Interop.ArsAppCntxtMenuCommand.dll and Interop.ArsSnapComponents.dll files to the Extensions folder. These files are available in the folder < Active Roles installation folder>/MMC/.

  6. In your policy script, declare the appropriate parameters using the onInit function. For details, refer to Declaring Parameters in Policy Scripts.

To clarify the above procedure, consider a sample .NET project that creates a DLL assembly used to customize the Edit Parameter dialog box. For details, refer to "Sample" in Related Topics.
