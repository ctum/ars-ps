IEDMExtInit

The IEDMExtInit interface defines the virtual method PutEDMServer. When you call this method,  Active Roles passes to it the name of the Active Roles Administration Service to which the Active Roles snap-in is currently connected. When redefining this method in your C++ code, you can use the Administration Service's name in binding strings. For example, you can use this information to bind to directory objects with the IAdsOpenDSObject interface.

To use the IEDMExtInit interface, it should be defined in your C++ project in the following way: 

MIDL_INTERFACE("249221D6-D8A6-464E-8E78-39147079ABBE")
IEDMExtInit : public IDispatch
{
    public:
        virtual HRESULT STDMETHODCALLTYPE PutEDMServer(
            BSTR bstrServer) = 0;
};
