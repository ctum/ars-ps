Adding Extension Snap-in's Property Page

This sample C++ project demonstrates how to add an extension snap-in's property page to a property sheet displayed by the primary Active Roles snap-in.

All files related to this project are saved in the folder '...\SDK\Samples\ARServer Snap-in\MMC\Property Page Extension'.

Building the Sample

To build the sample project

  1. In Microsoft Visual Studio 2008, open the solution file Ppgeext.sln.

  2. On the Standard toolbar, choose Release from the Solution Configurations list box.

  3. On the Build menu, click Rebuild Solution.

How the Sample Works

The sample creates the property sheet extension snap-in and registers it for the Active Roles snap-in node type "user". After you build the project, the following entry will be automatically added to the registry:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\MMC\NodeTypes\
{D842D417-3A24-48e8-A97B-9A0C7B02FB17}\Extensions\PropertySheet\
{CFCDC9F3-C50E-11D2-952B-00C04FB92EC2}=REG_SZ "Extension to the Rocket Node"

In this registry entry, {D842D417-3A24-48e8-A97B-9A0C7B02FB17} is the Active Roles snap-in node type for the object class "user" while {CFCDC9F3-C50E-11D2-952B-00C04FB92EC2} is CLSID of the property sheet extension snap-in the sample project creates.

After the sample project registers the extension snap-in, the Active Roles snap-in will display a new tab More Settings in the Properties dialog box for the user objects.

 

Enlarge figure

You can easily register this property sheet extension for other object classes. To do this, add the appropriate entry to the registry. For example, if you want to create the new tab More Settings for computers, add the following entry to the registry:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\MMC\NodeTypes\
{D842D405-3A24-48e8-A97B-9A0C7B02FB05}\Extensions\PropertySheet\
{CFCDC9F3-C50E-11D2-952B-00C04FB92EC2}=REG_SZ "New tab for Computers"

In this entry, {D842D405-3A24-48e8-A97B-9A0C7B02FB05} is the node type for the object class "computer" (see the table in Active Roles snap-in Node Types).
