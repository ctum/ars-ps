Registering Node Type

The Active Roles snap-in nodes are automatically registered at the Active Roles installation time. All available nodes are listed in Active Roles Snap-in Node Types.  Note that the Active Roles snap-in node types differ from those for the Active Directory Users and Computers snap-in.  For Active Directory Manager, the GUID for the node type is the GUID stored in the schemaIDGUID property of the classSchema object that represents the type of object you want to extend.

Registering Property Sheet and Context Menu Extension Snap-Ins

In the registry, the following keys must exist for each Active Roles snap-in node type: 

HKEY_LOCAL_MACHINE\Software\Microsoft\MMC\NodeTypes\{nodetype}

For Active Roles, the {nodetype} key is the GUID that depends on the object class you want to extend. You can find the GUID values for various object classes in the table in Active Roles Snap-in Node Types.

To add a new property page

  • Create the following registry entry:

    	
    HKEY_LOCAL_MACHINE\Software\Microsoft\MMC\NodeTypes\{nodetype}\
	Extensions\PropertySheet\
	{snapinCLSID}=REG_SZ "Description of the extension"

To add a new context menu item

  • Create the following registry entry:

    	
    HKEY_LOCAL_MACHINE\Software\Microsoft\MMC\NodeTypes\{nodetype}\
			Extensions\ContextMenu\
			{snapinCLSID}=REG_SZ "Description of the extension"

The {snapinCLSID} key is the CLSID of the MMC extension snap-in with which you want to extend the Active Roles snap-in.

Example

The following registry entry must be created to extend the property sheet for user objects:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\MMC\NodeTypes\
{D842D417-3A24-48e8-A97B-9A0C7B02FB17}\Extensions\
PropertySheet\{CFCDC9F3-C50E-11D2-952B-00C04FB92EC2}=
REG_SZ "New property page for Users"

In this example, {D842D417-3A24-48e8-A97B-9A0C7B02FB17} is the Active Roles snap-in node type for the class user taken from this table, while {CFCDC9F3-C50E-11D2-952B-00C04FB92EC2} is the CLSID of the MMC extension snap-in you use for the user's property sheet extension. The CLSID of the MMC extension snap-in is defined when you install that snap-in on the computer where you want to extend the Active Roles snap-in.
