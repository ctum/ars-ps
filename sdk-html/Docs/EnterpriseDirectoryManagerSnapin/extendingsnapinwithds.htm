Extending Active Roles Snap-in with Display Specifiers

The Active Roles snap-in supports a user interface (UI) that adapts to meet the needs of administrators and end users. Active Roles allows you to extend the UI that represents existing object classes as well as new classes added to the Active Roles consolidated schema.
This section explains how to use display specifiers to extend the UI for viewing and managing directory objects with the Active Roles snap-in.

Specifically, this section discusses the following:

  • Display Specifiers

  • Extending Context Menu

  • Extending Property Sheet

  • Registering Property Page and Context Menu Extensions
