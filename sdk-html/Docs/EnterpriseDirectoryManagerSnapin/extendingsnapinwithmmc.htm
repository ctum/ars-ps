Extending Active Roles Snap-in with the MMC Extension Snap-ins

You can use the Microsoft Management Console extension snap-ins (MMC extension snap-ins) to extend the following user interface features available in the Active Roles snap-in:

  • Scope pane tree view

  • Context menus

  • Property sheets

  • Taskpads 

  • Toolbars

  • Menu buttons

This section provides the steps required to implement the context menu and property sheet extension snap-ins. For information about how to extend other user interface features, refer to the MSDN library.

Note:

You can use any existing MMC extension snap-in for Active Directory Users and Computers to extend the Active Roles snap-in.

Specifically, this section discusses the following:

  • Context Menu and Property Page Extensions

  • Registering Node Type
