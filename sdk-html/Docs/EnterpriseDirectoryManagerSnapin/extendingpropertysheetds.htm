Extending Property Sheet

Active Roles allows you to add pages to the property sheet displayed for directory objects. To add a page to the property sheet, create and install a property sheet extension. A property sheet extension is a COM in-proc server that implements certain interfaces and is registered with Active Directory.

To create and install a context menu extension

  1. Create the property sheet extension DLL. A property sheet extension is a COM in-proc server that, at a minimum, implements the IShellExtInit and IShellPropSheetExt interfaces. In addition, you can implement the IEDMExtInit interface that defines a method allowing you to get the name of Administration Service to which the Active Roles snap-in is currently connected. For more information, see the MSDN library and Related Topics.

  2. Install and register the property sheet extension on the computers where the property sheet extension is to be used. For information about how to register the property sheet extension, refer to Registering COM Objects.
