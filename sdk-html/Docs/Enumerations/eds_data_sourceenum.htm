EDS_DATA_SOURCEENUM

The EDS_DATA_SOURCEENUM enumeration is used to specify the data storage for the object, such as Active Directory, Network resources, etc. The edsaDataSource attribute of any directory object contains one of these values.

Constant

Value

Description

EDS_DATA_SOURCE_UNDEF

0

Data storage is undefined

EDS_DATA_SOURCE_AD

1

Object is owned by an AD service

EDS_DATA_SOURCE_EDS

4

Object is owned by the Active Roles Administration Service (is stored in SQL database)

EDS_DATA_SOURCE_RES

8

Object is a Network resource

Remarks

Because VBScript cannot read data from a type library, VBScript applications do not recognize the symbolic constants as defined above. Use the numerical constants, instead, to set the appropriate flags in your VBScript applications. To use the symbolic constants as a good programming practice, create explicit declarations of such constants, as done here, in your VBScript applications.
