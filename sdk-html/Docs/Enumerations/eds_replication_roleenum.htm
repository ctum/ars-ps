EDS_REPLICATION_ROLEENUM

The EDS_REPLICATION_ROLEENUM enumeration specifies the replication role of the Administration service. The edsaReplicationRole property of the edsReplicationPartner class objects can take any of the following values:

Constant

Value

Description

EDS_REPLICATION_ROLE_UNDEF 

0

The replication role is undefined.

EDS_REPLICATION_ROLE_PUBLISHER

1

Administration service is the Publisher of a replication group.

EDS_REPLICATION_ROLE_SUBSCRIBER

2

Administration service is a Subscriber of a replication group.

EDS_REPLICATION_ROLE_STANDALONE 

3

Administration service is a standalone service.

Remarks

Because VBScript cannot read data from a type library, VBScript applications do not recognize the symbolic constants as defined above. Use the numerical constants, instead, to set the appropriate flags in your VBScript applications. To use the symbolic constants as a good programming practice, create explicit declarations of such constants, as done here, in your VBScript applications.
