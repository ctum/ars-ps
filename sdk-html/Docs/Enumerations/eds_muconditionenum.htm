EDS_MUCONDITIONENUM

The EDS_MUCONDITIONENUM enumeration specifies the type of the membership rule that applies to Managed Units and dynamic groups. The IEDMMembershipRule.Type property  can take any of the following values:

Constant

Value

Description

EDS_MUCONDITION_UNDEF 

0

No membership rules apply.

EDS_MUCONDITION_INCLUDE

1

Build queries to have Managed Unit include all objects that match specified search criteria.

EDS_MUCONDITION_EXCLUDE

2

Build queries to have Managed Unit exclude all objects that match specified search criteria.

EDS_MUCONDITION_INCLUDE_DIRECTLY 

3

Select objects to have Managed Unit statically include them. Active Roles retains the objects in Managed Unit even if they are renamed, moved or have properties changed.

EDS_MUCONDITION_EXCLUDE_DIRECTLY 

4

Select objects to have Managed Unit never include them. Active Roles ensures that the selected objects are excluded from Managed Unit even if they are renamed, moved or have properties changed.

EDS_MUCONDITION_INCLUDE_GROUP_MEMBERS

5

Select groups to have Managed Unit include all the group members.

EDS_MUCONDITION_EXCLUDE_GROUP_MEMBERS

6

Select groups to have Managed Unit never include all the group members.

EDS_MUCONDITION_RETAIN_DEPROVISIONED_USERS

7

This membership rule ensures that the user remains in the membership list. By default, the deprovisioned users are automatically removed from the membership list. You can override this behavior by adding this rule.
