Enumerations

Active Roles defines and uses the following enumerations.

Enumeration

Description

EDS_APE_PARAMENUM

Identifies the Policy Object Entry parameters.

EDS_APE_TYPEENUM

Specifies the Active Roles policy type.

EDS_CHAR_TRANSFORMATION_TYPEENUM

Specifies how to change the case of a string-type attribute value the Active Roles client sends to Administration Service.

EDS_DATA_SOURCEENUM

Specifies the data storage for the object, such as Active Directory, Network resources, etc.

EDS_EFFECTIVE_POLICY_ENUM

Specifies the policy constraints defined in the Request object for the specified attribute of the processed objects.

EDS_GROUP_TYPEENUM

Specifies the group type of the member.

EDS_MUCONDITIONENUM

Specifies the type of the membership rule that applies to Managed Unit.

EDS_POLICY_COMPLIANCE_ENUM

Identifies the type of message the policy script sends to the Active Roles client.

EDS_REPLICATION_ROLEENUM

Specifies the replication role of the Administration service.

EDST_REQUEST_TYPE

Identifies the type of the requested operation, such as the object creation, deletion, or modification.

EDS_RIGHTS_ENUM

Identifies the access rights to directory objects (a category of Permission)

EDS_SCOPEENUM

Specifies the scope of a directory search.

EDS_SCRIPT_TYPEENUM

Specifies the script type.
 
Note

In PowerShell scripts, to get values of constants defined in Active Roles enumerations, use the Constants object.
