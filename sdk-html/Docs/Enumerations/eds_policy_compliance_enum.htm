EDS_POLICY_COMPLIANCE_ENUM

The EDS_POLICY_COMPLIANCE_ENUM enumeration identifies the type of messages Active Roles sends to clients.

Constant

Value

Description

EDS_POLICY_COMPLIANCE_UNDEF

0

No messages to be sent to the Active Roles client. This value is not used.

EDS_POLICY_COMPLIANCE_ERROR

1

Error message about a significant problem, such as a non-compliance of the object attribute's value with the administrative policy.

EDS_POLICY_COMPLIANCE_INFORMATION

2

A message about the successful operation of the script.

EDS_POLICY_COMPLIANCE_WARNING

3

A message about a problem that might not be significant, but might indicate a future problem.

EDS_POLICY_COMPLIANCE_SUCCESS_AUDIT

4

A message about the validation process that succeeded. 

EDS_POLICY_COMPLIANCE_FAILURE_AUDIT

5

A message about the validation process that failed. 

Remarks

Because VBScript cannot read data from a type library, VBScript applications do not recognize the symbolic constants as defined above. Use the numerical constants, instead, to set the appropriate flags in your VBScript applications. To use the symbolic constants as a good programming practice, create explicit declarations of such constants, as done here, in your VBScript applications.
