EDS_APE_TYPEENUM

The EDS_APE_TYPEENUM enumeration is used to identify types of the Active Roles policies. The IEDMPolicyEntry.Type property can take one of the following values.

Constant

Value

Description of Policy

EDS_APE_TYPE_SCRIPT

3

Script Policy.
Starts a script upon specific events.

EDS_APE_TYPE_HOME_DIR_SHARE

5

User Home Folders and Home Shares.
Manages User home folders and home shares.

EDS_APE_TYPE_VALIDATE_SCRIPT

16

Validates the text of a script module.

EDS_APE_TYPE_VALIDATE_MANAGED_DOMAIN

17

Validates name and credentials of a managed domain.

EDS_APE_TYPE_UNIQUE_NAME

18

Verifies that logon name (pre-Windows 2000) of user (group, computer, inetOrgPerson) is unique.

EDS_APE_TYPE_PRINT_QUEUE_CREATION

19

Verifies whether it is possible to complete a request to create a printer object.

EDS_APE_TYPE_VALIDATE_MAIL_ADDRESS

20

Checks the recipient's e-mail addresses syntax and uniqueness.

EDS_APE_TYPE_EXCHANGE_ATTRIBUTES

21

Returns values and effective policy information for exchange attributes.

EDS_APE_TYPE_DYNAMIC_GROUPS

22

Configures functionality of the dynamic groups and processes membership rules.

EDS_APE_TYPE_GROUP_TYPE_RESTRICTIONS

23

Enforces restrictions on group types.

EDS_APE_TYPE_DEFAULT_GENERATION_RULES

24

Provides default generation rules for the attributes names.

EDS_APE_TYPE_CONTROL_RIGHTS

25

Enforces built-in AD limitations on object operations (such as Delete, Move, Rename) based on the object's properties.

EDS_APE_TYPE_SCHEMA_LIMITATIONS

26

Enforces schema restrictions. Provides user-friendly reporting and validate schema restrictions on namespaces other than the AD namespaces, such as the Active Roles Configuration namespace or Computer Management namespace.

EDS_APE_TYPE_MOVE_TARGETS

31

Provides a support for edsaMoveTargets feature. This policy is used for Enterprise Migration Manager integration.

EDS_APE_TYPE_SCHEMA_CACHE

32

Provides a support for schema cache updates in the Active Roles Administration Service.

EDS_APE_TYPE_PV

35

Property Generation and Validation.
Validates input data and generates default values.

EDS_APE_TYPE_PICTURE_MANAGEMENT_RULES

84

Defines the management rules and restrictions specific to pictures stored in directory object attributes, such as the user, group and contact pictures.

EDS_APE_TYPE_CREATE_MAILBOX

524288

Exchange Mailboxes.
Creates mailboxes on Exchange server.

Remarks

Because VBScript cannot read data from a type library, VBScript applications do not recognize the symbolic constants as defined above. Use the numerical constants, instead, to set the appropriate flags in your VBScript applications. To use the symbolic constants as a good programming practice, create explicit declarations of such constants, as done here, in your VBScript applications.
