EDST_REQUEST_TYPE

The EDST_REQUEST_TYPE enumeration identifies the type of the request the Active Roles client sends to the Administration Service, such as the object creation, deletion, modification, etc.

Constant

Value

Request Description

EDST_REQ_INVALID

1

Invalid request

EDST_REQ_SEARCH

2

Perform a directory search

EDST_REQ_CREATE

4

Create a directory object

EDST_REQ_DELETE

5

Delete a directory object

EDST_REQ_GET

6

Retrieve properties of a directory object

EDST_REQ_SET

7

Modify properties of a directory object

EDST_REQ_MOVE

8

Move a directory object to another container

EDST_REQ_RENAME

9

Rename a directory object

EDST_REQ_COPY

10

Make a copy of a directory object

EDST_REQ_GET_EFFECTIVE_POLICY

11

Retrieve information about Active Roles Policy constraints and definitions for a given directory object

EDST_REQ_CHECK_PROPERTY_VALUES

12

Check an object's property values supplied by the client against Active Roles Policy constraints and definitions for that object

Remarks

Because VBScript cannot read data from a type library, VBScript applications do not recognize the symbolic constants as defined above. Use the numerical constants, instead, to set the appropriate flags in your VBScript applications. To use the symbolic constants as a good programming practice, create explicit declarations of such constants, as done here, in your VBScript applications.
