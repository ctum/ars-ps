EDS_RIGHTS_ENUM

The EDS_RIGHTS_ENUM enumeration identifies the access rights to directory objects (a category of Permission)

Constant

Value

Description of Rights

EDS_RIGHT_DS_CREATE_CHILD 

1

The right to create children of the object. The InheritedObjectTypeADsPath property identifies type of child object whose creation is being controlled. If the InheritedObjectTypeADsPath property is not set, the Permission controls the creation of all child object types.

EDS_RIGHT_DS_DELETE_CHILD 

2

The right to delete children of the object. The InheritedObjectTypeADsPath property identifies type of child object whose deletion is being controlled. If the InheritedObjectTypeADsPath property is not set, the Permission controls the deletion of all child object types.

EDS_RIGHT_ACTRL_DS_LIST 

4

The right to list children of the securable object.

EDS_RIGHT_DS_SELF

8

The right to perform an operation controlled by a validated write access right.

EDS_RIGHT_DS_READ_PROP

16

The right to read a property of the securable object. The ObjectTypeADsPath property identifies that property. If ObjectTypeADsPath is not set, the Permission controls the right to read all  properties of the securable object.

EDS_RIGHT_DS_WRITE_PROP

32

The right to write a property of the securable object. The ObjectTypeADsPath property identifies that property. If ObjectTypeADsPath is not set, the Permission controls the right to write all  properties of the securable object.

EDS_RIGHT_DS_DELETE_TREE

64

The right to delete all children of the securable object, regardless of the permission on the children. 

EDS_RIGHT_DS_LIST_OBJECT

128

The right to list a particular object. If the user is not granted such a right, and the user does not have EDS_RIGHT_ACTRL_DS_LIST set on the object's parent, the object is hidden from the user.

EDS_RIGHT_DS_CONTROL_ACCESS

256

The right to perform an operation controlled by an extended access right.

EDS_RIGHT_EDS_COPY

512

The right to create copies of the object.

EDS_RIGHT_EDS_MOVE

2048

The right to move the object to another container.

EDS_RIGHT_EDS_MOVE_TO

4096

The right to move another objects to the container object.

EDS_RIGHT_DELETE

65536

The right to delete the securable object.

EDS_RIGHT_READ_CONTROL

131072

The right to read information from the security descriptor of the object, not including the information in the SACL.

EDS_RIGHT_WRITE_DAC

262144

The right to modify the discretionary access-control list (DACL) in the object's security descriptor.

EDS_RIGHT_WRITE_OWNER

524288

The right to assume ownership of the object. The user must be a trustee of the object. The user cannot transfer the ownership to other users.

 

Remarks

Because VBScript cannot read data from a type library, VBScript applications do not recognize the symbolic constants as defined above. Use the numerical constants, instead, to set the appropriate flags in your VBScript applications. To use the symbolic constants as a good programming practice, create explicit declarations of such constants, as done here, in your VBScript applications.
