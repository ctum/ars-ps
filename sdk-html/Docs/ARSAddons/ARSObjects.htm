ARSObjects Section

XML Elements

ARSObjects

Describes all Active Roles objects supplied by an add-on.

Parent Element: AddonDocument

Child Elements: ARSObject

Attributes: None

ARSObject

Describes an Active Roles object supplied by an add-on.

Parent Elements: ARSObjects

Child Elements: AllowPostInstallationModification, AllowPostInstallationDeletion, KeepOnUninstall, ReferenceOnly, References, Attributes, OnPreInstallObject, OnPostInstallObject, OnPreUninstallObject, OnPostUninstallObject

Attributes:

Name

Description

objectID

Required. Identifier of the Active Roles object in the XML file.

objectClass

Required. Contains the name of the object class to which the Active Roles object belongs.

dn

Required. Specifies the DN of the Active Roles object.

AllowPostInstallationModification

If set to TRUE, a user can modify an Active Roles object after its installation. Otherwise, modification of the Active Roles object is not allowed.

Parent Element: ARSObject

Child Elements: None

Attributes: None

AllowPostInstallationDeletion

If set to TRUE, a user can delete an Active Roles object after its installation. Otherwise, deletion of the Active Roles object is not allowed.

Parent Element: ARSObject

Child Elements: None

Attributes: None

KeepOnUninstall

If set to TRUE, an Active Roles object will not be deleted after the add-on is uninstalled. Otherwise, the Active Roles object will be removed when the add-on is uninstalled.

Parent Element: ARSObject

Child Elements: None

Attributes: None

ReferenceOnly

If set to TRUE, Add-on Manager cannot modify or delete a Active Roles object. Such object will be a reference for other add-on objects only. This object should not have any attributes.

Parent Element: ARSObject

Child Elements: None

Attributes: None

References

A list of Active Roles objects upon which the current Active Roles object depends.

Parent Element: ARSObject

Child Elements: ObjectReference

Attributes: None

ObjectReference

A reference to an Active Roles object upon which the current Active Roles object depends.

Parent Element: References

Child Elements: None

Attributes:

Name

Description

Value

Required. ObjectID of an Active Roles object upon which the current Active Roles object depends.

Attributes

A list of attributes of an Active Roles object.

Parent Element: ARSObject

Child Elements: Attribute

Attributes: None

Attribute

An attribute of an Active Roles object.

Parent Element: Attributes

Child Elements: Value

Attributes:

Name

Description

Name

Required. The attribute name.

Type

Required. The attribute type. The following types are supported:

  • String

  • Boolean

  • Integer

  • LargeInteger

  • DateTime

  • OctetString

  • GuidString

  • SidString

AllowPostInstallationModification

Optional. If set to TRUE, a user can modify this attribute, and the attribute value will not be reviewed or reset when the add-on is repaired.

Value

An attribute value of an Active Roles object.

Parent Elements: Attribute

Child Elements: None

Attributes: None
