ARSWebUICustomization Section

XML Elements

ARSWebUICustomization

Contains objects related to Web Interface customization.

Parent Element: AddonDocument

Child Elements: SupportedCustomizationVersion, CustomizationItems, ApplyCustomizationToAdminSiteFlag, ApplyCustomizationToHelpDeskSiteFlag, ApplyCustomizationToSSSiteFlag

Attributes:

Name

Description

PlainTextLabel

Optional. Contains any string. This attribute allows you to label elements modified by an add-on. The use of your custom labels allows your applications to easily update or remove custom elements from the configuration settings.

 

SupportedCustomizationVersion

Contains a list of supported Web Interface customization versions.

Parent Element: ARSWebUICustomization

Child Elements: Version

Attributes: None

Version

Number of the supported Web Interface customization version.

Parent Element: SupportedCustomizationVersion

Child Elements: None

Attributes: None

CustomizationItems

Contains a list of Web Interface customization items.

Parent Element: ARSWebUICustomization

Child Elements: CustomizationItem

Attributes: None

CustomizationItem

Contains a Web Interface customization item for a specified configuration template and operation.

Parent Element: CustomizationItems

Child Elements: edsaWIMenus, edsaWICommands, edsaWIForms, edsaWIEntries, edsaWIStrings, edsaWISettings

Attributes:

Name

Description

configurationTemplate

Required. The name of a configuration template. The specified Web Interface customization item will be applied or removed from all Web Interface sites of this configuration template. The possible values are:

  • FullDirectoryManagement - Site for Administrators

  • HelpDesk - Site for Help Desk

  • SelfService - Site for user self-administration

operation

Required. The possible values are:

  • Change - Default value. Adds or updates Web Interface customization items.

  • Remove - Removes Web Interface customization items. This operation applies to leaf elements only.

configurationId

Optional. GUID of a configuration from which a Web Interface customization item was exported.

configurationName

Optional. The name of a configuration from which a Web Interface customization item was exported.

configurationVersion

Optional. The version of a configuration from which a Web Interface customization item was exported.

edsaWIMenus

Describes the edsaWIMenus attribute. For detailed information on this attribute, see The Menus Settings.

Parent Element: CustomizationItem

Child Elements: TaskLists

Attributes: None

Note:

Child elements of the edsaWIMenus element may contain an optional attribute overwriteIfExist. The attribute overwriteIfExist is set to TRUE for the Web Interface customization items that you select to include in your add-on when creating the add-on in the Add-on Editor. When the attribute overwriteIfExist is set to TRUE, the corresponding Web Interface customization items will be overwritten when the add-on is installed.
