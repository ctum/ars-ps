Active Roles SDK > Active Roles Add-ons > Add-on Configuration

AddonConfiguration Section

XML Elements

AddonDocument

Describes the name and GUID of an add-on.

Parent Element: None

Child Elements: AddonConfiguration, SystemRequirements, ARSObjects, ARSWebUICustomization, PreInstallScript, PostInstallScript, PreUninstallScript, PostUninstallScript

Attributes:

Name

Description

Name

Required. The add-on name in the Registered Add-ons container.

GUID

Required. The add-on GUID.

 

AddonConfiguration

Describes the configuration of an add-on.

Parent Element: AddonDocument

Child Elements: Title, Supportability, Version, ShowInRawViewOnly, HasUIConfigurationLink, UIConfigurationDN, HasWICustomizationLink, Publisher, HasUninstallLink, Description, Summary, HomePageURL, HomePageLinkName, AddonIcon

Attributes: None

Title

The name of an add-on.

Parent Element: AddonConfiguration

Child Elements: None

Attributes: None

Supportability

Describes what type of support is available for an add-on.

Parent Element: AddonConfiguration

Child Elements: None

Attributes:

Name

Description

ID

Required. The type of support.

The possible values are:

0 - None (default value)

1 - One Identity

2 - Community

3 - Publisher

Description

Optional. The possible values are:

  • None

  • One Identity

  • Community

  • Publisher

 

Version

The version of an add-on.

Parent Element: AddonConfiguration

Child Elements: None

Attributes:

Name

Description

Major

Required. Represents the major part of the version number. For example, 3 in 3.2.1.

Minor

Optional. Represents the minor part of the version number. For example, 2 in 3.2.1.

Build

Optional. Represents the build part of the version number. For example, 1 in 3.2.1.

ShowInRawViewOnly

When set to TRUE, an add-on is shown in the Raw View Mode only in the Active Roles Console. Otherwise, the add-on is always available in the Applications node in the Active Roles Console.

Parent Element: AddonConfiguration

Child Elements: None

Attributes: None

HasUIConfigurationLink

When set to TRUE, the Add-on Configuration page is displayed when clicking an add-on title link on the Add-on Manager page. By default, the add-on title link opens the Add-on Details dialog box.

Parent Element: AddonConfiguration

Child Elements: None

Attributes: None

UIConfigurationDN

Describes the application object DN (this application object must be included in an add-on) used as an address in the add-on title link. By default, the add-on title link opens the Add-on Details dialog box. By specifying the DN of the application object included in the add-on you configure this link to open the Add-on Configuration page.

Parent Element: AddonConfiguration

Child Elements: None

Attributes: None

HasWICustomizationLink

When set to TRUE, the Web Interface customization link is shown for an add-on on the Add-on Manager page in the Active Roles console.

Parent Element: AddonConfiguration

Child Elements: None

Attributes: None

Publisher

The publisher of an add-on.

Parent Element: AddonConfiguration

Child Elements: None

Attributes:None

HasUninstallLink

When set to TRUE, the Uninstall link is shown for an add-on on the Add-on Manager page in the Active Roles console.

Parent Element: AddonConfiguration

Child Elements: None

Attributes: None

Description

A detailed description of an add-on shown on the Add-on Details page in the Active Roles console.

Parent Element: AddonConfiguration

Child Elements: None

Attributes:None

Summary

A short description of an add-on shown on the Add-on Manager page in the Active Roles console.

Parent Element: AddonConfiguration

Child Elements: None

Attributes: None

HomePageURL

Describes the Web site URL of publisher's home page.

Parent Element: AddonConfiguration

Child Elements: None

Attributes: None

HomePageLinkName

Describes the name of the link to publisher's home page.

Parent Element: AddonConfiguration

Child Elements: None

Attributes: None

AddonIcon

Describes custom icon in a base64 string format for an add-on.

Parent Element: AddonConfiguration

Child Elements: None

Attributes: None
