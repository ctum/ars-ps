Active Roles SDK > Active Roles Add-ons > SystemRequirements

SystemRequirements Section

XML Elements

SystemRequirements

Describes system requirements for an add-on.

Parent Element: AddonDocument

Child Elements: ActiveRolesServer, InstalledAddons

Attributes: None

ActiveRolesServer

Describes what version of Active Roles is required for an add-on.

Parent Element: SystemRequirements

Child Elements: MinVersion, MaxVersion

Attributes: None

MinVersion

Describes the minimum version of Active Roles required to install an add-on.

Parent Element: ActiveRolesServer

Child Elements: None

Attributes:

Name

Description

Major

Required. Represents the major part of the version number. For example, 3 in 3.2.1.

Minor

Optional. Represents the minor part of the version number. For example, 2 in 3.2.1.

Build

Optional. Represents the build part of the version number. For example, 1 in 3.2.1.

 

MaxVersion

Describes the maximum version of Active Roles required to install an add-on.  

Parent Element: ActiveRolesServer

Child Elements: None

Description

Build

Optional. Represents the build part of the version number. For example, 1 in 3.2.1.

 

InstalledAddons

Describes what add-ons are required to install a new add-on.

Parent Element: SystemRequirements

Child Elements: InstalledAddon

Attributes: None

InstalledAddon

Describes an add-on and its version required to install a new add-on.

Parent Elements: InstalledAddons

Child Elements: MinVersion, MaxVersion

Attributes:

Name

Description

AddonName

Required. Name of the required add-on.
