Files installed with the Active Roles SDK

The Active Roles SDK files are installed in the SDK subfolder of the Active Roles installation folder.

 

SDK Documentation

The help file ARServerSDK.chm is located in the folder '...\SDK'. This help provides basic concepts and instructions for using the Active Roles ADSI provider and developing custom policy scripts.

 

Sample Policy Scripts

Samples of policy scripts are located in the folder '...\SDK\Samples'. These samples are described in Sample Policy Scripts.

 

Sample Scripts

Samples of scripts designed to manage directory objects and Active Roles-specific objects are located in the folder '...\SDK\Samples\ADSI Provider'. These samples are described in Examples.

 

Sample Extensions to Active Roles Snap-in

Samples of the C++ projects that illustrate how to develop and install the context menu and property page extensions to Active Roles snap-in. The samples are located in the folder '...\SDK\Samples\ARServer Snap-in'. These samples are described in the sections Extending Active Roles Snap-in with the MMC Extension Snap-ins and Extending Active Roles Snap-in with Display Specifiers.
