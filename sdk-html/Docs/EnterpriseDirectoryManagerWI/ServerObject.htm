Server Object

The Serverobject allows to access methods and properties on the server. Most of these methods and properties serve as utility functions.

The Serverobject is available within the Get_XXX and Set_XXX functions used to create the custom entries. For more information about these functions, refer to Creating Custom Entries.

Methods

The Serverobject supports the following methods:

  • MapPath Method

MapPath Method

The Function procedure that maps the specified relative or virtual path to the corresponding physical directory on the server. This method returns the path to the corresponding physical directory.

Server.MapPath(ByVal strPath)

Arguments

strPath
[in] Required. String. Specifies the relative or virtual path to map to a physical directory.
 

Remarks

If strPath starts with either a forward (/) or backward slash (\), the MapPath method returns a path as if strPath were a full, virtual path. If strPath doesn't start with a slash, the MapPath method returns a path relative to the directory of the .asp file being processed.
