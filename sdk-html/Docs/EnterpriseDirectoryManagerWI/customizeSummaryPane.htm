Customize Summary Pane

When you select an object from the list of search results, the Web Interface displays information about that object in the Summary Pane. Depending on the type of object, the Summary Pane may provide different collections of object properties along with a picture retrieved from Active Directory. You can customize the Summary Pane on an object type basis by specifying the object properties to display, and choosing whether to display a picture and from which attribute to retrieve the picture.

The configuration of Summary Panes is determined by the SummaryPanes child elements in the edsaWISettings attribute of the Web Interface configuration object held in the Interface Settings container (see Storing Customization Settings). The Web Interface includes an object type specific Summary Pane for a number of widely used object types, such as User, Group, and Computer, and provides the default Summary Pane for other object types.

This topic describes how to add or remove a property displayed on the Summary Pane for the User object type.

To add a property to the Summary Pane for the User object type

  1. Save the value of the attribute edsaWISettings into a text file.

  2. Open this file with any XML editor, such as XML Spy.

  3. Under the Properties child element of the SummaryPane element that has the TargetClass attribute set to "user", add a new element named Property and specify the attributes of that child element as follows:

  • AttributeName="<LDAP Display Name of the desired property>"

  • LabelResId="<ID of the string resource to display next to the property value on the Summary Pane>"

You should define the desired string resource along with the ID by adding a Res element to the edsaWIStrings attribute (see The Language Settings).

  1. Save changes to the attribute edsaWISettings.

  2. Reload the Web Interface.

To remove a property from the Summary Pane for the User object type

  1. Save the value of the attribute edsaWISettings into a text file.

  2. Open this file with any XML editor, such as XML Spy.

  3. Under the Properties child element of the SummaryPane element that has the TargetClass attribute set to "user", remove the Property element whose AttributeName attribute matches the LDAP Display Name of the object property you want to remove from the Summary Pane.

  4. Save changes to the attribute edsaWISettings.

  5. Reload the Web Interface.
