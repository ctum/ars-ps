Creating a Custom Icon for Directory Object

This topic explains how you can create a custom icon for directory objects of the specified class.

To create the custom icon, use the following procedure to make the appropriate changes to the attribute edsaWISettings of the object WorkingCopy residing in the Interface Settings container.

To create a custom icon for directory objects

  1. Save the value of the attribute edsaWISettings into a text file.

  2. Open this file with any XML editor, such as XML Spy, and find the element CustomObjectIcons.

  3. For the element CustomObjectIcons, insert the child element Icon in the following way:

    <Icon ObjectClass=strClassName BigIconFileName=strBigIconFName SmallIconFileName=strSmallIconFName />
    Where:
    strClassName     - The name of the object class to which the custom icon will apply.
    strBigIconFName     - The file name for the big custom icon. 
    strSmallIconFName     - The file name for the small custom icon. 

    For detailed information about attributes of the element Icon, refer to Interface Settings. When finished, save the file.

  4. Save changes to the attribute edsaWISettings.

  5. Reload the Web Interface.

Example

The following code snippet illustrates how to create your custom icons for User objects.

<CustomObjectIcons>
  <Icon ObjectClass="user" BigIconFileName="customUser_big.gif" SmallIconFileName="customUser_small.gif" />
</CustomObjectIcons>

Note:

The files customUser_big.gif and customUser_small.gif must be saved into the folder <WI folder>\Public\images, where <WI folder> refers to the folder where the Web Interface is installed.
