Reloading the Web Interface

After modifying the Web Interface customization settings, for changes to take effect, reload the Web Interface. You can reload the Web Interface using the following script.

To reload the Web Interface with the custom script

  1. Copy the script code, paste it into Notepad or another text editor, and then save the file with a .vbs file extension.

  2. Set constants placed between the lines "' PARAMETERS OF SCRIPT YOU MAY MODIFY" and "PLEASE, DO NOT MODIFY THE SCRIPT BELOW THIS POINT" to the appropriate values specific to your environment.

  3. Run the script

The following script allows you to reload the specified Web Interface.

' This script allows you to restart the specified  Active Roles Web site
'
' -----------------------------------
' PARAMETERS OF SCRIPT YOU MAY MODIFY
' -----------------------------------
' Before using the script, modify the following constants
'
' Name of host where the Web Interface is installed
const strHost ="localhost"
' Web site name
const strSite="WI"
' -------------------------------------------------
' PLEASE, DO NOT MODIFY THE SCRIPT BELOW THIS POINT
' -------------------------------------------------
'
const scriptName = "/Customization/ReloadMetadata.aspx?ReloadFromWorkingCopy=1"
URL = "http://" & strHost & "/" & strSite & scriptName
Set objHTTP = CreateObject("Msxml2.XMLHTTP.6.0")
objHTTP.open "GET", URL, false
objHTTP.send
msgBox "Done!"
