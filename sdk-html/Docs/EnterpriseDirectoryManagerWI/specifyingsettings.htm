Specifying Customization Settings

Customization settings for the Web Interface can be changed in one of the following ways:

  • Using the Active Roles snap-in

  • Using your custom scripts

Using the Active Roles Snap-in

To specify a customization setting with Active Roles snap-in

  1. Choose the Raw mode for Active Roles snap-in display:

    • On the View menu, click Mode, and then in the View Mode dialog box, select the Raw Mode option and click OK.

  2. In the console tree, expand the Configuration node, expand the Application Configuration node, expand the Web Interface node, and then click the configuration container for the WI configuration you want to customize. The name of the WI configuration is saved in the attribute edsaWIConfigurationName of the corresponding configuration container.

  3. In the details pane, do one of the following:

    • To specify a user interface setting, double-click Interface Settings.

    • To specify a setting for menus, commands or forms, double-click Customization Settings.

  4. In the details pane, right-click WorkingCopy, point to All Tasks, and then click Advanced Properties.

  5. In the Advanced Properties dialog box, do the following:

    • Select the Show all possible attributes check box. Select the Use LDAP display names option. In the list of attributes, select the attribute you want to modify, and then click Edit to modify its value. Note that the Web Interface only uses the first value of multivalued attributes.

  6. For changes to take effect, reload the Web Interface.

 Using Your Custom Scripts

You can read and modify values of the attributes of the WorkingCopy object with the GetEx and PutEx methods of the IADs interface.

  To specify a customization setting with custom scripts

  1. Select a customization attribute to be modified and save its value into a text file.

  2. Make the appropriate changes to the file using any XML editor, such as XML Spy. For more information, see Customization Settings Format.

  3. Read customization settings from the file, and then save them into the customization attribute.

  4. For changes to take effect, reload the Web Interface.

This topic details the steps 1, 3 and 4 of the procedure above, and provides sample scripts (VBScript) that illustrate how to modify attributes of the WorkingCopy object.

Sample Scripts

To use sample scripts

  1. Copy the script code, paste it into Notepad or another text editor, and then save the file with a .vbs file extension.

  2. Set constants placed between the lines "' PARAMETERS OF SCRIPT YOU MAY MODIFY" and "PLEASE, DO NOT MODIFY THE SCRIPT BELOW THIS POINT" to the appropriate values specific to your environment.

  3. Run the script.

Writing the Attribute Values to a File

The following script illustrates how to write values of the edsaWIStrings attribute to the specified file.

'
' -----------------------------------
' PARAMETERS OF SCRIPT YOU MAY MODIFY
' -----------------------------------
' Before using the script, modify the following constants
'
' Path to XML file you want to create
Const strFileName = "mysettings.xml"
' ldapDisplayName of the attribute you want to save into XML file
Const strAttrName = "edsaWIStrings"
' Identifier of the Web Interface configuration you want to change
Const strWebSiteID = "88c0ace0-bd43-4858-b4dc-43afa5d95976"
' Configuration container name: Customization Settings or Interface Settings
Const strConfCont = "Customization Settings"
'
' -------------------------------------------------
' PLEASE, DO NOT MODIFY THE SCRIPT BELOW THIS POINT
' -------------------------------------------------
strWorkingCopyDN = "EDMS://CN=WorkingCopy,CN=" & strConfCont & ",CN=" & strWebSiteID & ",CN=Web Interface,CN=Application Configuration,CN=Configuration" 
set objWorkingCopy = GetObject(strWorkingCopyDN) 
objWorkingCopy.GetInfoEx Array(strAttrName),0 
arrayXML = objWorkingCopy.GetEx(strAttrName)
strXml = ""
For Each arrItem In arrayXML
  strXml = strXml & arrItem
Next 
Set xmlDoc = CreateObject("Msxml2.DOMDocument.6.0")
xmlDoc.async = False
xmlDoc.loadXML(strXML)
If xmlDoc.parseError.errorCode <> 0 Then
  Err.Raise vbObjectError + 1, nothing, "Error parse xml : " & mlDoc.parseError.reason
End If
xmlDoc.save(strFileName)
Set xmlDoc = nothing
MsgBox "The file " & strFileName & " has been created."

Reading Customization Settings from a File

The following script illustrates how to read customization settings from the specified file, and then save them to the edsaWIStrings attribute of the WorkingCopy object.

' This script allows you to read the  Active Roles Web Interface customization settings from an XML file
' and save them into the appropriate attribute of the object WorkingCopy
'
' -----------------------------------
' PARAMETERS OF SCRIPT YOU MAY MODIFY
' -----------------------------------
' Before using the script, modify the following constants
'
' Path to XML file you want to use
Const strFileName = "mysettings.xml"
' ldapDisplayName of the attribute you want to set
Const strAttrName = "edsaWIStrings"
' Identifier of the Web Interface configuration container where the WorkingCopy object resides
Const strWebSiteID = "88c0ace0-bd43-4858-b4dc-43afa5d95976"
' Configuration container name: Customization Settings or Interface Settings
Const strConfCont = "Customization Settings"
'
' -------------------------------------------------
' PLEASE, DO NOT MODIFY THE SCRIPT BELOW THIS POINT
' -------------------------------------------------
Const  ADS_PROPERTY_CLEAR = 1
Const  ADS_PROPERTY_UPDATE = 2
strWorkingCopyDN = "EDMS://CN=WorkingCopy,CN=" & strConfCont & ",CN=" & strWebSiteID & ",CN=Web Interface,CN=Application Configuration,CN=Configuration"
Set objWorkingCopy = GetObject(strWorkingCopyDN)
Dim 	xmlDoc
Set xmlDoc = CreateObject("Msxml2.DOMDocument.4.0")
xmlDoc.async = False
xmlDoc.load(strFileName)
If xmlDoc.parseError.errorCode <> 0 Then
  Err.Raice vbObjectError + 1, nothing, "Error parse xml : " & mlDoc.parseError.reason
End If
objWorkingCopy.PutEx ADS_PROPERTY_CLEAR, strAttrName, Array("0")
objWorkingCopy.SetInfo
objWorkingCopy.PutEx ADS_PROPERTY_UPDATE, strAttrName, Array(xmlDoc.xml)
objWorkingCopy.SetInfo
Set xmlDoc = nothing
MsgBox "The attribute " & strAttrName & " of the object WorkingCopy has been set"
