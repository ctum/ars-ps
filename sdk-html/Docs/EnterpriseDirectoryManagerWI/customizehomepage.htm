Customize the Web Interface Home Page

This topic explains how you can add your custom items to the Web Interface home page. By default, the home page includes 4 items (Directory Management, Customization, Settings, Approval).

To customize the home page, use the following procedure to make the appropriate changes to the attribute edsaWISettings of the object WorkingCopy residing in the Interface Settings container.

To add an item to the Web Interface home page

  1. Save the value of the attribute edsaWISettings into a text file.

  2. Open this file with any XML editor, such as XML Spy, and find the element TaskBoxes.

  3. For the element TaskBoxes, insert the child element Item in the following way:

    <Item ID=ItemID Location="default" IcontPath=strIconPath IconHoverPath= strIconHoverPath Header=strHeaderConst
    Description=strDescrConst, NavigateUrl=URL, NavigateCommand=Command/>
    Where:
    ItemID     - Required. Unique identifier of the item.
    strIconPath - Required. Path and file name of an icon file that contains the icon image assigned to this item.
    strHeaderConst     - Required. Path and file name of an icon file that contains the icon image displayed when you move the mouse pointer into the item area.
    strHeaderConst     - Optional. Identifier of the string constant containing a header of the item.
    strDescrConst     - Optional. Identifier of the string constant containing a description of the item.
    URL     - Optional. Contains a shortcut to a Web page to be opened after clicking the item.
    Command     - Optional. Contains a code snippet to be run when you click the item area.

    For detailed information about attributes of the element Item, refer to Interface Settings. When finished, save the file.

  4. Save changes to the attribute edsaWISettings.

  5. Reload the Web Interface.

Example

The following code snippet illustrates how to add a custom item MyItem on the WI home page. The constants CST_XXX must be set in the edsaWIStrings attribute of the object WorkingCopy residing in the Customization Settings container.

<TaskBoxes>
 ' .......
 ' Default items are described here
 ' .......
    <Item ID="MyItem" Location="default" IcontPath="~/CustomImages/myicon.gif" IconHoverPath="~/CustomImages/myiconhover.gif"
      Header="CST_HOME_MYITEM_HEADER"
      Description="CST_HOME_MYITEM_DESCRIPTION" NavigateUrl="~/Customization/MY.aspx" NavigateCommand="" />
</TaskBoxes>
