Customization Settings

This section provides you with information about how to customize the Active Roles Web Interface (Web Interface) using the Web Interface customization settings stored in the Administration Database. The customization allows users who have administrative access to the Administration Service to modify menus, commands, and forms used for self-administration and administering directory objects to suit the specific needs of your organization.

The Administration Database is maintained by Microsoft SQL Server. Using the replication functionality of SQL Server, the customization settings are distributed over the entire replication group. When you install a new Web Interface that uses any Administration Service from the replication group, the equivalent customization settings will be applied to the new Web Interface.

Customization settings allow you to customize the Web Interface programmatically. In addition, the Web Interface provides a "point-and-click" solution, the administrative task Customization that also allows the Active Roles administrators to customize the Web Interface. For more information about the Web Interface customization, see "Customizing the Web Interface" in the Active Roles Web Interface Administrator Guide.
