DropDownList Object

The DropDownListobject provides properties and methods used to configure a drop-down list control displayed on the current form. To get a reference to this object, use the AddDropDownList method provided by the FormPage object.

Properties and Methods

The DropDownListobject supports the following properties and methods:

  • Enabled Property

  • AddOption Method

  • AddProperty Method

  • AddEvent Method

Enabled Property

Read/Write. Boolean. Indicates whether the drop-down list control is enabled.

Obj.Enabled

Arguments

Obj
Required. Refers to a DropDownList type object.

AddOption Method

The Sub procedure that adds a new element (option) to the drop-down list.

Obj.AddOption(ByVal  strOptionValue, ByVal strOptionText, ByVal bSelected)

Arguments

Obj
Required. Refers to a DropDownList type object.
strOptionValue
[in] Required. String. Specifies the Value attribute for the Option element to add to the drop-down list.
strOptionText
[in] Required. String. Specifies the text for the option to add to the drop-down list.
bSelected
[in] Required. Boolean. When TRUE, the newly added element is selected.

AddProperty Method

The Sub procedure that adds attributes (such as "name," "size," etc.) to the Select element in the drop-down list control.

Obj.AddProperty(ByVal strPropertyName, ByVal strPropertyValue)

Arguments

Obj
Required. Refers to a DropDownList type object.
strPropertyName
[in] Required. String. Specifies the attribute name.
strPropertyValue
[in] Required. String. Specifies the attribute value.
 

Back to top

AddEvent Method

The Sub procedure that adds a client-side event to the Select element in the drop-down list control.

Obj.AddEvent(ByVal strEventName, ByVal strScriptName)

Arguments

Obj
Required. Refers to a DropDownList type object.
strEventName
[in] Required. String. Specifies the event type, such as "onactivate", "onchange", etc. For information about all available event types, refer to MSDN library.
strScriptName
[in] Required. String. Specifies the path to the script handler file.

Example Code (VBScript)

This code snippet illustrates how you can add a new drop-down list control to your custom entry.

Sub Get_MyCustomEntry(byRef objFormContext,byRef objFormPage)
 '
 ' --- your code goes here ---
 '
 Set objDropDownList = objFormPage.AddDropDownList("MyList")
 objDropDownList.Enabled = True
 objDropDownList.AddProperty("name","Cities")
 objDropDownList.AddOption("London","London (UK)",False)
 objDropDownList.AddOption("Paris","Paris (FR)",False)
 objDropDownList.AddOption("New York","New York (USA)",True)
 '
 ' --- your code goes here ---
 '
End Sub
