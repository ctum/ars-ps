Active Roles SDK

 

Purpose

The Active Roles SDK provides developers with documentation and samples to help them customize Active Roles by creating custom client applications and user interfaces, and implementing business rules based on running custom scripts (policy scripts). The Active Roles SDK serves as the main source of information that assists in customizing Active Roles in order to increase its flexibility and streamline the administrative workflow.

Active Roles caters to the needs of Active Directory administration and the Active Roles SDK helps expand its use by integrating it with the existing proprietary applications and network data sources, such as Human Resources databases and ERP systems. Custom user interfaces can also be created to meet specific enterprise needs.

 

Where applicable

Developers will use the Active Roles SDK as a reference when designing and authoring Active Roles customization projects. To enable custom client applications and user interfaces to communicate with its Administration Service, Active Roles offers Active Roles ADSI Provider. The ADSI Provider implements a set of interfaces that are ADSI-compliant, conform to Component Object Model (COM), and support standard COM features. The Active Roles SDK includes documentation and samples to make using the ADSI Provider easy.

Network administrators can considerably increase the power of Active Roles by using script policy. Based on running user-developed scripts, script policy could regulate data format and requirements, automate administrative tasks, and trigger additional actions to involve various network data sources in account provisioning. The Active Roles SDK provides conceptual information, instructions, and examples to consider when designing and developing policy scripts.

 

Benefits

The Active Roles SDK:

  • Offers basic concepts and instructions for using the ADSI Provider

  • Explains how to develop and implement policy scripts

  • Provides real-life examples of scripts

The Active Roles SDK is the main technical resource for customizing Active Roles. Customization increases the flexibility of Active Roles and allows organizations to tailor this product to suit their specific needs.
