Discontinued features

Web Interface

These Web Interface features are not supported in the current version of Active Roles.

Discontinued Feature

Description (in Earlier Versions)

Replacement (in Current Version)

The IsTaskAllowedFunction attribute of the XML element  Task.

The IsTaskAllowedFunction attribute contains the name of a function that checks whether a custom command is available for the Web Interface's user.

Use the IsCommandAllowedFilters  element and its child elements AVFilter, AAEFilter, and VBSFilter instead.

The  element IsCommandAllowedFilters describes a collection of the command filters that enable/disable a custom command basing on the certain criteria.

The GoToObjectpageAfterSave attribute of the XML element UIPolicies.

Specifies whether to open the  Properties form for a newly created object.

Use the WizardFinishOptionSettings element instead.

The WizardFinishOptionSettings element describes settings applied to the Open properties for this object when I click Finish check box displayed on the Completion page of the Create New Object wizard.

The Elements XML element - a child element of the XML element Settings.

Describes a collection of UI elements displayed on the Web Interface.

Use the UIPolicy element instead.

 

Script Policies

These features are not supported in the current version of Active Roles.

 

Discontinued Feature

Description  (in Earlier Versions)

Replacement  (in Current Version)

Add/Remove script policy parameters using the Active Roles MMC snap-in.

The Policies tab of the Policy Object Properties dialog box allows you to add/remove custom parameters used in policy scripts.

To add or remove custom parameters used in policy scripts, use the onInit handler or the appropriate methods provided by the IEDMPolicyEntry interface. For details, refer to Specifying Parameters for Policy Scripts.
