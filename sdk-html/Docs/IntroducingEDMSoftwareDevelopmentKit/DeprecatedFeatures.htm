Deprecated features

Deprecated features include features that are still supported in the current version of Active Roles, but they may be removed in the future.

When developing new applications, you should avoid using deprecated features. When modifying existing codes, you are strongly encouraged to remove any dependency on deprecated features. We recommend that you replace these features with the replacement item if possible.

ADSI Provider

These Active Roles ADSI provider features will not be supported in a future version of Active Roles.

 

Interface

Deprecated

Property or Method

 Replacement
IEDMDynamicGroup MembershipRules property: Sets and returns an array of membership rules for a dynamic group. Use the MemberShipRuleCollection property instead.
IEDMDynamicGroup ConvertToDynamicGroup method: Allows you to convert a basic group to a dynamic group. For sample code that illustrates how to convert a basic group to a dynamic group, see IEDMDynamicGroup.
IEDMManagedUnit MembershipRules property:Sets and returns an array of membership rules for Managed Unit. Use the MeberShipRuleCollection property instead.
IEDMPolicyEntry RemovePolicySetting method: removes the specified parameter from Policy Object entry. Use the RemovePolicySettingbyName method instead.
IEDMPolicySetting SettingID property: Sets and returns the ID of the Policy Object entry parameter. Use the Name property instead.

Feature Name

Deprecated
Technique

Replacement

Searching for directory objects using an attribute scope query (ASQ)

The ASQ search on the edsaMember and edsaMemberIndirect attributes of groups.

The ASQ search on the edsaMemberOf and edsaMemberOfIndirect attributes of groups and users.

Use the ASQ search on the member and memberOf attributes with the specified appropriate Active Roles controls instead. For details and sample scripts, refer to Searching Using an Attribute Scope Query.
