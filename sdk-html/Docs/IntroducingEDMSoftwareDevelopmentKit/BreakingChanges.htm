Breaking changes

This topic describes the changes made to the current Active Roles version that could cause applications based on earlier versions to break.

 

Changes to the Active Roles Schema and Configuration Namespace Containers

The following table lists changes in distinguished names of the Active Roles Schema and configuration namespace containers. If your applications use distinguished names of those containers, replace them with the replacement item. We recommend that you use the rootDSE object to get distinguished names of the Active Roles specific objects. For more information, refer to Binding and the rootDSE object.

Changes in Namespace of Active Roles Web Interface Customization Settings Format

The customization allows you to modify menus, commands, and forms used for administering directory objects with the Web Interface. All customization information are saved in the Extensible Markup Language (XML) format.

The namespace used in those XML files has changed. If your Web Interface customization uses those namespaces, replace them with the replacement item.
