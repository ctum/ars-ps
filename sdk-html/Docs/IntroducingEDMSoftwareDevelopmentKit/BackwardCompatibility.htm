Backward Compatibility

This section describes the changes in behavior between the current version $ProductVersionXYZB$ and earlier versions of the Active Roles SDK.

The following subjects are covered:

  • Deprecated Features: describes features that will not be supported in a next version of Active Roles

  • Discontinued Features: describes features from earlier versions that are not supported in the current Active Roles version.

  • Breaking Changes: describes the changes made to the current Active Roles version that could cause applications based on earlier versions to break.

  • Behaviour Changes: describes the changes in behavior of some features in the current version of Active Roles from their behavior in earlier versions of the Active Roles SDK.
