IEDMRDNString Properties

The following table describes properties provided by the IEDMRDNString interface.

Property

Description

Attribute

Access: Read-only

Data type: String

Returns the RDN attribute name, such as CN or OU.

Value

Access: Read-only

Data type: String

Returns the RDN attribute value.

String (default property)

Access: Read-only

Data type: DirectoryString

Returns the RDN attribute value. If this value contains reserved characters, they will be escaped (for more information on reserved characters, refer to "Distinguished Names" in Related Topics).
