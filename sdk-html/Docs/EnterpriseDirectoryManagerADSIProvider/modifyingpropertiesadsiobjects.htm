Modifying Properties of ADSI Objects

You can modify properties of a directory object to which you have bound using the Put method (for single value properties) or PutEx method (for multi value properties) of the IADs interface. 

The basic IADs properties of ADSI objects (listed in the table under IAds Properties in the IADs topic) can be modified using the dot operator.

The following code snippet (VBScript) shows how to modify properties of a user object: 

Dim usr, location, phone
Dim Ads_Property_Append
on error resume next
' Set the PutEx control code
Ads_Property_Append=3
' Bind to the user object
Set usr = GetObject("EDMS://ServerName/CN=bosse,OU=Sales,DC=mycompany,DC=COM")
' Retrieve the current values of the user locality
location=usr.get("l")
' Prompt to specify a new location for that user
location=InputBox("Specify the location of the " + usr.Name + " user. User's current locality is "_
                  + CStr(location))
' Set the new location for the user
usr.put "l", location
' Prompt to specify the first home phone of the user
phone=InputBox("Input the first home phone of the " + usr.Name + " user")
usr.PutEx ADS_PROPERTY_APPEND, "OtherhomePhone", Array(Cstr(phone))
usr.SetInfo
' Prompt to specify the second home phone of the user
phone=InputBox("Specify the second home phone of the " + usr.Name + " user")
usr.PutEx ADS_PROPERTY_APPEND, "OtherhomePhone", Array(Cstr(phone))
usr.SetInfo
' Modify the description of the user ("Description" is a basic IADs property) using the dot operator
usr.Description="Description can be modified using the dot operator"
usr.SetInfo

Unlike getting a property value, setting a property requires an additional step: you must call the SetInfo method to send property changes to the Active Roles Administration Service. If you modify a multi value property, you should invoke the SetInfo method directly after calling the PutEx method.

Note:

If you call GetInfo after making changes to properties, but before calling SetInfo, you will lose all those changes.
GetInfo overwrites the property values in the property cache.
