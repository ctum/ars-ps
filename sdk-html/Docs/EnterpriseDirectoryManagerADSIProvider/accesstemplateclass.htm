edsAccessTemplate Class

The edsAccessTemplate class is used to store information about Access Template. In addition to standard AD attributes, Active Roles defines the following Access Template object attributes, which can be accessed using the Active Roles ADSI Provider.

edsaATEList

Syntax: DirectoryString

Access: Read/Write

Searchable: Yes.

Contains information about ATEs in Access Template.
