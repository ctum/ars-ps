Converting a User Mailbox to Linked Mailbox

File name: Convert Mailbox to Linked Mailbox.vbs

This code snippet illustrates how to convert a user mailbox to a linked mailbox. A linked mailbox is a mailbox associated with an external account.

Notes

  • Before using this sample script, ensure that you have installed Exchange Management Tools on the computer running Active Roles Administration service.

  • This script can be applied only to disabled mailbox-enabled users.

Set objUser = GetObject ("EDMS://CN=John Smith,OU=Sales,DC=mycompany,DC=com ")
objUser.Put "edsva-MsExch-ConvertToLinkedMailbox", True
objUser.Put "edsva-msExch-LinkedMasterAccountDN", "CN=John Smith Shadow Copy,OU=Sales,DC=shadowDomain,DC=com"
objUser.SetInfo
MsgBox ("Done!")
