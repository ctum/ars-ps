IEDMMembershipRuleCollection Properties

The following table describes properties provided by the IEDMMembershipRuleCollection interface.

Property

Description

_NewEnum

Access: Read-only

Returns an interface on an enumerator object.

Note:

In Visual Basic and VBScript, the For Each …In … Next statement invokes this property implicitly.

Applying the For Each …In … Next statement to an IEDMMembershipRuleCollection object allows you to enumerate all members in the collection of membership rules applied to a Managed Unit or dynamic group.

Count

Access: Read-only

Data type: Long integer

Returns the number of rules in the membership rule collection.

GuidLookupMustBePerformed

Access: Read/Write

Data type: Boolean

The default value is TRUE.

When FALSE, the provider will not look up AdsPath to the directory object the membership rule uses. This allows you to specify membership rules that use objects that were deleted, moved or are unavailble. For a sample code that illustrates how to use this property, refer to IEDMMembershipRuleCollection.

Item

Access: Read-only

Data type: IEDMMembershiprule

Retrieves an IEDMMembershipRule instance by index. The Item property is a read-only indexer that allows for accessing a membership rule by its index.

ObjRuleCollection.Item(Indx)

Arguments

ObjRuleCollection
Required. Always a name of the instance of the IEDMMembershipRuleCollection object.
Indx
[in] Required. Long integer. The index of the membership rule you want to retrieve.

Note:

The Item property is default property of IEDMMembershipRuleCollection. You can make your source code more compact by omitting the property name Item (see the sample code fragments below).

IsModified

Access: Read only

Data type: Boolean

Returns a Boolean that represents whether the membership rule collection has been changed since the last time the GetInfo/SetInfo method was calleed.

Example Code (VBScript)

The following code fragments use both standard and default property syntax for the property Item.

' Standard syntax
Set objRule = objCollection.Item(2)


' Default property syntax
Set objRule = objCollection(2)

For examples of the VBScript code snippets illustrating the use of this interface, please refer to IEDMMembershipRuleCollection.
