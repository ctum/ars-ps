IEDMPolicyInfoList

The IEDMPolicyInfoList interface enables you to process an object returned by the GetPolicyInfoList method of IEDM.

Methods

The IEDMComplianceInfo interface defines the following methods:

Method

Description

_NewEnum

Allows you to enumerate the object attributes on which an administrative policy is set.

Item

Returns an object used to represent the IEDMPolicyInfo interface.
