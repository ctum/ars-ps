IEDMLookup::Initialize2

The Initialize2 method initializes an object used to represent the IEDMLookup interface. This method must be called after creation of that object and before using any other methods provided by this interface.

Sub  Initialize2(ByVal ServicePath, ByVal User, ByVal Passw, ByVal Flag)

Arguments

ServicePath
[in] Required. String. Specifies the ADsPath of the Administration service computer to which you want to connect. Set this argument to "" (empty string) to connect to the service running on the local machine.
User
[in] Required. String. Specifies the user name to be used for securing permission from the namespace service. Set this argument to "" (empty string) to use the current user's name.
Passw
[in] Required. String. Specifies the password to be used for obtaining permission from the namespace server. Set this argument to "" (empty string) to use the current user's password.
Flag
[in] Required. Long integer. Authentication flag used to define the binding options. For more information about the binding options, please refer to IADsOpenDSObject.

Remarks

To create an instance of the object used to represent the IEDMLookup interface, use the following string: 

Set ObjLookup = CreateObject("EDSILookup")
