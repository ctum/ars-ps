domainDNS Class

The domainDNS class provides a formal description of a domain object. In addition to the standard ADSI domain class attributes, Active Roles defines the following domain object attributes, which can be accessed using the Active Roles ADSI Provider.

Note:

For more information about the Domain class, please refer to the MSDN Library.

 

edsaConfigNC

Syntax: DirectoryString (Unicode).

Access: Read-only.

Searchable: No.

Returns the distinguished name of Configuration Naming Context for this domain.

 

edsaDomainNetbiosName

Syntax: DirectoryString.

Access: Read-only.

Searchable: No.

Returns the NETBIOS domain name for this domain.

 
edsaDnsName

Syntax: DirectoryString.

Access: Read-only.

Searchable: No.

Returns DNS name of this domain.

 

edsaDomainControllers

Syntax: DirectoryString.

Access: Read-only.

Searchable: No.

Returns a list of the Domain Controllers available in this domain in the following form: "<DC DNS name>\07\04\07<Site Name>".

 
edsaForestName 

Syntax: DirectoryString.

Access: Read-only.

Searchable: No.

Returns name of the root domain of the forest to which this domain belongs.

 
edsaIsMsExchangePresent

Syntax: Boolean.

Access: Read-only.

Searchable: No.

Indicates whether Exchange Tasks can be performed for the recipients residing within the domain. You can get a list of all possible Exchange Tasks for a recipient by reading its attribute edsaExchangeTasksAvailable. For details, see Exchange related tasks.

 
edsaLDAPServer

Syntax: DirectoryString.

Access: Read/Write.  

Searchable: No.

Stores the DNS name of the operational domain controller to which the client is connected.

 
edsaPreferredDC

Syntax: DirectoryString.

Access: Read-only.

Searchable: Yes.

Returns the preferred (DirSync) domain controller the Active Roles Administration Service uses to operate with this domain.

 
edsaSchemaNC

Syntax: DirectoryString.

Access: Read-only.

Searchable: No.

Returns the distinguished name of Schema Naming Context for this domain.

 
edsvaIsRestoreEnabled

Syntax: Boolean

Access: Read-only.  

Searchable: No.

Indicates whether the Recycle Bin feature is enabled for this domain. This feature is supported only for Windows 2008 R2.
