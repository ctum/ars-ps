IADsPropertyEntry

The IADsPropertyEntry interface is used to manage a property entry in the property cache. A property entry holds a value (or values) of an attribute as defined in the schema. It is identified by the name of the corresponding attribute. A property entry object allows a user to specify how its values are to be manipulated. Examples of such operations include Update, Modify and Delete.

Multiple property entries are managed by a property list. To access a property entry, you call the Item or GetPropertyItem method on the IADsPropertyList interface (see the sample code in IADsPropertyList).

Note:

Before calling the methods of this interface, you should call the GetInfo or GetInfoEx method of IADs to load the assigned property values of the ADSI object from the Active Roles Administration Service into the property cache. After calling the methods of IADsPropertylist, you should call the SetInfo method of IADs to send all changes in the persistent store to the Active Roles Administration Service.

IADsPropertyEntry Properties

The IADsPropertyEntry interface defines the following properties.

Property

Description

Name

Access: Read/Write

DataType: BSTR

Name of the property entry. This name should correspond to the name of an attribute as defined in the schema.

ADsType

Access: Read/Write

DataType: LONG

The data type of the Name property. The values of the data type is defined in the ADSTYPEENUM enumeration.

ControlCode

Access: Read/Write

DataType: LONG

A constant that specifies the operation to be performed on the named property. The value is defined in the ADS_PROPERTY_OPERATION_ENUM enumeration. In addition, that constant can be equal to 0.

Values

Access: Read only

DataType: VARIANT

A VARIANT array. Each element in this array represents a value of the named property. Such property values are represented by ADSI objects implementing the IADsPropertyValue interfaces. Therefore, the VARIANT array holds an array of pointers to the IDispatch interface on the ADSI objects implementing the IADsPropertyValue interface.

Note:

For more information about the IADsPropertyEntry interface and for examples that illustrate the use of this interface, refer to the MSDN Library.
