Creating a Share

File name: ShareCreate.vbs

This code snippet illustrates how to share the specified folder on the computer in question.

option explicit
Const strLocalFolder = "C:\Temp"
Const strShareName = "Temp"
' Set samAccountName of the computer
Const strComputer = "mycomputer$"
Const strDomainNetBiosName = "MYCOMPANY"
Const strShareDescription = "Temp Folder"
Dim ContainerObject
Set ContainerObject=GetObject("EDMS://CN=Shares,CN=" + strComputer + ",cn=" + strDomainNetBiosName + ",CN=Network")
 'Create a new share within the specified container
Dim ShareObject
Set ShareObject = ContainerObject.Create("edsShare", "cn="+strShareName)
 'Set properties of the newly created share
ShareObject.Put "cn", strShareName
ShareObject.Put "edsaSharePath", strLocalFolder
ShareObject.Put "edsaShareCreateFolder", False
ShareObject.Put "edsaShareComment", strShareDescription
ShareObject.SetInfo

This scripts shares the existing folder C:\Temp on the computer mycomputer in the domain mycompany.com. The share name is Temp.

Note:

If you want the script to create a folder to be shared, in the script source, replace the line
'ShareObject.Put "edsaShareCreateFolder", False'
with the line
'ShareObject.Put "edsaShareCreateFolder", True'
