Enumerating User Memberships

The following code example shows how to enumerate the groups that a user is the member of. 

[ C# ]
using System;
using System.DirectoryServices;
using ActiveDs;
using EDMLib;
namespace EnumerateMemberships
{
	
	class Class1
	{
		[STAThread]
		static void Main(string[] args)
		{
			string strUserADsPath = "EDMS://cn=UserName,ou=Sales,dc=mydomain,dc=local";
			DirectoryEntry oUser = new DirectoryEntry(strUserADsPath);
			Console.WriteLine("Groups to which belongs:"+ oUser.Name);
			// Invoke IADsUser::Groups method.
			IADsMembers groups = oUser.Invoke("Groups") as IADsMembers;
			foreach ( object group in groups)   
			{
				// Get the Directory Entry.
				DirectoryEntry groupEntry  = new DirectoryEntry(group);
				Console.WriteLine(groupEntry.Name); 
			}
		}
	}
}


[ Visual Basic .NET ]
Imports System.DirectoryServices
Imports ActiveDs
Module Module1
    Sub Main()
        Dim usr As IADsUser
        Dim grp As IADsGroup
        Dim strUserADsPath As String = "EDMS://cn=UserName,ou=Sales,dc=mydomain,dc=local"
        Dim obj As New DirectoryEntry(strUserADsPath)
        usr = obj.NativeObject
        For Each grp In usr.Groups
            Console.WriteLine(grp.Name & " (" & grp.Class & ")")
        Next
        usr = Nothing
    End Sub
End Module

To use this example

  1. Start Visual Studio and create a new C# or Visual Basic project using the procedure outlined in Getting Started with Microsoft .NET Projects.

  2. Copy this sample code and then paste it into the Code Pane. Set the variable strUserAdsPath to the distinguished name of the user whose memberships you want to enumerate.

  3. Compile and run the application.
