IEDMAccessTemplate Properties

The following table describes properties provided by the IEDMAccessTemplate interface.

Property

Description

ID

Access: Read-only

Data type: GUID

Returns the value of the objectGUID property of the Access Template object, such as 1606E8E0-FBC9-4B8B-BC01-66F420E9B4F1

NestedTemplateCollection

Access: Read-only

Data type: IEDMAccessTemplateCollection

The returned object describes a collection of nested Access Template for a given Access Template. For more information, see IEDMAccessTemplateCollection. 

Links

Access: Read-only

Returns an object used to represent the IEDMATLinks interface. The returned object comprises information about permissions that have been specified by applying Access Template. For more information, see IEDMATLink.

PermissionEntriesCount

Access: Read/Write

Data type: Long integer

Sets and returns a number of Permission Entries the Access Template includes.
