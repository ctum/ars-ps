IEDMLookupInfoList::_NewEnum

Retrieves an interface on an enumerator object.

Note:

In Visual Basic and VBScript, the For Each …In … Next statement invokes this method implicitly.

Applying the For Each …In … Next statement to an IEDMLookupInfoList object allows you to enumerate objects returned by the LookupGUIDs, LookupSIDs or LookupSIDsOnComputer methods of the IEDMLookup interface.
