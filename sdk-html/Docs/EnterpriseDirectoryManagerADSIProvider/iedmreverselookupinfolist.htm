IEDMReverseLookupInfoList

This interface allows you to enumerate objects found with the RevesreLookup method of the IEDMLookup interface.

Properties

The IEDMReverseLookupInfoList interface has the following properties.

Property

Description

Count

Returns a number of objects found with the RevesreLookup method of the IEDMLookup interface.

Methods

The IEDMReverseLookupInfoList interface defines the following methods.

Method

Description

_NewEnum

Alows you to enumerate objects returned by the RevesreLookup method of the IEDMLookup interface.
