Creating Policy Object Container

File name: CreateContainer.vbs

This code snippet illustrates how to create a Policy Object container.

' Bind to the rootDSE
Set rootDSE = GetObject("EDMS://rootDSE")
rootDSE.GetInfoEx Array("edsvaAdministrationPoliciesDN"),0
' Get DN of the container for Policy Objects.
contDN = rootDSE.Get("edsvaAdministrationPoliciesDN")
Set ParentContainer=GetObject("EDMS://" & contDN)
Set poContainerObject = ParentContainer.Create("edsPolicyObjectsContainer", "CN=Restrict Policies")
poContainerObject.SetInfo

This script creates the "Restrict Policies" Policy Object container.
