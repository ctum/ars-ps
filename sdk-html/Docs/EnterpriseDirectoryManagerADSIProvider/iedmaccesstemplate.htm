IEDMAccessTemplate

The IEDMAccessTemplate interface is designed to manage Access Templates. The interface allows you to do the following:

  • Get a list of Access Template Links for Access Template

  • Create a new Permission Entry and add it to Access Template

  • Remove a Permission Entry from Access Template

  • List all Permission Entries that the Access Template includes

Methods

IEDMAccessTemplate defines the following methods:

Method

Description

_NewEnum

Allows you to obtain a collection of objects used to represent the IEDMComplianceInfoRecord interface. 

AddPermissionEntry

Adds a Permission Entry to Access Template.

CreatePermissionEntry

Creates a new Permission Entry.

RemovePermissionEntry

Removes the specified Permission Entry from Access Template.

Properties

The IEDMAccessTemplate interface has the following properties:

Property

Description

ID

Returns the GUID of the Access Template object.

Links

Allows you to get information about permissions that have been specified by applying Access Template.

NestedTemplateCollection

Returns an object used to represent the IEDMAccessTemplateCollection interface.

PermissionEntriesCount

Sets and returns a number of Permission Entries the Access Template includes.
