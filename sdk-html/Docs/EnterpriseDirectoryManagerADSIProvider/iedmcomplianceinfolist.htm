IEDMComplianceInfoList

The IEDMComplianceInfoList interface enables you to process an object returned by the CheckPropertyValues method of IEDM. The methods provided by this interface allow you to get the information on:

  • Results of checking administrative policy on the specified object's attribute

  • Results of checking administrative policy on all object's attributes

Methods

The IEDMComplianceInfo interface defines the following methods:

Method

Description

_NewEnum

Allows you to obtain a collection of objects used to represent the IEDMComplianceInfo interface. 

Item

Returns an object used to represent the IEDMComplianceInfo interface 

Example Code (VBScript)

The following  code snippet illustrates how to use the properties and methods provided by the IEDMComplianceInfoList, IEDMComplianceInfoRecord and IEDMComplianceInfo interfaces to get information about violations of the administrative policy affected the specified user object.

Dim usr,strUser,Policy, PP, PP1,PP2
str="Violations of the administrative policy:" + vbCrLf
strUser="EDMS://CN=Test,OU=Sales,DC=Mycompany,DC=com"
Set usr=GetObject(strUser)
usr.Put "l", "Berlin"
usr.Put "department", "Berlin Software Department"
usr.Put "co", "Germany"
' Check whether values of the user attributes are in compliance with the administrative policy
set PP = usr.CheckPropertyValues
' An implicit call of _NewEnum of IEDMComplianceInfoList
For Each PP1 in PP
   ' An implicit call of _NewEnum of IEDMComplianceInfo
   For Each PP2 in PP1
      str = str + "Attribute name: " + PP2.PropertyName + " " + "Policy checking status: " + Cstr(PP2.Status) + _
      + " " + PP2.Message + vbCrLf
   Next
Next
MsgBox str

The script lists attributes of the user Test whose values are not in compliance with the administrative policy.
