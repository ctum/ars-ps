edsClientSession Class

Contains information about a client connection to Administration Service.

edsaClientProtocol 

Optional.

Syntax: Integer, SingleValued

Searchable: Yes.

Identifies the version of the Active Roles Administration Service's client. This attribute contains &H40000 for the client's versions 4.x or &H50000 for versions 5.x.

 

edsaXMLDCsForDomains

Optional.

Syntax: CaseIgnoreString, SingleValued

Not used.

 

edsvaAccountName

Optional.

Syntax: DirectoryString, SingleValued

Searchable: No.

Specifies the domain user account Administration Service uses to access the managed domain.
