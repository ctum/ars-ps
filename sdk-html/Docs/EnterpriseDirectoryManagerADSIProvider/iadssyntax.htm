IADsSyntax

The IADsSyntax interface specifies methods to identify and modify the available Automation data types used to represent its data.

Use the IADsSyntax interface to process the property values of any instance of the Active Roles Schema class object.

IADsSyntax Properties

The property methods of the IADsSyntax interface get or set the properties described in the following table.

Property

Description

OleAutoDataType

Access: Read/Write

Data Type: Long

Gets and sets a LONG that contains the value of the  VT_xxx constant for the Automation data type that represents this syntax. For more information, refer to the MSDN Library.
