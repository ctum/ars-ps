IEDM::ExecuteRequest

The ExecuteRequest method performs the requested operation and returns a response the Administration Service sends after performing the operation. The request and response are saved in an XML format.

You can use the ExecuteRequest method to monitor a status of operations requiring approval.  For more informations, refer to Monitoring Operations Requiring Approval.

Function  ExecuteRequest([ByVal strXMLRequest])

Arguments

strXMLRequest

[in] Required. An XML string that contains a request to perform a desired operation.

Remarks

The syntax of the string strXMLRequest depends on the request type. For more information, refer to XML Schema Reference.
