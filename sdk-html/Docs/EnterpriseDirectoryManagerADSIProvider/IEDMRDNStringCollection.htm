IEDMRDNStringCollection

The IEDMStringCollection interface is designed to manage a collection of RDN objects a distinguished name string includes. Each RDN object implements the IEDMRDNString interface.

Properties

The IEDMRDNStringCollection interface has the following properties:

Property

Description

Count

Returns a number of elements in the RDN collection.

Methods

The IEDMRDNStringCollection interface defines the following methods:

Method

Description

AddAtBegin

Adds a new RDN element at the first position in RDN collection.

AddAtEnd

Appends a new RDN element to the end of RDN collection.

InsertAt

Inserts a new RDN element into the RDN collection at the specified index position.

RemoveAt

Removes the RDN element at the specified index of RDN collection.

RemoveRange

Removes a range of elements from RDN collection.

Clear

Removes all elements from RDN collection.

GetAt

Returns the RDN element at the specified index.

SetAt

Replaces the existing RDN element at the specified index with another element.

Example Code (VBScript)

The following code snippet illustrates how to use properties and methods defined in the IEDMRDNStringCollection interface.

Set objDN = CreateObject("AelitaEDM.EDMDistinguishedName")
 ' Define the DN string for User
objDN.SetDNString("CN=John Smith,OU=Sales,OU=London,OU=Locations,DC=mycompany,DC=com")
' --- Get the number of the RDN elements ---
NumberElements = objDN.Count
 ' --- Get the user city and department ---
strCity = objDN.RDNItems.GetAt(2).Value
strDept = objDN.RDNItems.GetAt(1).Value
'
