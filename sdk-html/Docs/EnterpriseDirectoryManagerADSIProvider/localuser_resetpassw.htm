Resetting a Local User Password

File name: UserResetPass.vbs

This code snippet illustrates how to reset the password of the specified local user.

option explicit
const strLocalUser = "EDMS://CN=Administrator,CN=Local Users,cn=mycomputer$,cn=MYCOMPANY,CN=Network"
const strNewPass = "star995polo815"
Dim objLocalUser
Set objLocalUser=GetObject(strLocalUser)
objLocalUser.Put "edsaLocalUserPassword", strNewPass
objLocalUser.SetInfo

This script resets the password for the local Administrator of the computer mycomputer in the domain mycompany.com. The new password is set to star995polo815.
