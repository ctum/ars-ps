IADsPropertyValue

The IADsPropertyValue interface is used to represent in a property value of a property entry in the cache in one of the pre-defined data types. This interface exposes several properties for rendering data values in the corresponding data format. For example, the LargeInteger property allows you to obtain a number of LargeInteger type.

A property value is held by a property entry. You can call the Values property method to obtain the property values of an ADSI object. 

Note:

Before calling the methods of this interface, you should call the GetInfo or GetInfoEx method of IADs to load the assigned property values of the ADSI object from the Active Roles Administration Service into the property cache. After calling the methods of IADsPropertyValue, you should call the SetInfo method of IADs to send all changes in the persistent store to the Active Roles Administration Service.

IADsPropertyValue Properties

The property methods of the IADsPropertyValue interface provide access to the properties described in the following table.

Property

Description

ADsType

Access: Read only

DataType: LONG

The data type of the value of the property, taken from the ADSTYPEENUM enumeration, of the value property.

DNString

Access: Read only

DataType: BSTR

String identifying the distinguished name (path) of a directory service value object.

CaseExactString

Access: Read only

DataType: BSTR

String to be interpreted case-sensitively.

CaseIgnoreString

Access: Read only

DataType: BSTR

String to be interpreted without regard to case.

PrintableString

Access: Read only

DataType: BSTR

Displayable or printable string.

NumericString

Access: Read only

DataType: BSTR

Text to be interpreted as numeral.

Boolean

Access: Read only

DataType: LONG

Boolean value.

Integer

Access: Read only

DataType: LONG

Integer value.

Note:

For more information about the IADsPropertyValue interface and for examples that illustrate the use of this interface, refer to the MSDN Library.
