IEDMDistinguishedName

Active Roles defines the Automation object AelitaEDM.EDMDistinguishedName that supports the IEDMDistinguishedName interface. This interface defines properties and methods that help you process distinguished name strings. A distinguished name (DN) is a sequence of relative distinguished names (RDN) connected by commas.

To use this interface, you must first create an instance of the object AelitaEDM.EDMDistinguishedName with the function CreateObject, as shown in the following example: 

set objTimeConv = CreateObject("AelitaEDM.EDMDistinguishedName")

Properties

The IEDMDistinguishedName interface has the following properties:

Property

Description

RDNItems

Returns a collection of Relative Distinguished Names (RDNs) the distinguished name string includes.

Methods

The IEDMDistinguishedName interface defines the following methods:

Method

Description

SetDNString

Sets a distinguished name (DN) string you want to process.

SetADsPathString

Sets an AdsPath you want to process.

GetDNString

Returns the DN string defined in the AelitaEDM.EDMDistinguishedName object. The returned string contains the reserved characters escaped, if any.

GetDNStringForADSIBind

Returns the DN string defined in the AelitaEDM.EDMDistinguishedName object. The returned string can be used to compose ADSI Provider binding strings.

GetDNStringForLDAPQuery

Returns the DN string defined in the AelitaEDM.EDMDistinguishedName object. The returned string can be used to compose LDAP queries.

GetParentDNString

Returns the distinguished name of parent of the object defined in the AelitaEDM.EDMDistinguishedName object.

GetCanonicalNameString

Returns the canonical name of the object defined in the AelitaEDM.EDMDistinguishedName object.

CreateRDN

Returns a reference to an IEDMRDNString type object.

Example Code (VBScript)

The following code snippet illustrates how to use properties and methods defined in the IEDM DistinguishedNameinterface.

Set objDN = CreateObject("AelitaEDM.EDMDistinguishedName")
 ' Bind to an OU
Set objOU = GetObject("EDMS://OU=Sales,OU=London,OU=Locations,DC=mycompany,DC=com")
 ' Create a DN string
objDN.SetADsPathString(objOU.ADsPath)
 ' Specify a new name that contains reserved characters
strName = "<Super,New=>"
 ' Create a new RDN element
Set objRDN = objDN.CreateRDN("OU", strName)
 ' Add the newly created RDN element at the first position in RDN collection
objDN.RDNItems.AddAtBegin objRDN
 ' Get the DN string used to compose the ADSI provider binding string.
 ' The DN string contains the reserved characters escaped, if any.
strNewDN = objDN.GetDNStringForADSIBind()
 ' Bind to the above specified OU
Set objOU2 = GetObject("EDMS://" & strNewDN)
 '
 ' Your code goes here
 '
