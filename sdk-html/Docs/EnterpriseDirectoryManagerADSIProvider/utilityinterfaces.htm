Utility Interfaces

The utility interfaces provide advanced helper functions for managing directory objects. This section describes the following utility Active Roles ADSI interfaces:

  • IEDMLargeInteger

  • IEDMOctetString

  • IEDMDistinguishedName

  • IEDMRDNString

  • IEDMRDNStringCollection

  • IEMDTimeConversion
