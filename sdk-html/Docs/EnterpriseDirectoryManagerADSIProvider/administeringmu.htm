Administering Managed Units

This section contains the VBScript code samples that demonstrate how to administer Managed Unit (MU) with the Active Roles ADSI Provider. You can use these examples to perform the following tasks on Managed Units:

  • Create

  • Add Rules

  • Create MU Container

  • List

To use the scripts, copy the appropriate script code, paste it into Notepad or another text editor, and then save the file with a .vbs file extension. You can find these examples as separate .vbs files in the folder '...\SDK\Samples\ADSI Provider\Managed Unit'.
