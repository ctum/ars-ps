IEDMLargeInteger::GetIADsLargeInteger

The GetIADsLargeInteger method returns a reference to an object of the IADsLargeInteger type representing the value defined in the current instance of the AelitaEDM.EDMLargeInteger object. For information about the IADsLargeInteger interface, refer to the MSDN library.

Function  GetIADsLargeInteger()

Arguments

This method requires no arguments.

Example Code (VBScript)

The following code snippet illustrates how to use the GetIADsLargeInteger method to set the expiration date for the user account.  The account of the John Doe user will expire in 180 days from the current date.

 ' Set DN of the user object
strObjDN = "CN=John S. Dow,OU=Sales,DC=mycompany,DC=com"
set objLargeInteger = CreateObject("AelitaEDM.EDMLargeInteger")
' Specify the account expiration date in 6 months from current date
ExpirationDate = DateAdd("m", 6, Now)
objLargeInteger.SetDate(ExpirationDate)
' Get a reference to the IADsLargeInteger object
set objIADsLargeInteger = objLargeInteger.GetIADsLargeInteger
' Bind to the user object
Set obj = GetObject("EDMS://" + strObjDN)
' Set the account expiration date for the user account
obj.Put "accountExpires", objIADsLargeInteger
obj.SetInfo
' Process the accountExpires attribute
objLargeInteger.Set obj.Get("accountExpires")
MsgBox "The user account expires on : " + CStr(objLargeInteger.GetDate)
