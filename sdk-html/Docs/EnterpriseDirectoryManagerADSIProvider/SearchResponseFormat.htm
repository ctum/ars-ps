OperationSearchResponse Format

This topic provides information about the format of a response the Administration Service returns after performing a requested operation. You can get the service response with the ExecuteRequest method of the IEDM interface. You can also get the service response using the Active Roles built-in control XmlOperation.

The response is in an Extensible Markup Language (XML) format. XML elements are listed below.

XML Elements

Operation

Describes the requested operation.

Parent Elements: None

Child Elements: Initiator, Target, Response, Request

Attributes:

Name

Description

creationTime

Required. dateTime.

UTC date and time of the operation initiation

guid

Required. string.

Operation GUID.

modificationTime

Required. dateTime.

UTC date and time of the of the last update to the stored operation.

id

Required. string.

Contains the operation ID.

status

Required. string.

Contains the completion status for the operation. The following values are possible:

  • Completed

  • Pending

  • Denied

  • Canceled

  • Failed

type

Required. string.

Identifies the operation type. The following values are possible:

  • CreateObject

  • ModifyObject

  • DeleteObject

  • Deprovision

  • MoveObject

  • CopyObject

xmlns

Required. string.

Declares a namespace for tags used in the XML request. This attribute must be set to "urn:schemas-quest-com:ActiveRolesServer"
 
Note
 

The Operation element uses the namespace declarator xmlns. The declarator must be set to "urn:schemas-quest-com:ActiveRolesServer"

Initiator

Provides information about the operation initiator.

Parent Elements: Operation

Child Elements: None

Attributes:

Name

Description

computer

Optional. string.

Contains the domain name of the initiator's computer.

dn

Optional. string.

Contains the initiator's distinguished name.

guid

Optional. string.

Contains the initiator's GUID.

isadmin

Optional. boolean.

Specifies, whether the operation initiator is an administrator.

ntAccount

Optional. string.

Contains the pre-Windows 2000 logon name for the initiator in the following format:
<Domain name>\<User name>

objectClass

Required. string.

Contains the initiator's object class.

sid

Optional. string.

Contains the initiator's SID.

Target

Provides information about the directory object (target object) on which the operation was performed.

Parent Elements: Operation

Child Elements: None

Attributes:

Name

Description

dn

Required. string.

Contains the target objects's distinguished name.

guid

Required. string.

Contains the target object's GUID.

objectClass

Required. string.

Contains the target object's class.

sid

Required. string.

Contains the target object's SID.

 

Request

Contains the stored request in an XML format.

Parent Elements: Operation

Child Elements: Any

 

Response

Contains the stored response in an XML format.

Parent Elements: Operation

Child Elements: Any

 

 

Important:

All names of attributes are case sensitive.
