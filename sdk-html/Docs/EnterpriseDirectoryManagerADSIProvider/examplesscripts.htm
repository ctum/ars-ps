Sample Scripts

This section provides examples of the VBScript code snippets that will help you develop custom scripts designed to perform the following tasks:

  • Managing AD Objects

  • Managing Active Roles-Specific Objects

  • Managing Computer Resources

  • Managing Active Roles Configuration

  • Monitoring Operations Requiring Approval

Note:

The scripts described in this section connect to the Active Roles Administration Service running on the local computer where you want the script to start. For information about how to have a script connect to the Administration Services running on a remote computer, see Connecting to a Service.
