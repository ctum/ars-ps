IEDMPolicyEntry

The IEDMPolicyEntry interface is designed to manage Policy Object Entries (Policies) a Policy Object includes. The interface allows you to do the following:

  • Get and set properties of Policy

  • Create, add and remove parameters of Policy

Properties

The IEDMPolicyEntry interface has the following properties.

Property

Description

GUID

Returns the GUID of a Policy.

Name

Sets and returns the Policy name. The name is displayed on the Policies tab of the Properties dialog box for the appropriate Policy Object.

ErrorMessage

Sets and returns the message displayed to the Active Roles clients when the administrative policy fails.

Type

Sets and returns the Policy type. For more information, refer to Types of Administrative Policies.

PolicySettingsCount

Returns a number of parameters defined for Policy.

Item

Retrieves an IEDMPolicySetting instance by index.

Methods

The IEDMPolicyEntry interface defines the following methods.

Method

Description

_NewEnum

Allows you to enumerate settings defined for Policy.

CreatePolicySetting

Creates a new parameter of Policy with no settings.

AddPolicySetting

Adds a parameter to Policy.

RemovePolicySetting (Deprecated)

Removes the specified parameter from Policy.

RemovePolicySettingByName

Removes the specified parameter from Policy.

Example Code (VBScript)

The following code snippet illustrates how to create a Policy Object and add a Policy to a newly created Policy Object . 

Dim cont
Dim objPO
' Bind to the rootDSE object
Set rootDSE = GetObject("EDMS://RootDSE")
rootDSE.GetInfoEx Array("edsvaAdministrationPoliciesDN"),0
' Get DN of the container for Policy Objects
contDN=rootDSE.Get("edsvaAdministrationPoliciesDN")
' Bind to the container within which a Policy Object
Set cont = GetObject("EDMS://" & contDN)
' Invoke the Create method of IADsContainer
Set objPO = cont.Create("edsPolicyObject", "CN=My_Policy_Object")
' Set properties of the newly created PO
objPO.Description="A sample PO"
' Create a Policy Entry
Set APE=objPO.CreatePolicyEntry
' Set the Policy Entry properties
' Type=35 corresponds to Property Validation Policy
APE.Type=35
APE.Name="Property Validation Policy"
' Create a new parameter
set Param=APE.CreatePolicySetting
' SettingID=1 corresponds to the Error Message parameter
Param.SettingID=1
Param.Value="The Administrative policy failed..."
' Add the newly created parameter to the Policy Object Entry
APE.AddPolicySetting(Param)
' Add the Policy Entry to the newly created PO
objPO.AddPolicyEntry(APE)
' Send the changes you made to Active Roles Administration Service
objPO.SetInfo

For more examples of the VBScript code samples, please refer to Managing Policy Objects.
