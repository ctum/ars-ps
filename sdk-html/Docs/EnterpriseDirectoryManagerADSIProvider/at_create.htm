Creating Access Template

File name: Create.vbs

This code snippet illustrates how to create Access Template.

Set ContainerObject=GetObject("EDMS://CN=Access Templates,cn=Configuration")
Set atObject = ContainerObject.Create("edsAccessTemplate", "CN=TestAT")
atObject.SetInfo

This script creates the TestAT Access Template.
