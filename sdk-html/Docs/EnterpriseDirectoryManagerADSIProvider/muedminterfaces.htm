Managed Unit and Dynamic Group Interfaces

These interfaces provide methods for administering Managed Units and dynamic groups. They consist of the following interfaces:

  • IEDMManagedUnit

  • IEDMMembershipRule

  • IEDMMembershipRuleCollection

  • IEDMDynamicGroup
