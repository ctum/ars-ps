edsReplicationPartner Class

Stores information about the replication partners of the Active Roles replication group.

edsaAuthenticationMode

Optional.

Syntax: Integer, SingleValued 

Searchable: Yes.

Specifies the authentication mode the Administration Service uses when accessing SQL Server: 0 - SQL Server authentication; 1 - Windows authentication.

 

edsaDatabaseName

Optional.

Syntax: DirectoryString, SingleValued 

Searchable: Yes.

Contains the name of the Administration database.

 

edsaDistributionDatabaseFolder

Optional.

Syntax: DirectoryString, SingleValued 

Searchable: No.

Specifies the path to the folder that contains the distribution database file.

 

edsaDistributionLogFolder

Optional.

Syntax: DirectoryString, SingleValued 

Searchable: No.

Specifies the path to the folder for the distribution log file.

 

edsaDRPSyncPeriod 

Optional.

Syntax: Integer, SingleValued 

Searchable: Yes.

Specifies how often (in minutes) the Active Roles Administration Service selects the DirSync server or tries to connect to the managed domains that were unavailable at the service start time. This value is determined by the DirSync Server Selection settings. You cannot change the attribute value using the Active Roles snap-in.

 

edsaEdmServiceComputerName 

Optional.

Syntax: CaseIgnoreString, SingleValued 

Searchable: Yes.

Contains the DNS name of the computer running the Active Roles Administration Service.

 

edsaEdmServiceLastAccess 

Optional.

Syntax: UTCTime, SingleValued 

Searchable: Yes.

Contains the date and time the Active Roles configuration database was last accessed.

 

edsaEdmServiceOwnershipPeriod 

Optional.

Syntax: Integer, SingleValued 

Searchable: Yes.

Specifies the time period (in minutes). If the Administration Service, specified in the attribute edsaEdmServiceComputerName, did not access its database within that time period, the database can be used by another Administration Service.

 

edsaErrorCode 

Optional.

Syntax: Integer, SingleValued 

Searchable: No.

Contains the error code returned by the last replication-related operation.

 

edsaErrorMessage 

Optional.

Syntax: CaseIgnoreString, SingleValued 

Searchable: No.

Contains the message describing the result of the last replication-related operation.

 

edsaIsCurrentServer 

Optional.

Syntax: Boolean, SingleValued 

Searchable: No.

When TRUE, the request is run on the current Administration service.

 

edsaLastActionTime 

Optional.

Syntax: UTCTime, SingleValued 

Searchable: No.

Contains the time the last replication-related operation was performed.

 

edsaNetworkName 

Optional.

Syntax: DirectoryString, SingleValued 

Searchable: Yes.

Specifies the network name of the computer running the Active Roles Administration Service.

 

edsaPublicationGuidString 

Optional.

Syntax: DirectoryString, SingleValued

The GUID of publication that uniquely identifies Replication Group.

 

edsaPublicKey 

Optional.

Syntax: OctetString, SingleValued 

Searchable: Yes.

Public key of a replication partner. This key is used to encrypt the replication group session key stored in the attribute edsaReplicationGroupSessionKeyEnc.

 

edsaReplicationGroupSessionKeyEnc 

Optional.

Syntax: OctetString, SingleValued 

Searchable: Yes.

Contains the encrypted replication group session key used by replication partners for encrypting the sensitive data.

 

edsaReplicationRole 

Optional.

Syntax: Integer, SingleValued 

Searchable: Yes.

Specifies the Administration service role in replication group: 1 - Publisher; 2 - Subscriber; 3 - standalone.

 

edsaReplicationServerAlias
