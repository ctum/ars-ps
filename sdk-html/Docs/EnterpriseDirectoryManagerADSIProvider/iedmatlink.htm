IEDMATLink

The IEDMATLink interface is designed to manage the Access Template Link objects. The interface allows you to do the following:

  • Specify a Trustee, Access Template and Securable Object for Access Template Link

  • Specify whether to propagate permissions that Active Roles assigns to Trustee to Active Directory

Properties

The IEDMATLink interface has the following properties.

Property

Description

AccessTemplateADsPath

Sets and returns the distinguished name of Access Template used to specify permissions.

AccessTemplateName

Returns the relative distinguished name (RDN) of Access Template used to specify permissions.

SecurableObjectADsPath

Sets and returns the distinguished name of a securable object (a directory object to which Access Template is linked).

Synchronizable

Specifies the synchronization status of Access Template Link.

TrusteeADsPath

Sets and returns the distinguished name of Trustee (a user or group to whom permissions are assigned for a securable object).

TrusteeName

Sets and returns the Trustee logon name (pre-Windows 2000) in the following format: DomainName\Logon name; for example: mycompany.com\JSmith.

IsInherited

Indicates whether Access Template Link is applied to a securable object itself (FALSE) or to its ancestors (parent objects) up the directory tree (TRUE).

AccessTemplateGuid

Sets and returns the GUID of Access Template used to specify permissions.

AccessTemplateGuidAsString

Sets and returns the GUID (as a string) of Access Template used to specify permissions.

SecurableObjectGuid

Sets and returns the GUID of a securable object.

SecurableObjectGuidAsString

Sets and returns the GUID (as a string) of a securable object.

TrusteeSid

Sets and returns the SID of Trustee.

TrusteeSidAsString

Sets and returns the SID of Trustee as a string.

Example Code (VBScript)

This code snippet illustrates how to create an Access Template Link and set its properties.

option explicit
 ' PARAMETERS OF SCRIPT YOU MAY MODIFY
 ' -----------------------------------
 '
 ' DN of Access Template
const strAccessTemplateDN = "EDMS://CN=TestAT,CN=Access Templates,cn=Configuration"
 ' DN of securable object
const strObjectToSecure = "EDMS://DC=mycompany,DC=com"
 ' DN of Trustee
const strTrusteeObject = "EDMS://CN=Everyone,CN=WellKnown Security Principals,CN=Configuration,DC=mycompany,DC=com"
 ' Access template link name - must be unique
const strACEName = "cn=TestATLink"
 ' PLEASE, DO NOT MODIFY THE SCRIPT BELOW THIS POINT
 ' -------------------------------------------------
 '
const strACEsContainerDN = "EDMS://cn=AT Links,cn=Configuration"
Dim ContainerObject
Set ContainerObject = GetObject(strACEsContainerDN)
 ' Create AT Link object
Dim atLinkObject
Set atLinkObject = ContainerObject.Create("edsACE", strACEName)
' Set properties of a newly created AT Link
atLinkObject.TrusteeADsPath = strTrusteeObject
atLinkObject.AccessTemplateADsPath = strAccessTemplateDN
atLinkObject.SecurableObjectADsPath = strObjectToSecure
 ' Commit changes
atLinkObject.SetInfo

For more examples of the VBScript codes, refer to Managing Access Templates.
