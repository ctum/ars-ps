IADsGroup

The IADsGroup interface is designed to represent and manage group membership information. It allows you to retrieve member objects, to test if a given object belongs to the group, and to add (or remove) an object to (or from) the group. 

IADsGroup is a dual interface that inherits from IADs.

IADsGroup Methods

The IADsGroup interface defines the following methods that can be used by Automation (Visual Basic, VBScript, JScript) clients.

IADsGroup Methods

Description

Members

Gets the IADsMembers interface on an ADSI collection object that represents the members of this group.

IsMember

Tests for membership. The method determines if an object is a member of the group representing the object implementing this interface.

Add

Adds an object to a group.

Remove

Removes the specified object from the group. The operation does not remove the group object itself even when there is no member left in the group.

IADsGroup Properties

The property methods of the IADsGroup interface get or set the properties described in the following table.

Property

Description

Description

Access: Read /Write

Data Type: BSTR

Indicates the textual description of the group membership.

Sample Code (Visual Basic)

The following code snippet binds to the Administrators group and outputs its description.

Dim grp As IADsGroup
Set grp = GetObject("EDMS://CN=Administrators,CN=Builtin,DC=mycompany,DC=com")
Debug.Print grp.Description
