Creating Script Module

File name: ImportScript.vbs

This code snippet illustrates how to create a policy script module and import an existing script to that module.

Set FSO = CreateObject("Scripting.FileSystemObject")
Set ScriptFile = FSO.OpenTextFile("MyScript.vbs", 1)
Set ContainerObject=GetObject("EDMS://CN=Script Modules,cn=Configuration")
Set newScriptModule=ContainerObject.Create("edsScriptModule", "CN=TestScriptModule")
newScriptModule.put "edsaScriptLanguage", "VBScript"
newScriptModule.put "edsaScriptText", ScriptFile.ReadAll
 ' Set the script module type: 0 - policy script; 1 - scheduled task script; 2 - library script
newScriptModule.put "edsaScriptType", 0
newScriptModule.SetInfo

The script creates the script module TestScriptModule, and then import the VBScript saved in the file MyScript.vbs to that module.
