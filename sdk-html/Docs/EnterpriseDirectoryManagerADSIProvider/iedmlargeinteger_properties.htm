IEDMLargeInteger Properties

The following table describes properties provided by the IEDMLargeInteger interface.

Property

Description

HighPart

Access: Read\Write

Data type: Integer

Sets and returns the upper 32 bits of the large integer.

LowPart

Access: Read\Write

Data type: Integer

Sets and returns the lower 32 bits of the large integer.

Remarks

Handling the IEDMLargeInteger in VBScript is made difficult by the fact that VBScript has no native unsigned numeric data type. This can cause errors in data conversion if either the LowPart or HighPart has the high bit set, which causes VBScript to handle the number as negative. The VBScript sample code below shows how to properly handle the IIEDMLargeInteger in VBScript 

Example Code (VBScript)

The following sample code illustrates how to properly convert the accountExpires property of a user account to a hexadecimal string.

strObjUserDN = "EDMS://CN=TestUser,OU=Sales,DC=mycompany,DC=com"
Set objUser = GetObject(strObjUserDN)
objUser.GetInfoEx Array("accountExpires"),0
Set oLargeInt = objUser.Get("accountExpires")
ss=""
intHigh= oLargeInt.HighPart
intLow=  oLargeInt.LowPart
ss = ss & "High part: " & Cstr(intHigh) & " Lower part: " &  Cstr(intLow) & vbNewLine & vbNewLine
ss = ss & "&H" & CStr(Hex(intHigh)) &  CStr(Hex(intLow))
MsgBox ss
