IEDMClass Properties

The following table describes properties provided by the IEDMClass interface.

Property

Description

SchemaObjectADsPath

Access: Read-only

Data type: String

Returns the ADs path to the classSchema object in the schema container.

SchemaObjectCN

Access: Read-only

Data type: String

Returns the common name of the classSchema object.
