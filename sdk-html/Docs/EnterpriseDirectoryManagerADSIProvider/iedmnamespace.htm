IEDMNamespace

The IEDMNamespace interface allows you to get information about a version of the Active Roles ADSI Provider installed on your local computer.

Properties

The IEDMNameSpace interface has the following properties.

Property

Description

VersionInfo

Returns an XML string that contains information about version of the Active Roles ADSI Provider installed on your local computer.

Example Code (VBScript)

The following code snippet illustrates how to get information about version of the Active Roles ADSI provider installed on the user's local computer.

set xmlDoc = CreateObject("Msxml2.DOMDocument")
xmlDoc.async = false
xmlDoc.resolveExternals = false
ss=""
Set NameSpace = GetObject("EDMS:")
xmlDoc.loadXML(Cstr(NameSpace.VersionInfo))
Set objNode = xmlDoc.selectSingleNode("//VersionInfo/FileVersion")
strVersion = objNode.text
MsgBox strVersion
