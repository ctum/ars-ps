IEDMAccessTemplateCollection Properties

The following table describes properties provided by the IEDMAccessTemplateCollection interface.

Property

Description

_NewEnum

Access: Read-only

Returns an interface on an enumerator object.

Note:

In Visual Basic and VBScript, the For Each …In … Next statement invokes this property implicitly.

Applying the For Each …In … Next statement to an IEDMAccessTemplateCollection object allows you to enumerate all nested Access Templates (AT) in the collection of ATs for the given AT.

Count

Access: Read-only

Data type: Long integer

Returns the number of Access Templates in the collection.

Ids

Access: Read-only

Data type: Array of strings

Returns an array of strings that contains the GUIDs of  the Access Templates in the collection.

IsModified

Access: Read-only

Data type: Boolean

Returns a Boolean that represents whether the Access Template collection has been changed since the last time the GetInfo/SetInfo method was called.
