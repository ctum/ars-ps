Completion Status for the Operation with a Specific ID

File name: GetOperationStatus.vbs

This script (VBScript) illustrates how to get the completion status for an operation specified by the operation ID. The script returns one of the following possible statuses: "Unknown", "Pending", "Completed", "Rejected", "Failed" or "Canceled."

Notes

  • Before using the script, set the constant OperationID to the ID of the operation of interest. For more information, see Getting the Operation ID.

  • Alternately, you can get the completion status for an operation using Active Roles Management Shell, see Using the Get-QARSOperation Cmdlet.

' Displays the status for the operation specified by the operation ID
' Possible status: "Unknown", "Pending", "Completed", "Rejected", "Failed" and "Canceled"
Option Explicit
'Const DomDocumentProgID = "Msxml2.DOMDocument.4.0" ' for MSXML 4.0
Const DomDocumentProgID = "Msxml2.DOMDocument.6.0" ' for MSXML 6.0
' Specify the operation ID 
Const Operation_ID = "1-209"
WScript.Echo GetOperationStatusByID(Operation_ID)
'--------------------------------------------------------------
Function GetOperationStatusByID(id)
 Dim xmlOperation
 xmlOperation = FindOperationByID(id)
 If xmlOperation <> "" Then GetOperationStatusByID = GetOperationStatus(xmlOperation)
 If GetOperationStatusByID = "" Then GetOperationStatusByID = "Unknown"
End Function
Function GetOperationStatus(xmlOperation)
 Dim xmlDoc, xmlNode
 Set xmlDoc = CreateObject(DomDocumentProgID)
 xmlDoc.loadXML(xmlOperation)
 xmlDoc.setProperty "SelectionNamespaces", "xmlns:a=""urn:schemas-quest-com:ActiveRolesServer"""
 Set xmlNode = xmlDoc.selectSingleNode("/a:Operation/@status")
 If Not (xmlNode Is Nothing) Then GetOperationStatus = xmlNode.value
 If GetOperationStatus = "" Then GetOperationStatus = "Unknown"
End Function
Function FindOperationByID(id)
 FindOperationByID = ""
 Dim xmlRequest, xmlResponse
 xmlRequest = "<OperationSearchRequest xmlns=""urn:schemas-quest-com:ActiveRolesServer"" pageSize=""1"">"
 xmlRequest = xmlRequest &  "<Filter operator=""EqualityMatch"" name=""OperationID"" "
 xmlRequest = xmlRequest &  "stringValue=""" & id & """/>"
 xmlRequest = xmlRequest &  "</OperationSearchRequest>"
 Dim edmsAD
 Set edmsAD = GetObject("EDMS://CN=Active Directory")
 xmlResponse = edmsAD.ExecuteRequest(xmlRequest)
 Dim xmlDoc
 Set xmlDoc = CreateObject(DomDocumentProgID)
 xmlDoc.loadXML(xmlResponse)
 xmlDoc.setProperty "SelectionNamespaces", "xmlns:a=""urn:schemas-quest-com:ActiveRolesServer"""
 Dim operationNode
 Set operationNode = xmlDoc.selectSingleNode("/a:OperationSearchResponse/a:Operations/a:Operation")
 If Not (operationNode Is Nothing) Then FindOperationByID = operationNode.xml
End Function

Getting the Operation ID

Once an operation (such as creation or modifying a directory object) is initiated with your custom script, you can get the operation ID using the Active Roles built-in control XMLOperation.

The following sample script illustrates how to modify the "department" property of the user John Doe, and then get the ID of that operation. This operation is suggested to be a subject to approval (see Understanding Workflows). 

Set usr = GetObject("EDMS://CN=John Doe,OU=Sales,DC=mycompany,DC=com")
usr.Control("ReturnXmlOperation") = "1"
usr.Control("AllowApproval")="Confirm"
usr.Put "department", "Sales"
usr.SetInfo
XMLOperation = usr.Control("XmlOperation")
Wscript.Echo  XMLOperation

The script outputs the XmlOperation control that contains an XML report on the requested operation. The report is similar to the following text:

To get the operation ID

  • In the report on the requested operation, consult the value of the attribute id for the element Operation. In this example, the operation ID is 1-212.

Using the Get-QARSOperation Cmdlet

To reduce the complexity of VBScript scripts used to get the completion status for an operation specified by the operation ID, you can use the Get-QARSOperation cmdlet provided by Active Roles Management Shell.

Let us suppose that the operation ID is 1-212. To get the completion status for this operation, open Active Roles Management Shell, and type the following command at the command prompt:

[PS] C:\PS>Get-QARSOperation -Proxy -OperationID 1-212 | %{$_.Status}

For information about how to install and open Active Roles Management Shell, refer to the Active Roles Management Shell Release Notes.
