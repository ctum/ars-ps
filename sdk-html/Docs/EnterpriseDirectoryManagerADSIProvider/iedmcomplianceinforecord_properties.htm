IEDMComplianceInfoRecord Properties

The following table describes properties provided by the IEDMComplianceInfoRecord interface.

Property

Description

Message

Access: Read-only

Data type: String

Returns the message that informs the Active Roles clients about results of checking the administrative policy.

PolicyEntryName

Access: Read-only

Data type: String

Returns the name of the policy entry identifying a policy in the Policy Object. The name is displayed on the Policies tab in the Properties dialog box of the appropriate Policy Object.

PolicyObjectGuid

Access: Read-only

Returns GUID of the Policy Object.

PropertyName

Access: Read-only

Data type: String

Returns the Ldap display name of the attribute whose value is checking.

Status

Access: Read-only

Data type: Long integer

Returns a status of the administrative policy checking. This property returns one of values from the EDS_POLICY_COMPLIANCE_ENUMenumeration.
