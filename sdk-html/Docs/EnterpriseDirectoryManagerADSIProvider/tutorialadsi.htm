Tutorial: ADSI Scripting

This tutorial consists of a set of topics that walk you through typical tasks you might face when managing directory objects.

The tutorial provides the VBScript code snippets that are intended to demonstrate how developers can implement the major features provided by the Active Roles ADSI provider to create the custom applications.

The tutorial focuses on the following tasks:

  • Binding to Active Directory Objects

  • Binding to Active Roles-Specific Objects

  • Accessing Properties of ADSI Objects

  • Modifying Properties of ADSI Objects

  • Searching for ADSI Objects

  • Managing Computer Resources

  • Sample Scripts
