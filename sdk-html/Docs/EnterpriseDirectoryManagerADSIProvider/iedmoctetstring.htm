IEDMOctetString

Active Roles defines the Automation object AelitaEDM.EDMOctetString that supports the IEDMOctetString interface. This interface defines properties and methods that help you create and process the binary AD attributes such as objectGUID, objectSid, etc. For example, with this interface you can generate GUID for directory objects you want to create.

To use this interface, you must first create an instance of the object AelitaEDM.EDMOctetString with the function CreateObject, as shown in the following example: 

set objOctetString = CreateObject("AelitaEDM.EDMOctetString")

Properties

The IEDMOctetString interface has the following properties.

Property

Description

Length

Sets and returns the length of the binary array (in bytes) processed with the object AelitaEDM.EDMOctetString. Note that the object AelitaEDM.EDMOctetString stores the binary value to be processed as an array of bytes.

Methods

The IEDMOctetString interface defines the following methods:

Method

Description

Set

Sets the binary value to be processed with the object AelitaEDM.EDMOctetString.

GetString

Returns a string representing the binary value defined in the object AelitaEDM.EDMOctetString.

GetOctetString

Returns an array of bytes representing the binary value defined in the object AelitaEDM.EDMOctetString.

GetSidString

Returns a string conforming to the syntax Sid (Syntax ID: 2.5.5.17).

SetSidString

Sets the binary value to be processed with the object AelitaEDM.EDMOctetString.

GetGuidString

Returns a string conforming to the syntax OctetString (Syntax ID: 2.5.5.10). 

SetGuidString

Sets the binary value to be processed with the object AelitaEDM.EDMOctetString.

GenerateGuid

Generates a new GUID and stores it in the object AelitaEDM.EDMOctetString.

Example Code (VBScript)

The following code snippet illustrates how to use the object AelitaEDM.EDMOctetString to output GUID and SID of the specified user object.

'Set the User object's DN
Const strDN="CN=Douglas P. Joyse,OU=Sales,DC=MyDomain,DC=local"
ss="Object: " + strDN + vbNewLine + vbNewLine
Set obj = GetObject("EDMS://" + strDN)
Set OctetString=CreateObject("AelitaEDM.EDMOctetString")
With OctetString
    .Set obj.Get("objectGUID")
    ss = ss + "GUID: " + .GetGuidString + vbNewLine
    ss = ss + "GUID length= " + Cstr(.Length) + " bytes" + vbNewLine
    .Set obj.Get("objectSid")
    ss = ss + "SID: " + .GetSidString + vbNewLine
    ss = ss + "SID length= " + Cstr(.Length) + " bytes" + vbNewLine
End With
MsgBox ss

For more examples of the code snippets using the IEDMOctetString interface, refer to Managing Access Templates, Managing Policy Objects.
