IEDMPolicyObject Properties

The following table describes properties provided by the IEDMPolicyObject interface.

Property

Description

Links

Access: Read-only

Data type: IEDMPOLinks

Returns an object used to represent the IEDMPOLinks interface.

PolicyEntriesCount

Access: Read-only

Data type: Long integer

Returns a number of Policies the Policy Object includes.

PolicyEntry

Access: Read-only

Data type: IEDMPolicyEntry

Retrieves an IEDMPolicyEntry instance by index. The PolicyEntry property is a read-only indexer that allows for accessing a Policy Entry by its GUID.

ObjPO.PolicyEntry(GUID)

Arguments

ObjPO
Required. Always a name of the processed Policy Object.
GUID
[in] Required. GUID. The GUID of the Policy Entry you want to retrieve, such as "6132fc40-e8ea-4997-8b18-8145bcef054c" or "6132fc40e8ea49978b188145bcef054c". For information about how to get the GUID of the Policy Entry, see "Getting GUID of Policy Entry" in Related Topics.
