IEDMAccessTemplateCollection::RemovebyID

The Add method removes an Access Template identified by its GUID from the collection of the nested Access Templates.

Sub  RemovebyID(ByVal GUID)

Arguments

GUID

[in] Required. The GUID of the Access Template to be removed.
