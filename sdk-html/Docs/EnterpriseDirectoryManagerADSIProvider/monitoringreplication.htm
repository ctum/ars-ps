Monitoring Active Roles Replication

This section contains the VBScript code samples that demonstrate how to monitor the Active Roles replication with the Active Roles ADSI Provider. You can use these examples to perform the following tasks:

  • Monitor the Replication Status

  • Notify Administrator When Failing the Replication

For more information about the Active Roles Replication function, refer to "Configuring Replication" in Active Roles Administrator's Guide.

To use the scripts, copy the appropriate script code, paste it into Notepad or another text editor, and then save the file with a .vbs file extension. You can find these examples as separate .vbs files in the folder '...\SDK\Samples\ADSI Provider\EDM Replication'.
