Creating Users

To create new directory objects, use the Children property of the DirectoryEntry class. The Children property returns a DirectoryEntries object that exposes the Add method. For more information about those classes and methods, refer to the MSDN library.

The following code example shows how to create a user in an organizational unit and enable the newly created user account. The variable strDN specifies the distinguished name of the Organizational Unit within which you want to create a new user account. 

[ C# ]
using System;
using System.DirectoryServices; 
using ActiveDs;
using EDMLib;
namespace CreateUser
{	class Class1
	{		
		[STAThread]
		static void Main(string[] args)
		{
           // Set the OU's name
		    string strDN = "EDMS://OU=MyOU,DC=mycompany,DC=com";
          			DirectoryEntry ou = new DirectoryEntry(strDN);
			 
			{
				DirectoryEntry usr = ou.Children.Add("CN=New User","user");
				// Set the samAccountName, then commit changes to the directory.
				usr.Properties["samAccountName"].Value = "newuser";
				usr.Properties["edsaAccountIsDisabled"].Value = 0;
				usr.CommitChanges();
				Console.WriteLine("New user has been created");
			}
		}
	}
}


[ Visual Basic .NET ]
Imports System.DirectoryServices
Module Module1
    Sub Main()
        ' Bind to the appropriate container, add a new user.
        Dim strDN As String = "EDMS://OU=MyOU,DC=mycompany,DC=com"
        Dim de As New DirectoryEntry(strDN)
        Dim newUser As DirectoryEntry = de.Children.Add("CN=New User", "user")
        newUser.Properties("samAccountName").Value = "newuser"
        newUser.Properties("edsaAccountIsDisabled").Value = 0
        newUser.CommitChanges()
        Console.WriteLine("New user has been created")
    End Sub
End Module

To use this example

  1. Start Visual Studio and create a new C# or Visual Basic project using the procedure outlined in Getting Started with Microsoft .NET Projects.

  2. Copy this sample code and then paste it into the Code Pane. Set the variable strDN to the distinguished name of the Organizational Unit within which you want to create a new user account.

  3. Compile and run the application.
