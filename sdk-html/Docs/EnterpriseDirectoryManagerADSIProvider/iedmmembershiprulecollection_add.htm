IEDMMembershipRuleCollection::Add

The Add method adds a rule to the membership rule collection.

Sub  Add(ByVal Rule)

Arguments

Rule

[in] Required. Identifies a rule created with the IEDMMembershipRule interface. This argument is of the IEDMMembershipRule type.
