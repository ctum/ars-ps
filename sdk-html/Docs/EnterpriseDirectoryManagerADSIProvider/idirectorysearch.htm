IDirectorySearch

The IDirectorySearch interface is a pure COM interface that provides a low overhead method that non-Automation clients can use to perform queries in the underlying directory. For more information about IDirectorySearch, refer to MSDN Library.

IDirectorySearch Methods

The IDirectorySearch interface defines the following methods. 

IDirectorySearch methods

Description

SetSearchPreference

Specifies a search preference for obtaining data in a subsequent search.

ExecuteSearch

Executes an individual search and passes the results to the caller.

AbandonSearch

Abandons a search initiated by an earlier call to the ExecuteSearch method.

GetFirstRow

Gets the first row of a search result. This method will issue or re-issue a new search, even if you have called this method before.

GetNextRow

Gets the next row of the search result. If the GetFirstRow method is never called before, this method will issue a new search beginning from the first row. Otherwise, this method will advance to the next row.

GetPreviousRow

Gets the previous row of the search result. If the provider does not provide cursor support, it should return E_NOTIMPL.

GetNextColumnName

Gets the name of the next column in the search result that contains data.

GetColumn

Gets the item in a specified column from the current row of the search result.

FreeColumn

Releases memory that the GetColumn method allocated for data for the column.

CloseSearchHandle

Closes the handle to a search result and frees the associated memory.
