IEMDTimeConversion::SetUTCTime

The SetUTCTime method sets the UTC time value you want to process with the AelitaEDM.EDMTimeConversion object.

Sub SetUTCTime(ByVal UTCTime)

Arguments

UTCTime

[in] Required. Specifies the value you want to set (always a Generalized-Time type string representing the date and time, such as "10/14/2009 6:47:53 PM").
