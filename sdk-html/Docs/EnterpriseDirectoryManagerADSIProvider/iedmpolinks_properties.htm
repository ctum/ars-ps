IEDMPOLinks Properties

The following table describes properties provided by the IEDMPOLinks interface.

Property

Description

_NewEnum

Access: Read-only

Returns an interface on an enumerator object.

Note:

In Visual Basic and VBScript, the For Each …In … Next statement invokes this property implicitly.

Applying the For Each …In … Next statement to an IEDMPOLinks object allows you to enumerate all Policy Object links (objects of the IEDMPOLink type) for a Policy Object.
