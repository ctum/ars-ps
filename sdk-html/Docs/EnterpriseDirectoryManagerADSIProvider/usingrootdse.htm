Binding and the rootDSE object

The rootDSE object is defined as the root of the data tree for the Active Roles Administration Service. The properties of the rootDSE object allow you to get distinguished names of the Active Roles schema and configuration namespace containers, and other data about the Administration Service. These properties rarely change on a particul Active Roles. An application can read these properties at startup and use them throughout the session.

We recommend that you use the rootDSE object to bind to the objects specific to Active Roles, such as Managed Units, Policy Objects, etc.

The following is the binding string that is used to bind to rootDSE: 

EDMS://<DomainDCName>/rootDSE

The <DomainDCName> specifies the name of the domain controller or domain where you want the provider to search for an available Active Roles Administration Service. The <DomainDCName> is optional, as shown in the following format. 

EDMS://rootDSE

For information about how the provider searches for Administration Service when the <DomainDCName> is omitted, see Connecting to Service.

To get the property value, use the Get or GetEx method of the IADs interface. Before using those methods, preload that property to the cache using the GetInfoEx method. The following code snippet illustrates how to get the multi-value property namingContexts. 

' Bind to the rootDSE
Set rootDSE = GetObject("EDMS://rootDSE")
ss=""
 ' Load the property namingContexts to the cache
rootDSE.GetInfoEx Array("namingContexts"),0
ss = ss + "namingContexts:" + vbNewLine
 ' Get the namingContexts value
zz=rootDSE.Get("namingContexts")
 ' Enumerate the namigContexts property
For Each s1 in zz
    ss = ss + s1 + vbNewLine
Next
MsgBox ss

Sample Code

The following sample code illustrates how to list all Managed Units found on the local instance of the Active Roles Administration Service. 

' Bind to the rootDSE
Set rootDSE = GetObject("EDMS://rootDSE")
ss=""
 ' Load the property edsvaManagedUnitsDN to the cache
rootDSE.GetInfoEx Array("edsvaManagedUnitsDN"),0
 ' Get the edsvaManagedUnitsDN value
strNC=rootDSE.Get("edsvaManagedUnitsDN")
 ' DN of the Managed Units container
strDN="EDMS://" + strNC
 ' Bind to the Managed Units container
set objCont = GetObject(strDN)
 ' List all Managed Units
For Each MU in objCont
    ss = ss + MU.Name + vbNewLine
Next
MsgBox ss
