IEDMDynamicGroup::ConvertToRegularGroup

The Sub procedure that allows you to convert a dynamic group to a basic group. 

Sub  ConvertToRegularGroup

Arguments

This method does not require arguments.

Remarks

After you call this method and the SetInfo method of the IADs interface, the IsDynamicGroup property is set automatically to FALSE. This also clears the Membershiprules property of the group.

When you convert a dynamic group to a basic group, the group retains all its members included due to the membership rules.

Example Code (VBScript)

The following code snippet illustrates how to use the ConvertToRegularGroup method. 

const strGroupDN = "EDMS://CN=TestGroup,OU=TestOU,DC=mycompany,DC=com"
Set objGroup = GetObject(strGroupDN)
objGroup.ConvertToRegularGroup
objGroup.SetInfo
MsgBox "Done!"

The script converts the group TestGroup to a basic group.
