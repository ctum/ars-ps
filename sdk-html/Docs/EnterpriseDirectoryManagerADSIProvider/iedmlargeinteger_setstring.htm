IEDMLargeInteger::SetString

The SetString method sets the LargeInteger type value that you want to process. 

Sub  SetString(ByVal strVal)

Arguments

strVal

[in] Required. Specifies the value you want to set  (always a string representing the large integer number, such as "9223372036854775807")
