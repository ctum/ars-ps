Finding Computer Accounts

File name: Search.vbs

This code snippet illustrates how to find the computers whose names start with V.

Set oConnect = CreateObject("ADODB.Connection")
oConnect.Provider = "ADsDSOObject"
oConnect.Open "DS Query", "", ""
Set Command = CreateObject("ADODB.Command")
Set Command.ActiveConnection = oConnect
Command.CommandText = "SELECT ADsPath FROM '" _
	+ "EDMS://DC=mycompany,DC=com" + "' WHERE objectClass = 'computer' AND "_
         + "Name = 'V*'"
Set rs = Command.Execute
Result=""
Do Until rs.EOF
   	Result=Result+rs.Fields("ADsPath").Value + vbNewLine
	rs.MoveNext
Loop
WScript.Echo Result
