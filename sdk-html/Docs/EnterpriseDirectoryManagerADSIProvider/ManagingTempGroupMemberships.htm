Managing Temporal Group Memberships

Active Roles provides for the temporal group memberships feature. By using this feature, you can manage group memberships of objects such as user or computer accounts that need to be members of particular groups for only a certain time period. For example, you can specify when an object is to be actually added to a group or removed from a group.

This topic explains how you can develop custom scripts designed to manage group memberships of objects. For more information about the temporal group memberships feature, refer to "Using Temporal Group Memberships" in Active Roles Help.

Adding Temporal Members to Group

Active Roles supports the ScheduledOperation-SetTime control. Using this control, you can schedule the time when an object should be actually added to a group or removed from a group.

The following sample code (VBScript) illustrates how to add a user account to a group, and then cause Active Roles to remove that user from the group at specific time.

Const strUserDN = "CN=John Doe,OU=Sales,DC=mycompany,DC=com"
Const EDS_PROPERTY_APPEND= 3
Const EDS_PROPERTY_DELETE= 4
Dim objGroup
' --- Bind to the group object
Set objGroup= GetObject ("EDMS://CN=SalesDept,OU=Groups,DC=mycompany,DC=com")
' --- Add the specified user to the group
' --- Setting the ScheduledOperation-SetTime control to "Now" causes
' --- Active Roles to immediately add the user to the group 
objGroup.Control("ScheduledOperation-SetTime") = "Now"
objGroup.PutEx EDS_PROPERTY_APPEND, "member", Array(strUserDN)
objGroup.SetInfo

' --- Issue a request to remove the user from the group at the specific time
' --- The user will be removed from the group on January 10, 2009 at 08:00 a.m. (GMT)
' --- The ScheduledOperation-SetTime control must conform to the GeneralizedTime
' --- syntax,
' --- such as the YYYY-MM-DDTHH:00:00Z format.
objGroup.Control("ScheduledOperation-SetTime") = "2009-01-10T08:00:00Z"
objGroup.PutEx EDS_PROPERTY_DELETE, "member", Array(strUserDN)
objGroup.SetInfo

This script immediately adds the John Doe user account to the SalesDept group as a temporal member that will be removed from the SalesDept group on January 10, 2009.

 

The following script illustrates how to add pending members to a group. The pending members are temporal members that are scheduled to be added to the group in the future but are not actual members of the group so far. The pending member can be scheduled to be removed from the group in the future.

Const strUserDN = "CN=John Doe,OU=Sales,DC=mycompany,DC=com"
Const EDS_PROPERTY_APPEND = 3
Const EDS_PROPERTY_DELETE = 4

Dim objGroup
' --- Bind to the group object
Set objGroup= GetObject ("EDMS://CN=SalesDept,OU=Groups,DC=mycompany,DC=com")

' --- Issue a request to add the user to the group at the specific time 
' --- The user will be added to the group on January 5, 2009 at 08:00 a.m. (GMT)
' --- The ScheduledOperation-SetTime control must conform to the GeneralizedTime
' --- syntax, 
' --- such as the YYYYMMDDHH0000.0Z format
objGroup.Control("ScheduledOperation-SetTime") = "20090105080000.0Z"
objGroup.PutEx EDS_PROPERTY_APPEND, "member", Array(strUserDN)
objGroup.SetInfo

' --- Issue a request to remove the user from the group at the specific time 
' --- The user will be removed from the group on January 10, 2009 at 08:00 a.m. (GMT)
' --- The ScheduledOperation-SetTime control must conform to the GeneralizedTime
' --- syntax,
' --- such as the YYYY-MM-DDTHH:00:00Z format
objGroup.Control("ScheduledOperation-SetTime") = "2009-01-10T08:00:00Z"
objGroup.PutEx EDS_PROPERTY_DELETE, "member", Array(strUserDN)
objGroup.SetInfo
 

This script adds the John Doe user to the SalesDept group as a pending member. Active Roles will add the user to the SalesDept group on January 5, 2009. The user will be automatically removed from the group on January 10, 2009.

Canceling a Scheduled Removing Operation

If you want to cancel the scheduled removing of the John Doe user account from the SalesDept group, use the following script:

 

Const strUserDN = "CN=John Doe,OU=Sales,DC=mycompany,DC=com"

Const EDS_PROPERTY_DELETE = 4

Dim objGroup

' --- Bind to the group object

Set objGroup= GetObject ("EDMS://CN=SalesDept,OU=Groups,DC=mycompany,DC=com")

objGroup.Control("ScheduledOperation-SetTime") = "Never"

objGroup.PutEx EDS_PROPERTY_DELETE, "member", Array(strUserDN)

objGroup.SetInfo
 

Listing the Group Members

The list of the group members includes both the objects that are actual members of the group and the objects that are scheduled to be added to the group in the future (so called pending members). Active Roles allows you to get the list of pending and temporal members using an ASQ search on the "member" attribute of the group. This section provides for the sample script illustrating how to get the list of the group members using the ASQ search method.

Important

When performing an ASQ search, you must set the ScheduledLink-GetPending and ScheduledLink-GetStartEndTime controls to any value. This will cause the search result set to include the pending and temporal members, if any.

 

 ' Set the group's DN
const strDN = "CN=SalesDept,OU=AllDepts,DC=mycompany,DC=com"
Set oConnect = CreateObject("ADODB.Connection")
oConnect.Provider = "ADsDSOObject"
oConnect.Open "DS Query", "", ""
Set Command = CreateObject("ADODB.Command")
Set Command.ActiveConnection = oConnect

 ' --- Specify the ASQ search string.
' --- The search string sets the ScheduledLink-GetPending and ScheduledLink-GetStartEndTime controls to 1.
Command.CommandText = "<EDMS://"  & strDN _
& "//EDS_SEARCHPREF_ATTRIBUTE_QUERY=member//ScheduledLink-GetPending=1//ScheduledLink-GetStartEndTime=1>;(objectClass=*);distinguishedName,edsva-ScheduledLink-StartTime,edsva-ScheduledLink-EndTime,AdsPath;base"
Dim strSS
strSS = "The group contains the following direct members (including pending and temporal members): " & vbNewLine & vbNewLine
' --- Start the search.
Set rs = Command.Execute
' --- Process the search results
Do Until rs.EOF
If (rs.Fields("edsva-ScheduledLink-StartTime").Value <> empty) Then
strSS = strSS & "Pending member: "
Else
If (rs.Fields("edsva-ScheduledLink-EndTime").Value <> empty) Then
strSS = strSS & "Temporal member: "
Else
strSS = strSS & "Permanent member: "
End IF
End IF
strSS = strSS & "'" & rs.Fields("distinguishedName").Value & "'"
If (rs.Fields("edsva-ScheduledLink-StartTime").Value <> empty) Then
strSS = strSS & " will be added on " & rs.Fields("edsva-ScheduledLink-StartTime").Value
End If
If (rs.Fields("edsva-ScheduledLink-EndTime").Value <> empty) Then
strSS = strSS & " will be deleted on " & rs.Fields("edsva-ScheduledLink-EndTime").Value
End If
strSS = strSS + vbNewLine + vbNewLine
rs.MoveNext
Loop
' --- Output the search results

WScript.Echo strSS

This script outputs the members list of the SalesDept group. The list includes the pending and temporal members, if any.
