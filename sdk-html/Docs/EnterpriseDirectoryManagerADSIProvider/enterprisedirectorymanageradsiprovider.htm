Active Roles ADSI Provider

The Active Roles ADSI Provider makes it possible to access and manipulate directory objects and computer resources through Active Roles using ADSI-compliant interfaces. The data exposed by the Active Roles ADSI Provider is organized in a namespace that is identical to the namespace of the Windows system LDAP provider. The name of the Active Roles ADSI Provider namespace is EDMS:.

The Active Roles ADSI Provider enables developers to use standard Microsoft ADSI 2.5 interfaces to gain access to the Active Roles Administration Services from custom applications, scripts and user interfaces, including Web applications that use Active Server Pages. In addition, the provider supports the standard system OLE DB provider, so that OLE DB and ActiveX Data Objects (ADO) can be used to query directory services through Active Roles.
