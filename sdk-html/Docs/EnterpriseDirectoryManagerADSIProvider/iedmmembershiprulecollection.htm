IEDMMembershipRuleCollection

The IEDMMembershipRuleCollection interface is designed to manage a collection of membership rules for Managed Units or dynamic groups. The interface allows you to do the following:

  • Enumerate membership rules in the collection

  • Add and remove membership rules from the collection

Properties

The IEDMMembershipRuleCollection interface has the following properties.

Property

Description

_NewEnum

Enumerates membership rules for Managed Unit or dynamic group.

Count

Returns the number of rules in the membership rule collection.

GuidLookupMustBePerformed

Determines if the provider will use the GUID to look up AdsPath to directory objects the membership rule uses.

Item

Retrieves an IEDMMembershipRule instance by index.

IsModified

Determines if the collection has been modified. 

Methods

The IEDMMembershipRuleCollection interface defines the following methods.

Method

Description

Add

Adds a rule to the membership rule collection.

RemoveAt

Removes the specified rule from the membership rule collection.

Example Code (VBScript)

The following code snippet illustrates how to use the IEDMMembershipRuleCollection interface to administer Managed Units.

s=""
Set obj = GetObject("EDMS://CN=TestMU,CN=Managed Units,CN=Configuration")
TestMembershipRules obj
MsgBox s
Sub TestMembershipRules (obj)
     s = s + "TestMembershipRules{ ADsPath: " & obj.AdsPath + vbCrLf
     Set mrc = obj.MembershipRuleCollection
      'Enumerate membership rules for the specified Managed Unit
     i=0
     For each r in mrc
            s = s + "Rule #" & i & vbTab & "Base: " & r.Base & vbTab & "BaseGuid:" & r.BaseGuid + vbCrLf
            i = i+1
      Next
      'Add a rule
      Set mr = CreateObject("EDSIManagedUnitCondition")
      mr.Base = "EDMS://CN=Builtin,DC=mycompany,DC=com"
      mr.Filter = vbNullString
      mr.Type=2
      mrc.Add mr
      obj.SetInfo
      s = s + "Rule added" + vbCrLf
      'Get the number of rules
      lCount = mrc.Count
      s = s + "Current number of rules: " & Cstr(lCount) + vbCrLf
      'Remove the last added rule from the collection
      If lCount > 0 Then
            lIndex = lCount-1
            Set mr = mrc(lIndex) ' the same as mrc.Item(lIndex)
            s = s + "Rule #" &lIndex & vbTab &" Type: " & mr.Type & vbTab & "Base: " & mr.Base & vbTab & _
                  "BaseGuid: " & mr.BaseGuid & vbTab & "Filter: " & mr.Filter + vbCrLf
            mrc.RemoveAt lIndex     
            obj.SetInfo
            s = s + "Rule #" &lIndex & " was deleted." & vbTab & mrc.Count & " rules remain." + vbCrLf
      End If
      ' Disable GUID lookup
      mrc.GuidLookupMustBePerformed = false
      'Add a rule using the GUID of currently unavailable object (e.g. a domain that is currently disconnected) 
      Set mr = CreateObject("EDSIManagedUnitCondition")
      mr.BaseGuid = "95502f88-8bbd-4179-a147-9992aa541a03"
      mr.Filter = vbNullString
      mr.Type=3
      mrc.Add mr
      obj.SetInfo 
      s = s + "Current number of rules: " & Cstr(mrc.Count) + vbCrLf
      ' List all membership rules for Managed Unit
      i=0
      For each r in mrc
            s = s + "Rule #" & i & vbTab & "Base: " & r.Base & vbTab & "BaseGuid:" & r.BaseGuid + vbCrLf
            i = i+1
      Next           
End Sub
