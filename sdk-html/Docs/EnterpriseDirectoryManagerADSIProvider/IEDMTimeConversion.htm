IEDMTimeConversion

Active Roles defines the Automation object AelitaEDM.EDMTimeConversion that supports the IEDMTimeConversion interface. This interface defines properties and methods that help you process data of the DateTime type. The DateTime type can hold date-times, dates only or times only.

To use this interface, you must first create an instance of the object AelitaEDM.EDMTimeConversion with the function CreateObject, as shown in the following example: 

set objTimeConv = CreateObject("AelitaEDM.EDMTimeConversion")

Properties

The IEDMTimeConversion interface has the following properties:

Property

Description

LocalNow

Returns the current date and local time on this computer.

UTCNow

Returns the current date and time on this computer, expressed as the Coordinated

Universal Time (UTC).

Methods

The IEDMTimeConversion interface defines the following methods:

Method

Description

SetLocalTime

Sets the local time value you want to process.

SetUTCTime

Sets the UTC time value you want to process.

GetLocalTime

Converts time based on UTC to local time.

GetUTCTime

Converts local time to time based on UTC.

Example Code (VBScript)

The following code snippet illustrates how to use properties and methods defined in the IEDMTimeConversion interface.

Set objConv = CreateObject("AelitaEDM.EDMTimeConversion")
 ' Set the local time value to process with the AelitaEDM.EDMTimeConversion object
objConv.SetLocalTime(CDate("10/14/2009 6:47:53 PM"))                
 ' Get the time value expressed as UTC
MsgBox objConv.GetUTCTime()
 ' Get the current time on this computer expressed as UTC
MsgBox objConv.UTCNow 
 ' Get the current local time on this computer
MsgBox objConv.Localnow
