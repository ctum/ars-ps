Applying Policy Object

File name: Apply.vbs

This code snippet illustrates how to apply an existing Policy Object to a directory object.

Set ContainerObject = GetObject("EDMS://CN=AP Links,cn=Configuration")
Set OctetString=CreateObject("AelitaEDM.EDMOctetString")
OctetString.GenerateGuid
APLinkName="CN="+OctetString.GetGuidString
Set poLinkObject = ContainerObject.Create("edsPolicyObjectLink", APLinkName)
Set objPO=GetObject("EDMS://CN=TestPOlicy,CN=Administration,CN=Policies,CN=Configuration")
OctetString.Set objPO.GUID
Set objPO=Nothing
poLinkObject.Put "edsaAPOGUID", OctetString.GetOctetString
Set objToApply=GetObject("EDMS://OU=Sales,DC=mycompany,DC=com")
OctetString.Set objToApply.GUID
Set objToApply=Nothing
poLinkObject.Put "edsaSecObjectGUID", OctetString.GetOctetString
poLinkObject.SetInfo

This script allows you to link the "TestPolicy" Policy Object  to the "Sales" OU.
