IADsPropertyList

The IADsPropertyList interface is designed to modify, read, and update a list of property entries in the property cache of an ADSI object. It serves to enumerate, modify, and purge the contained property entries. You can use this interface to find out only all the initialized properties. This is different from using the schema to determine all possible attributes that the ADSI object can have and which properties have been set.

Note:

Before calling the methods of this interface, you should call the GetInfo or GetInfoEx method of IADs to load the assigned property values of the ADSI object from the Active Roles Administration Service into the property cache. After calling the methods of IADsPropertylist, you should call the SetInfo method of IADs to send all changes in the persistent store to Administration Service.

IADsPropertyList Methods

The IADsPropertyList interface inherits the methods of the standard COM interfaces IUnknown and IDispatch (see the MSDN Library). In addition, IADsPropertyList defines the following methods:

IADsPropertyList methods

Description

Next

Gets the next item in the property list. The returned item is a Property Entry object.

Skip

Skips a specified number of items, counting from the current cursor position, in the property list.

Reset

Moves back to the start of the list.

Item

Gets a property that is specified by name or by index.

GetPropertyItem

Retrieves the item that matches the name from the list.

PutPropertyItem

Updates the values for an item in the property list.

ResetPropertyItem

Removes the specified item from the list (that is, from the cache). You can specify the item to be removed by name (as a string) or by index (as an integer).

PurgePropertyList

Deletes all properties from the list and, therefore, releases the property caches as well.

IADsPropertyList Properties

The IADsPropertyList interface methods include access methods for the following properties.

Property

Description

PropertyCount

Access: Read

DataType: LONG

The number of items in the property list.

Sample Code (VBScript)

The following code snippet shows how to enumerate all initialized properties of an ADSI object using the IADsPropertyList interface.

Dim obj,str
ADSTYPE_INVALID = 0
ADSTYPE_DN_STRING = 1
ADSTYPE_CASE_EXACT_STRING = 2
ADSTYPE_CASE_IGNORE_STRING = 3
ADSTYPE_PRINTABLE_STRING = 4
ADSTYPE_NUMERIC_STRING = 5
ADSTYPE_BOOLEAN = 6
ADSTYPE_INTEGER = 7
ADSTYPE_OCTET_STRING = 8
ADSTYPE_UTC_TIME = 9
ADSTYPE_LARGE_INTEGER = 10
ADSTYPE_PROV_SPECIFIC = 11
ADSTYPE_OBJECT_CLASS = 12
ADSTYPE_CASEIGNORE_LIST = 13
ADSTYPE_OCTET_LIST = 14
ADSTYPE_PATH = 15
ADSTYPE_POSTALADDRESS = 16
ADSTYPE_TIMESTAMP = 17
ADSTYPE_BACKLINK = 18
ADSTYPE_TYPEDNAME = 19
ADSTYPE_HOLD = 20
ADSTYPE_NETADDRESS = 21
ADSTYPE_REPLICAPOINTER = 22
ADSTYPE_FAXNUMBER = 23
ADSTYPE_EMAIL = 24
ADSTYPE_NT_SECURITY_DESCRIPTOR = 25
ADSTYPE_UNKNOWN = 26
ADSTYPE_DN_WITH_BINARY = 27
ADSTYPE_DN_WITH_STRING = 28
' Bind to an ADSI object (see Note)
Set obj = GetObject("EDMS://Object_DN")
str="The initialized properties of the " + CStr(obj.Name)+ " object" +Chr(13)+Chr(10)
' Load the object properties from Administration Service to the cache
obj.GetInfo
' Enumerate only the initialized properties of the ADSI object
For i=1 To obj.PropertyCount-1
   Set item = obj.Item(i)
   str = str + item.Name + ": "
   For Each value In item.Values
      Select Case value.ADsType
         Case ADSTYPE_DN_STRING
            str = str + value.DNString + ", "
         Case ADSTYPE_CASE_EXACT_STRING
            str = str + value.CaseExactString + ", "
         Case ADSTYPE_CASE_IGNORE_STRING
            str = str + value.CaseIgnoreString + ", "
         Case ADSTYPE_PRINTABLE_STRING
            str = str + value.PrintableString + ", "
         Case ADSTYPE_NUMERIC_STRING
            str = str + value.NumericString + ", "
         Case ADSTYPE_BOOLEAN
            str = str + CStr (value.Boolean) + ", "
         Case ADSTYPE_INTEGER
            str = str + CStr (value.Integer) + ", "
     End Select
   Next
   str=str+ vbCrLf
Next
' Output the initialized properties
MsgBox str

Note:

In the preceding code snippet, Object_DN is the distinguished name of the object of interest.
