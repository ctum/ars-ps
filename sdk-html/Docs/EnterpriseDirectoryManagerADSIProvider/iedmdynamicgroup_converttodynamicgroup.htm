IEDMDynamicGroup::ConvertToDynamicGroup (deprecated)

The Sub procedure that allows you to convert a basic group to a dynamic group. 

Sub  ConvertToDynamicGroup(ByVal arrMembershipRules)

Arguments

arrMembershipRules

[in] Required. Contains an array of the membership rules you want the dynamic group to include. Each element of the array can be created using the properties of the IEDMMembershipRule interface.

Remarks

After you call this method and the SetInfo method of the IADs interface, the IsDynamicGroup property is set automatically to TRUE.

When you convert a basic group to a dynamic group, the group looses all members that were added to the group when it was basic.

Note:

The ConvertToDynamicGroup method is considered obsolete. While this method is still supported in the current version of Active Roles, it may be removed in the future. When writing new scripts, you should avoid using this method. When modifying existing scripts, you are strongly encouraged to remove any dependency on this method.
