Organizational Unit User Interface Mapping

This topic discusses the tabs of the Organizational Unit Properties dialog box in in the Active Roles MMC snap-in.

In the tables below, for each attribute, the LDAP Display Name column indicates the attribute's lDAPDisplayName, while the Display Name column indicates the attribute's display name as defined by the attributeDisplayNames property in display specifiers. If display specifiers do not define the display name, the Display Name column indicates the attribute's common name (cn). All display specifiers in Active Directory are found in the locale-specific containers under the container named "DisplaySpecifiers".

In the Active Roles MMC snap-in, display names are used in the Add Permission Entries wizard and on the Advanced tab of the Find dialog box.

Note:

The Managed by tab displays property values of the user managing the Organizational Unit.  These properties can be also found on the General, Address, Telephones, and Object tabs of the corresponding User Properties dialog box.
The distinguished name of the user that manages the Organizational Unit is saved in the managedBy property of the Organizational Unit object.
