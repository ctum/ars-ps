edsOneViewSpecifier Class

This class specifies the contents and appearance of a single Entitlement Profile item.

edsaEntitlementType

Mandatory.

Syntax: Enumeration (possible values: 1 - self, 2 - membership, 3 - managedBy) , SingleValued

Access: Stored, Read/Write

Searchable: Yes.

Specifies how the target user is entitled to the resource to be represented by the Entitlement Profile item. Possible values:

  • For Personal resource entitlement, target user is entitled to the resource by setting particular attributes of the target user itself.

  • For Managed resource entitlement, resource is an object for which the target user is assigned to a management role, such as a group for which the target user holds the Manager role.

  • For Shared resource entitlement, target user is entitled to the resource through membership in a certain group.

 

edsaEntitlementRule

Optional

Syntax: DirectoryString, MultiValued

Access: Stored, Read/Write

Searchable: Yes.

Contains conditions that determine if the resource is to be represented by the Entitlement Profile item for a given target user:

  • For Personal resource entitlement, the resource is represented if the target user meets the conditions specified by this attribute.

  • For Managed resource entitlement, the resource represented if the object managed by the target user meets the conditions specified by this attribute.

  • For Shared resource entitlement, the resource is represented if the group that provides the target user with access to that resource meets the conditions specified by this attribute.

  • If this attribute is not set, the resource is represented unconditionally.

 

edsaResourceTypeDisplayName

Optional

Syntax: DirectoryString, SingleValued

Access: Stored, Read/Write

Searchable: Yes.

Contains the display name that identifies the type of the resource represented by the Entitlement Profile item, such as Mailbox, Security Group, Distribution List, or Computer.

 

edsaResourceTypeIcon

Optional

Syntax: CaseIgnoreString, SingleValued

Access: Stored, Read/Write

Searchable: Yes.

Contains an icon to distinguish the resource represented by the Entitlement Profile item.

 

edsaResourceAttributes

Optional

Syntax: DirectoryString, MultiValued

Access: Stored, Read/Write

Searchable: Yes.

Lists the resource-specific attributes to be displayed by the Entitlement Profile item:

  • For Personal resource entitlement, the attribute values are retrieved from the target user object.

  • For Managed resource entitlement, the attribute values are retrieved from the object managed by the target user.

  • For Shared resource entitlement, the attribute values are retrieved from the group that provides the target user with access to the resource.

 

edsaResourceNamingAttribute

Optional

Syntax: DirectoryString, SingleValued

Access: Stored, Read/Write

Searchable: Yes.

Specifies a single resource-specific attribute to identify the resource within the Entitlement Profile item:

  • For Personal resource entitlement, the attribute value is retrieved from the target user object.

  • For Managed resource entitlement, the attribute value is retrieved from the object managed by the target user.

  • For Shared resource entitlement, the attribute value is retrieved from the group that provides the target user with access to the resource.

 

edsaSpecifierIsDisabled

Optional

Syntax: Boolean, SingleValued

Access: Stored, Read/Write

Searchable: Yes.

When set to TRUE, indicates that the specifier is not included into the Entitlement Profile reports.

 

edsaHTMLDetailsAttribute

Optional

Syntax: DirectoryString, SingleValued

Access: Stored, Read/Write

Searchable: Yes.

Specifies the attribute of the resource target object that holds HTML to be displayed in the Details section of the Entitlement Profile report.

 

edsaPriority

Optional

Syntax: Integer, SingleValued

Access: Stored, Read/Write

Searchable: Yes.

If the resource target object meets the entitlement conditions of multiple specifiers, the specifier with lower value of this attribute is applied, overriding the specifiers with a higher value. This attribute is not applicable to Personal resource entitlement.

 

edsaAuxiliaryInformation

Optional

Syntax: DirectoryString, MultiValued

Access: Stored, Read/Write

Searchable: Yes.

Multi-valued attribute to hold auxiliary information, such as the resource type name and resource name to be displayed by the default specifier in case of multiple managed objects. This attribute is not applicable to Personal resource entitlement.
