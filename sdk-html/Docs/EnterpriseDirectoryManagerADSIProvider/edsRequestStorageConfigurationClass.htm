edsRequestStorageConfiguration Class

The attributes of this class store templates in XSLT (Extensible Stylesheet Language for Transformations) format. The combinations of templates and scripting are used to transform XML reports based on change history data stored by Active Roles.

edsvaDeprovisioningReportXsltStylesheet 

Syntax: DirectoryString

Access: Read-only

Searchable: No.

Contains an XSLT template used to transform XML-format data saved into the user attribute edsvaDeprovisionReportXML to HTML-format report  on the user deprovisioning results.
