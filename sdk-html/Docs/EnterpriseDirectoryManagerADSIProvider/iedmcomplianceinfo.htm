IEDMComplianceInfo

This interface allows you to get information about results of checking the administrative policy set on attributes of the processed directory object. Using this interface you can perform the following operations:

  • Enumerate all attributes of the processed object on which a policy is set.

  • For each attribute, obtain a reference to an object used to represent the IEDMComplianceInfoRecord interface. The IEDMComplianceInfoRecord interface allows you to get information on Policy Objects that affect the processed object's attribute.

  • Get a list of the IEDMComplianceInfoRecord objects filtered by status of the administrative policy checking.

Methods

IEDMComplianceInfo defines the following methods:

Method

Description

NewEnum

Allows you to obtain a collection of objects used to represent the IEDMComplianceInfoRecord interface. 

RecordbyStatus

Allows you to filter a list of the administrative policy violations by status of the administrative policy checking. 

Properties

The IEDMComplianceInfo interface has the following properties:

Property

Description

PropertyName

Returns the Ldap display name of the object's attribute whose value is not in compliance with the administrative policy.

Example Code (VBScript)

The following  code snippet illustrates how to use the properties and methods provided by the IEDMComplianceInfoList, IEDMComplianceInfoRecord and IEDMComplianceInfo interfaces to get information about violations of the administrative policy affected the specified user object. 

Dim usr,strUser,Policy, PP, PP1,PP2
str="Violations of the administrative policy:" + vbCrLf
strUser="EDMS://CN=Test,OU=Sales,DC=Mycompany,DC=com"
Set usr=GetObject(strUser)
usr.Put "l", "Berlin"
usr.Put "department", "Berlin Software Department"
usr.Put "co", "Germany"
' Check whether values of the user attributes are in compliance with the administrative policy
set PP = usr.CheckPropertyValues
' An implicit call of _NewEnum of IEDMComplianceInfoList
For Each PP1 in PP
   ' An implicit call of _NewEnum of IEDMComplianceInfo
   For Each PP2 in PP1
      str = str + "Attribute name: " + PP2.PropertyName + " " + "Policy checking status: " + Cstr(PP2.Status) + _
      + " " + PP2.Message + vbCrLf
   Next
Next
MsgBox str

The script lists attributes of the user Test whose values are not in compliance with the administrative policy.
