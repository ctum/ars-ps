Creating and Deleting Directory Objects

This topic discusses how to create and delete directory objects using the IADsContainer interface.

Creating Directory Objects

The Create and CopyHere methods of the IADsContainer interface allow you to create directory objects within a container object. Examples of containers are domains, folders and Organizational Units.

To create a new directory object

  1. Bind to a directory container within which you want to create a new directory object.

  2. Do one of the following:

    • Invoke the Create method of IADsContainer. This method requires two arguments: class of the object to be created and the object relative distinguished name.

    • If you have a directory object, you can create a copy of that object within the current container using the CopyHere method of IADsContainer. This method requires two arguments: ADsPath to the existing object and the relative distinguished name of the object to be created.

  3. Set or modify the appropriate properties of the newly created object (see Modifying Properties of ADSI Objects). 

  The following code snippet (VBScript) shows how to create a new user object using the Create method of IADsContainer.

Dim cont
Dim usr
' Bind to the container within which a user will be created
Set cont = GetObject("EDMS://OU=Sales,DC=mycompany,DC=COM")
' Invoke the Create method of IADsContainer
Set usr = cont.Create("user", "CN=John MCDuglas")
' Set properties of the newly created user
usr.Put "samAccountName", "JMCDuglas"
usr.Description="Sales manager"
' Send the changes you made to Administration Service
usr.SetInfo

The following code snippet (VBScript) shows how to create a new user object using the CopyHere method of IADsContainer.

Dim objOU, newuser, sourceObjectPath
' Set the ADsPath to the source object to be used as a template object
sourceObjectPath="EDMS://CN=SourceUser,OU=Sales,DC=mycompany,DC=COM"
' Bind to the container where the source object resides
Set objOU = GetObject("EDMS://OU=Sales,DC=mycompany,DC=COM")
' Copy the existing ADSI object (the user object in the given example)
Set newuser = objOU.CopyHere(sourceObjectPath, "CN=Jane MCCornic")
' Set or modify properties of the newly created user object
newuser.Put "EmployeeID", "151"
newuser.FirstName = "Jane"
newuser.Put "sAMAccountName", "JMCCornic"
newuser.Put "sn","McCormic"
' Send the changes you made to Administration Service
newuser.setInfo

Note:

The CopyHere method only allows you to create a copy of a user, a group, a Managed Unit, an inetOrgPerson within a certain Organizational Unit (OU). Copying objects across OUs is not supported.

Deleting Directory Objects

The Delete method of IADsContainer allows you to delete directory objects that reside into a directory container.

To delete a directory object

  1. Bind to the directory container where the object you want to delete resides.

  2. Invoke the Delete method of IADsContainer. This method requires two arguments: the class of the object to be deleted and the object relative distinguished name.

The following code snippet (VBScript) shows how to delete a user object:

Dim objContainer
' Bind to the container where the user to be deleted resides
Set objContainer = GetObject("EDMS://cn=Users,DC=mycompany,DC=Com")
' Delete the user
objContainer.Delete "user", "cn=John Doe"
