edsDomainCacheConfig Class

This class is used to store information about the algorithm Active Roles uses to choose Dir Sync Server.

edsaAccountName

Optional.

Syntax: DirectoryString, SingleValued

Searchable: Yes.

Specifies the domain user account Administration Service uses to access the managed domain.

 

edsaAccountPassword

Optional.

Syntax: CaseExactString, SingleValued

Searchable: No.

Virtual attribute used to cipher/decipher account password stored in the attribute edsaStoredAccountPassword.

 

edsaDomainCacheStatus

Optional.

Syntax: DirectoryString, SingleValued 

Searchable: No.

Displays the status of the Managed Domain.

 

edsaEdmServicePublicationPath

Optional.

Syntax: DN, SingleValued

Searchable: Yes.

Specifies the path to publish Administration Service in Active Directory.

 

edsaFlags

Optional.

Syntax: Integer, SingleValued

Searchable: Yes.

General purpose flags used by the object to store bit information.

 

edsaLDAPPort 

Optional.

Syntax: Integer, SingleValued

Searchable: Yes.

Specifies the LDAP port number used for the request processing. The default port number is 389.

 

edsaPublishEdmService 

Optional.

Syntax: Boolean, SingleValued

Searchable: Yes.

When TRUE, Administration Service will be published in Active Directory.

 

edsaSavedDnsName

Optional.

Syntax: DirectoryString, SingleValued

Searchable: Yes.

Contains the managed domain DNS name. This attribute is used to validate the uniqueness of the managed domain name.

 

edsaSSLOption

Optional.

Syntax: Integer, SingleValued

Searchable: Yes.

The value of this attribute specifies whether Active Roles uses SSL when processing a request: 0 - SSL is not used; 1 - SSL is used; 2 - SSL is used if it is available.

 

edsaStoredAccountPassword

Optional.

Syntax: OctetString, SingleValued

Searchable: Yes.

Contains the encrypted domain account password.

 

edsaUseOverrideAccount

Optional.

Syntax: Boolean, SingleValued

Searchable: Yes.

When TRUE, Administration Service uses the account specified in the attribute edsaDomainAccount to access the managed domain. Otherwise, the service account of the Active Roles Administration Serviceis used for this purpose.

 

edsvaDomainDNS

Optional.

Syntax: DirectoryString, SingleValued

Searchable: No.

Contains the distinguished name of the object DomainDNS.

 

edsvaPreferredDC

edsvaPreferredSite 

Optional.

Syntax: DirectoryString, SingleValued

Searchable: No.

Virtual attribute of the object edsDomainCacheConfig (managed domain). This attribute is identical to edsaPreferredSite of the object DomainOptions.

 

edsvaSelectDCOption 

Optional.

Searchable: No.

Virtual attribute of the object edsDomainCacheConfig (managed domain). This attribute is identical to edsaSelectDCOption of the object DomainOptions.

 

edsvaUseDCFromAnotherSite

Optional.

Syntax: Boolean, SingleValued

Searchable: No.

Virtual attribute of the object edsDomainCacheConfig (managed domain). This attribute is identical to edsaUseDCFromAnotherSite of the object DomainOptions.
