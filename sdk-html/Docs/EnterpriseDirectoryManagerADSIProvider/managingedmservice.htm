Managing the Administration Service

This section contains the VBScript code samples that demonstrate how to manage Active Roles Administration Service with the Active Roles ADSI Provider. You can use these examples to perform the following tasks:

  • Get the Active Roles Administration Service Version

  • Get the Active Roles ADSI Provider Version

To use the scripts, copy the appropriate script code, paste it into Notepad or another text editor, and then save the file with a .vbs file extension. You can find these examples as separate .vbs files in the folder '...\SDK\Samples\ADSI Provider\EDM Service'.
