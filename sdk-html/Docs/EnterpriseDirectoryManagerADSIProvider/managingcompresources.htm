Managing Computer Resources

The Active Roles Administration Service implements a special namespace of scriptable objects used to manage computer resources. For more information, please refer to Managing Computer Resources.

This section contains the VBScript code samples that demonstrate how to manage computer resources with the Active Roles ADSI Provider. You can use these examples to perform the following tasks:

  • Get a List of Printers

  • Purge the Printer Queue

  • Get a List of Services

  • Restart a Service

  • Create a Share

  • Get a List of Shares

  • Add a Local User

  • Reset a Local User Password

To use a script, copy the script text, paste it into Notepad or another text editor, and then save the file with the .vbs file name extension. You can also find these examples as separate .vbs files in the folder '...\SDK\Samples\ADSI Provider\Computer Resources'.

Note:

Binding strings employed in the sample codes include the NETBIOS name of the domain where the computer in question resides. In this example, the domain name is "mycompany.com" while the domain NETBIOS name is "mycompany". For more information, please refer to Managing Computer Resources.
