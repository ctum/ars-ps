Creating Groups

The following code example shows how to create a new universal security group. 

[ C# ]
using System;
using System.DirectoryServices; 
using ActiveDs;
using EDMLib;
namespace CreateNewGroup
{
	class Class1
	{
		[STAThread]
		static void Main(string[] args)
		{
			// Set the OU's name
			string strDN = "EDMS://OU=Sales,DC=mycompany,DC=com";
			DirectoryEntry ou = new DirectoryEntry(strDN);
			// Add the Managers group.
			DirectoryEntry mgr = ou.Children.Add("CN=New group","group");
			// Set the group type to a secured global group.
			mgr.Properties["groupType"].Value = -2147483640;
			// Commit the new group to the directory.
			mgr.CommitChanges();
			Console.WriteLine("New group has been created.");
		}
	}
}
}


[ Visual Basic .NET ]
Imports System.DirectoryServices
Module Module1
    Sub Main()
        ' Bind to the OU where you want to create a new group
        Dim strDN As String = "EDMS://OU=Sales,DC=mycompany,DC=com"
        Dim de As New DirectoryEntry(strDN)
        Dim newGroup As DirectoryEntry = de.Children.Add("CN=New group", "group")
        newGroup.Properties("groupType").Value = -2147483640
        newGroup.CommitChanges()
        Console.WriteLine("New group has been created")
    End Sub
End Module

To use this example

  1. Start Visual Studio and create a new C# or Visual Basic project using the procedure outlined in Getting Started with Microsoft .NET Projects.

  2. Copy this sample code and then paste it into the Code Pane. Set the variable strDN to the distinguished name of the Organizational Unit within which you want to create a new group.

  3. Compile and run the application.
