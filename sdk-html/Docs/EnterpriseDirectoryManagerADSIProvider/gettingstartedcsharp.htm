Getting Started With Microsoft .NET Projects

This topic describes how to create a Microsoft .NET project for developing console applications that use the Active Roles ADSI provider to perform basic directory tasks.

To create a console application

  1. Open Visual Studio .NET, on the File menu, point to New, and then click Project.

  2. In the New Project dialog box, do the following:

    • From Project Types, select Visual C# or Visual Basic

    • From Templates, select Console Application.

    • Name the project using the Name and Location text boxes. When finished, click OK.

  3. On the Project menu, click Add Reference.

  4. In the Add Reference dialog box, do the following:

    • On the .NET tab, select System.DirectoryServices.dll from the provided list

    • On the COM tab, select "Active DS Type Library" and " Active Roles ADSI Provider Type Library" from the provided list

    • When finished, click OK to close the dialog box.

  5. Insert the application's code into the Code Pane.

For examples of the C# and Visual Basic projects, refer to Sample Projects.
