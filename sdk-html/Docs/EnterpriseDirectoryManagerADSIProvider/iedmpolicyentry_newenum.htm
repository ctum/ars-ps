IEDMPolicyEntry::_NewEnum

The _NewEnum method retrieves an interface on an enumerator object. Note, that in Visual Basic and VBScript, the For Each …In … Next statement invokes this method implicitly.

Remarks

Applying the For Each …In … Next statement to a Policy Object Entry (Policy) allows you to enumerate parameters defined  for that Policy.  

Example Code (VBScript)

This code snippet illustrates how to enumerate parameters defined for Policies the specified Policy Object includes. For each parameter, the script outputs the parameter ID and value. 

Set objPO = GetObject("EDMS://CN=MyPO,CN=Administration,CN=Policies,CN=Configuration")
For Each APE In objPO
      ss= ss + APE.name + " / " + "APE Type= " + Cstr(APE.Type) + " Events: " + Cstr(APE.Events)_
      + " ParamCount= " + Cstr(APE.PolicySettingsCount) + vbCrLf
      For Each Param in APE
         ss = ss + "ParamID=" + Cstr(Param.Name) + " ParamValue=" + Cstr(Param.Value) + vbCrLF
      Next   
Next
MsgBox ss
