IEDMMembershipRule Properties

The following table describes properties provided by the IEDMMembershipRule interface.

Property

Description

Type

Access: Read/Write

Data type: Long integer

Sets and returns the type of the membership rule that applies to Managed Unit. This property can take any value from the EDS_MUCONDITIONENUM enumeration.

Base

Access: Read/Write

Data type: String

Sets and returns the scope of an LDAP search string used for the membership rule. The scope of the search specifies a container object where the search for directory objects will start.

BaseGuid

Access: Read/Write

Data type: String

Sets and returns the GUID of the object that represents this membership rule.

Filter

Access: Read/Write

Data type: String

Sets and returns the filter of an LDAP search string used for the membership rule. The filter of the search specifies the objects to be found according to the object classes, names or properties. 

IsModified

Access: Read only

Data type: Boolean

Returns a Boolean that represents whether the membership rule has been changed since the last time the GetInfo/SetInfo method was called.
