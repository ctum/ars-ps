IEDMPOLink Properties

The following table describes properties provided by the IEDMPOLink interface.

Property

Description

IsBlockingLink

Access: Read/Write

Data type: Boolean

When TRUE, blocks the policy inheritance for a securable object (a directory object to which Policy Object has been applied).

LinkDisabled

Access: Read/Write

Data type: Boolean

When TRUE, disables the Policy Object link.

LinkFlags

Access: Read/Write

Data type: Integer

Sets and returns the edsaLinkFlags property of the Policy Object link.

PolicyObjectADsPath

Access: Read/Write

Data type: DN String

Sets and returns the distinguished name of Policy Object used to enforce the administrative policy.

PolicyObjectGuid

Access: Read/Write

Data type: Octet string

Sets and returns the GUID of Policy Object used to enforce the administrative policy.

PolicyObjectGuidAsString

Access: Read/Write

Data type: String

Sets and returns the GUID (as a string) of Policy Object used to enforce the administrative policy.

SecurableObjectADsPath

Access: Read/Write

Data type: String

Sets and returns the distinguished name of the securable object.

SecurableObjectGuid

Access: Read/Write

Data type: Octet string

Sets and returns the GUID of the securable object.

SecurableObjectGuidAsString

Access: Read/Write

Data type: String

Sets and returns the GUID (as a string) of the securable object.

SecurableObjectName

Access: Read-only

Data type: String

Returns samAccountName of the securable object. Note that this property returns an empty string if the securable object is not a security principal.
