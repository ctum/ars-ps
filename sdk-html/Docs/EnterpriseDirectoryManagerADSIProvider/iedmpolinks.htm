IEDMPOLinks

The IEDMPOLinks interface allows you to get a list of the Policy Object links for a Policy Object.

Properties

The IEDMPOLinks interface has the following properties:

Property

Description

_NewEnum

Enumerates the Policy Object links for a Policy Object.
