edsPolicyObjectLink Class

This class is used to store information about Policy Object Link (APO Link).

edsaAPOGUID

Mandatory.

Syntax: OctetString, SingleValued

Searchable: Yes.

Contains the GUID of the Policy Object (used in APO-Link object).

 

edsaFlags

Optional.

Syntax: Integer, SingleValued

Searchable: Yes.

General purpose flags used by the object to store bit information.

 

edsaIsInherited

Optional.

Syntax: Boolean, SingleValued

Searchable: Yes.

When set to TRUE, the APO Link is inherited.

 

edsaLinkDisabled

Optional.

Syntax: Boolean, SingleValued

Searchable: Yes.

When set to TRUE, the APO Link is disabled.

 

edsaLinkFlags

Optional.

Syntax: Integer, SingleValued

Searchable: Yes.

Specifies propagation behavior of the Administrative policies defined with this APO Link: 0 - policies will be enforced only onto the directory object to which the appropriate Policy Object has been linked; 2 - policies will be enforced onto the directory object and to all its child objects; 6 - policies will be enforced onto the directory object and its first-level child objects; 10 - policies will be enforced only onto child objects of the directory object; 12 - policies will be enforced only onto first-level child objects of the directory object.

 

edsaOrdinalNumber

Optional.

Syntax: Integer, SingleValued

Searchable: Yes.

Ordinal number (such as APO Links ordered in Policy Descriptor).

 

edsaSecObjectGUID

Mandatory.

Syntax: OctetString, SingleValued

Searchable: Yes.

Contains the 'objectGUID' attribute value of the directory object to which the Policy Object has been linked.

 

edsaTrusteeSID

Optional.

Syntax: Sid, SingleValued

Searchable: Yes.

Contains SID of Trustee. When this attribute is set, the Administrative policies defined with this APO Link are valid only for that Trustee.
