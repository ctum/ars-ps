Contact Object User Interface Mapping

This topic discusses the tabs of the Contact Properties dialog box in the Active Roles MMC snap-in.

In the tables below, for each attribute, the LDAP Display Name column indicates the attribute's lDAPDisplayName, while the Display Name column indicates the attribute's display name as defined by the attributeDisplayNames property in display specifiers. If display specifiers do not define the display name, the Display Name column indicates the attribute's common name (cn). All display specifiers in Active Directory are found in the locale-specific containers under the container named "DisplaySpecifiers".

Active Roles uses the display names in the Add Permission Entries wizard, and on the Advanced tab of the Find dialog box.

The values of properties listed in the last row of the table above depend on the options selected under Message restrictions. 
- When the From everyone option is selected, the  dLMemSubmitPerms, dLMemRejectPerms, authOrig, unauthOrig properties are not set.
- When the Only From option is selected, the authOrig property contains DN's of recipients from which the recipient can receive e-mails.
- When the From Everyone Except option is selected, the unauthOrig property contains DN's of recipients from which the recipient cannot receive e-mails.
