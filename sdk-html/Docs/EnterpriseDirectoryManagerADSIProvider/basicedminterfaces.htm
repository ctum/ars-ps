Basic Interfaces

Basic interfaces are designed to process information on administrative policies that apply to directory and Active Roles-specific objects. In addition, these interfaces include the IEDMNameSpace interface designed to get the Active Roles ADSI Provider version. The following interfaces are covered:

  • IEDM

  • IEDMComplianceInfoList

  • IEDMComplianceInfo

  • IEDMComplianceInfoRecord

  • IEDMPolicyInfoList

  • IEDMPolicyInfo

  • IEDMNameSpace
