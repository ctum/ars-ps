Group Object User Interface Mapping

This topic discusses the tabs of the Group Properties dialog box in the Active Roles MMC snap-in.

In the tables below, for each attribute, the LDAP Display Name column indicates the attribute's lDAPDisplayName, while the Display Name column indicates the attribute's display name as defined by the attributeDisplayNames property in display specifiers. If display specifiers do not define the display name, the Display Name column indicates the attribute's common name (cn). All display specifiers in Active Directory are found in the locale-specific containers under the container named "DisplaySpecifiers".

Active Roles uses the display names in the Add Permission Entries wizard, and on the Advanced tab of the Find dialog box.

Note:

The values of the reportToOwner and reportToOriginator attributes depend on the selected options in the following way:

- When the Send delivery reports to group owner option is selected, the reportToOwner attribute is set to True while the reportToOriginator attribute  is set to False;
- When the Send delivery reports to message originator option is selected, the reportToOwner attribute is set to False while the reportToOriginator attribute  is set to True;
- When the Do not send delivery reports option is selected, both the reportToOwner and reportToOriginator attributes are set to False.

Clicking the Custom Attributes button, you can display the Exchange Custom Attributes dialog box allowing you to specify additional parameters of the group mailbox.
