IEDMComplianceInfo::RecordByStatus

The RecordbyStatus method returns an object used to represent the IEDMComplianceInfoRecord interface. This method allows you to filter a list of the administrative policy violations by status of the administrative policy checking. 

Function  RecordByStatus(byVal lStatus)

Arguments

lStatus

[in] Required. A long integer that sets the Status property of the IEDMComplianceInfoRecord objects you want the method to return.

Settings

The lStatus argument can take any value from the EDS_POLICY_COMPLIANCE_ENUM enumeration.

Example Code (VBScript)

This code snippet illustrates how to use the RecordbyStatus method. 

Dim usr,strUser,Policy, PP, PP1,PP2
str="Violations of the administrative policy:" + vbCrLf
strUser="EDMS://CN=Test,OU=Sales,DC=Mycompany,DC=com"
Set usr=GetObject(strUser)
usr.Put "l", "Berlin"
usr.Put "department", "Berlin Software Department"
usr.Put "co", "Germany"
' Check whether values of the user attributes are in compliance with the administrative policy
set PP = usr.CheckPropertyValues
' An implicit call of _NewEnum of IEDMComplianceInfoList
For Each PP1 in PP
   ' Lists only the policy violations whose checking status is equal to 1
   PP1 = PP1.RecordbyStatus(1)
   ' An implicit call of _NewEnum of IEDMComplianceInfo
   For Each PP2 in PP1
      str = str + "Attribute name: " + PP2.PropertyName + " " + "Policy checking status: " + Cstr(PP2.Status) + _
      + " " + PP2.Message + vbCrLf
   Next
Next
MsgBox str
