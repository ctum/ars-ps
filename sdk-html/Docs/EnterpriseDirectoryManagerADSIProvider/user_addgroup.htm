Adding a User to a Group

File name: Add.vbs

This code snippet illustrates how to add a user as a member of group.

Set Group=GetObject("EDMS://CN=SalesStuff,Ou=Sales,DC=mycompany,DC=com")
If not Group.IsMember("EDMS://CN=Joghn Smith,Ou=Sales,DC=mycompany,DC=com") Then
    Group.Add "EDMS://CN=Joghn Smith,Ou=Sales,DC=mycompany,DC=com"
    Group.SetInfo
End If
