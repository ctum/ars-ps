IEDMLargeInteger::Set

The Set method sets the LargeInteger type value that you want to process. 

Sub  Set(ByVal lInt)

Arguments

lInt

[in] Required. Specifies the value you want to set. You can only pass a value of the IAdsInteger type as the argument of this method. For information about the IAdsInteger interface, refer to the MSDN library.

Remarks

For AD attributes conforming to the syntax LargeInteger (INTEGER*8), you can obtain values of the IAdsInteger type using the Get method of the IADs interface, as shown in the following code snippet:

valIAdsInt = obj.Get("pwdLastSet")
objLargeInt.Set valIAdsInt
