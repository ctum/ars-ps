IEDMReverseLookupInfo

This interface allows you to process objects found with the ReverseLookup method of the IEDMLookup interface.

Properties

The IEDMReverseLookupInfo interface has the following properties.

Property

Description

DomainNB

Returns the NET BIOS name of the domain where the found object resides.

DN

Returns the object's distinguished name.

SAMAccountName

Returns the object's sAMAccount name.

ObjectClass

Returns the class of the found object.

Sid

Returns the SID of the found object.

Example Code (VBScript)

The following code snippet illustrates how to use the IEDMLookup and IEDMReverseLookupInfo interfaces to find objects with the sAMAccountName property "Stuff" residing in the Test domain.

ss=""
Set objLookup = CreateObject("EDSILookup")
objLookup.Initialize2 "", "", "",0
Set objLookupInfoCollection = objLookup.ReverseLookup("Test","Stuff")
For Each objLookupInfo In objLookupInfoCollection
   TargetDN = objLookupInfo.DN
   DomainNB = objLookupInfo.DomainNB
   sacc = objLookupInfo.sAMAccountName
   oClass = objLookupInfo.objectClass
   ss = ss + CStr(TargetDN) + vbCRLF + Cstr(DomainNB) + vbCRLF +_
   sacc + vbCRLF + oClass + vbCRLF
Next
MsgBox ss
