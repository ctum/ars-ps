IEDMPolicyInfo

The IEDMPolicyInfo interface allows you to manage information about administrative policies that affect an attribute of directory objects or Active Roles-specific objects.

Properties

The IEDMPolicyInfo interface has the following properties.

Property

Description

AutoGenerated

Returns True if the attribute value has been automatically generated by the policy.

AdminNote

Returns a note to be displayed on the Administrative Note tab of the Policy Information dialog box.

CaseAjusting

Specifies how to change the case of the attribute value.

GeneratedValue

Returns a value of the attribute generated by the policy.

MaxLength

Returns the maximum allowed length of the attribute.

Name

Returns the Ldap display name of the directory object attribute about which the interface gets the information.

PolicyRule

Returns a rule used by the policy to generate the attribute values.

PossibleValues

Returns acceptable values for the attribute.

ServerSideGenerated

Returns True if Administration Service prompts the client to select the object attribute value from a provided list.

ValueRequired

Specifies whether the attribute must be set.

ValueRestricted

Specifies whether the attribute must take one of the values specified using the PossibleValues property.

Example Code (VBScript)

The following code snippet illustrates how to list user attributes affected by administrative policy. The script also illustrates how to get a list of values the policy offers for attributes with the required values, if any. The script uses:

  • GetPolicyInfoList method (IEDM interface)

  • PossibleValues property (IEDMPolicyInfo interface)

Dim cont
Dim usr,strDNUser,Policy,PP,PP1,str,Value,strReq
str="      Attributes affected by Administrative Policy:" + vbCrLf +vbCrLf
strDNUser="EDMS://CN=NewUser,OU=SalesNew,DC=mycompany,DC=com"
Set usr=GetObject(strDNUser)
usr.GetInfo
set PP = usr.GetPolicyInfoList
For Each PP1 in PP
    str = str + PP1.Name
    If PP1.ValueRequired Then
       str = str + " (required)" + vbCrLf
                         else
       str = str + vbCrLf                  
    End if 
    For Each Value in PP1.PossibleValues
       str = str + "    Possible value: " + Cstr(Value) + vbCrLf
    Next
Next
MsgBox str
