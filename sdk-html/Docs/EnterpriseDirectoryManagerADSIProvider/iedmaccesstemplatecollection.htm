IEDMAccessTemplateCollection

The IEDMAccessTemplateCollection interface is designed to manage a collection of Access Templates nested in the given Access Template. The interface allows you to do the following:

  • Enumerate Access Templates in the collection

  • Add and remove Access Templates from the collection

Properties

The IEDMAccessTemplateCollection interface has the following properties.

Property

Description

_NewEnum

Enumerates Access Templates in the collection.

Count

Returns the number of elements in the Access Template collection.

Ids

Returns an array of strings that contains GUIDs of all elements in the Access Template collection.

IsModified

Determines if the collection has been modified. 

Methods

The IEDMMembershipRuleCollection interface defines the following methods.

Method

Description

Add

Adds an object used to represent the IEDMAccessTemplate interface to the collection.

AddByID

Adds the Access Template identified by its ID to the collection.

Remove

Removes the specified rule from the membership rule collection.

RemoveAll

Removes all Access Templates from the collection.

RemovebyID

Removes the specified Access Template from the collection.

Example Code (VBScript)

The following code snippet illustrates how to use the IEDMAccessTemplateCollection interface.

' Set the distinguished name for the Access Template in question
const strATDN = "EDMS://CN=MyAT,CN=Access Templates,CN=Configuration"
ss=""
Set obj = GetObject(strATDN)
 ' Return the number of the nested Access Templates in the collection
ss = ss + "Number of the nested Access Templates: " & Cstr(obj.NestedTemplateCollection.Count) & vbCRLf & vbCRLf
Set clxn = obj.NestedTemplateCollection
 ' Enumerate nested Access Templates
n=0 
ss = ss + "List nested Access Templates' GUIDs:" & vbCRLf
For each a in clxn.IDs 
        n = n+1
	ss = ss + Cstr(n) + ": " + a + vbCRLf

	If (N=1) then clxn.RemoveByID a
	
Next
ss = ss + vbCRLf
ss = ss + "Is the AT collection has been modified?: " & clxn.IsModified & vbCRLf
ss = ss + "Current number of the nested Access Templates: " & clxn.Count & vbCRLf
MsgBox ss
