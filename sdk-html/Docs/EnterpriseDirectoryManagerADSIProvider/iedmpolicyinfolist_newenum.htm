IEDMPolicyInfoList::_NewEnum

The _NewEnum method retrieves an interface on an enumerator object. Note, that in Visual Basic and VBScript, the For Each …In … Next statement invokes this method implicitly.

Remarks

Applying the For Each …In … Next statement to an IEDMPolicyInfoList object allows you to enumerate the object attributes on which an administrative policy is set.
