edsWICustomizationSettings Class

Contains information about the interface settings and customization settings of the Active Roles Web Interface.

edsaWICommands 

Optional.

Syntax: CaseExactString, Multi-valued

Searchable: Yes.

Describes commands used in the menus defined in the property edsaWIMenus.

 

edsaWIEntries

Optional.

Syntax: CaseExactString, Multi-valued

Searchable: Yes.

Describes entries used in the forms defined in the property edsaWIForms.

 

edsaWIForms 

Optional.

Syntax: CaseExactString, Multi-valued

Searchable: Yes.

Describes forms associated with the Form-type commands defined in the property edsaWICommands.

 

edsaWIMenus 

Optional.

Syntax: CaseExactString, Multi-valued

Searchable: Yes.

Describes menus associated with directory objects managed with the Web Interface.

 

edsaWIStrings 

Optional.

Syntax: CaseExactString, Multi-valued

Searchable: Yes.

Defines names and values of the string constants used in the Web Interface customization settings.
