IEDMLookupInfo

This interface allows you to process objects found with the IEDMLookup interface. This topic describes properties provided by the IEDMLookupInfo interface. The properties enable you to obtain the following information about the found objects:

  • Distinguished name

  • Printable name

  • Class

Properties

The IEDMLookupInfo interface has the following properties.

Property

Description

DN

Returns the object's distinguished name.

PrintableName

Returns the object's printable name.

ObjectClass

Returns the object's class.

Example Code (VBScript)

The following code snippet illustrates how to find objects using the LookupGUIDs method.

ss="The found objects: " +vbCRLF
Set objLookup = CreateObject("EDSILookup")
objLookup.Initialize2 "", "", "", 0
' Define array of GUIDs (binary strings)of the objects you want to find
arrGUIDs=Array(binGUID1,binGUID2,...) ' Complete this line
Set objLookupInfoCollection = objLookup.LookupGUIDs(arrGUIDs)
For Each objLookupInfo In objLookupInfoCollection
   TargetDN = objLookupInfo.DN
   oClass = objLookupInfo.objectClass
   ss = ss + CStr(TargetDN) + vbCRLF + "Class: " + CStr(oClass) + vbCRLF 
Next
MsgBox ss
