edsACE Class

This class is used to store information about Access Control Entry object.

edsaAccessTemplateGUID

Mandatory.

Syntax: OctetString, SingleValued

Searchable: Yes.

Contains GUID of the Access Template used to specify permissions.

 

edsaIsInherited

Optional.

Syntax: Boolean, SingleValued

Searchable: Yes.

When TRUE, the AT Link is inherited.

 

edsaIsSynchronizedWithAD

Optional.

Syntax: Integer, SingleValued

Searchable: Yes.

Specifies whether Active Roles permissions are synchronized with Active Directory: 0 - Active Roles permissions are not propagated to AD; 1 - Active Roles permissions are  propagated to AD.

 

edsaLinkDisabled

Optional.

Syntax: Boolean, SingleValued

Searchable: Yes.

When TRUE, the AT Link is disabled.

 

edsaLinkFlags

Optional.

Syntax: Integer, SingleValued

Searchable: Yes.

Specifies propagation behavior of the permissions defined with this AT Link: 0 - permissions will be applied only onto the directory object to which the appropriate Access Template has been linked; 2 - permissions will be applied to the directory object and to all its child objects; 6 - permissions will be applied onto the directory object and its first-level child objects; 10 - permissions will be applied only onto child objects of the directory object; 12 - permissions will be applied only onto first-level child objects of the directory object.

 

edsaSecObjectGUID

Mandatory.

Syntax: OctetString, SingleValued

Searchable: Yes.

Contains the objectGUID attribute value of the securable object.

 

edsaTrusteeSID

Mandatory.

Syntax: Sid, SingleValued

Searchable: Yes.

Contains SID of Trustee to whom permissions specified by the Access Template have been assigned.
