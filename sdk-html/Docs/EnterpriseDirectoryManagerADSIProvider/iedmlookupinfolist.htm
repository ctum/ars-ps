IEDMLookupInfoList

This interface allows you to process objects found with the IEDMLookup interface.

Properties

The IEDMLookupInfoList interface has the following properties.

Property

Description

Count

Returns a number of objects found with the IEDMLookup interface.

Methods

The IEDMLookupInfoList interface defines the following methods.

Method

Description

_NewEnum

Allows you to enumerate objects returned by the LookupGUIDs, LookupSIDs or LookupSIDsOnComputer methods of the IEDMLookup interface.
