IEDMMembershipRule

The IEDMMembershipRule interface enables you to process membership rules for Managed Units and dynamic groups. Membership rules are stored into an array that can be set or returned using the MembershipRules property of the IEDMManagedUnit or IEDMDynamicGroup interface. Each element of the array can be processed with the IEDMMembershipRule interface's properties.

Properties

The IEDMMembershipRule interface has the following properties.

Property

Description

Type

Sets and returns the type of the membership rule that applies to Managed Unit.

Base

Sets and returns the scope of an LDAP search string used for the membership rule.

BaseGuid

Sets and returns the GUID of the object that represents this membership rule.

Filter

Sets and returns the filter of an LDAP search string used for the membership rule.

IsModified

Determines if the membership rule has been modified.

Types of Membership Rules

Active Roles supports six predefined types of membership rules for Managed Units and dynamic groups. The following code snippets illustrate how you can create membership rules of the specified type.  Note that a Managed Unit or a dynamic group can include any number of membership rules of the same type.

Important:

Before defining a new membership rule, create an instance of the object used to represent the IEDMMembershipRule interface with the CreateObject function (ProgID="EDSIManagedUnitCondition") as shown in the following examples.

Include by Query Rule

This membership rule has Managed Unit include all the User and Computer objects residing in the OU Sales in the domain mycompany.com.

Set Rule = CreateObject("EDSIManagedUnitCondition")
Rule.Base = "EDMS://OU=Sales,DC=mycompany,DC=com"
Rule.Filter = "(|(objectClass=user)(objectClass=computer))"
Rule.Type=1

Exclude by Query Rule

This membership rule ensures that all the User and Computer objects residing in the OU Sales in the domain mycompany.com are excluded from Managed Unit.

Set Rule = CreateObject("EDSIManagedUnitCondition")
Rule.Base = "EDMS://OU=Sales,DC=mycompany,DC=com"
Rule.Filter = "(|(objectClass=user)(objectClass=computer))"
Rule.Type=2

Include Explicitly Rule

This membership rule has Managed Unit only include the OU Sales2 in the domain mycompany.com regardless of whether it is renamed, moved, or has any properties changed.

Set Rule = CreateObject("EDSIManagedUnitCondition")
Rule.Base = "EDMS://OU=Sales2,DC=mycompany,DC=com"
Rule.Filter = vbNullString
Rule.Type=3

Exclude Explicitly Rule

This membership rule ensures that the OU Developpers in the domain mycompany.com is excluded from Managed Unit regardless of whether it is renamed, moved, or has any properties changed.

Set Rule = CreateObject("EDSIManagedUnitCondition")
Rule.Base = "EDMS://OU=Developpers,DC=mycompany,DC=com"
Rule.Filter = vbNullString
Rule.Type=4

Include Group Members Rule

This membership rule has Managed Unit include all members of the built-in group Administrators in the domain mycompany.com regardless of whether they are renamed, moved, or have any properties changed.

Set Rule = CreateObject("EDSIManagedUnitCondition")
Rule.Base = "EDMS://CN=Administrators,CN=Builtin,DC=mycompany,DC=com"
Rule.Filter = vbNullString
Rule.Type=5

Exclude Group Members Rule

This membership rule ensures that the members of the built-in group Administrators in the domain mycompany.com are excluded from Managed Unit regardless of whether they are renamed, moved, or have any properties changed.

Set Rule = CreateObject("EDSIManagedUnitCondition")
Rule.Base = "EDMS://CN=Administrators,CN=Builtin,DC=mycompany,DC=com"
Rule.Filter = vbNullString
Rule.Type=6

Example Code (VBScript)

The following code snippet shows how to get a list of membership rules for the Managed Unit in question. The script makes use of the IEDMManagedUnit, IEDMMembershipRule, and IEDMMembershipCollection interfaces.

' Set the distinguished name of Managed Unit
const strManagedUnit = "CN=MyMU,CN=Managed Units,CN=Configuration"
' -----------------------------------------------------
Dim objMU, strText, RulesCollection,ObjMUCondition,N
' Bind to the Managed Unit object
Set objMU = GetObject("EDMS://" + strManagedUnit)
objMU.GetInfo
strText ="Membership rules applied to the " + strManagedUnit + "Managed Unit:" + vbCrLf + vbCrLf
' Retrieve a list of the membership rules
Set RulesCollection=objMU.MembershipRuleCollection
N=0
    ' Process membership rules
    For Each objMUCondition in RulesCollection
        N = N+1
        strText = strText + "----- Rule N " + CStr(N) + " ----" + vbCrLf
        strText = strText + "Scope: "  + CStr(objMUCondition.Base) + vbCrLf
        strText = strText + "Filter: " + Cstr(objMUCondition.Filter) + vbCrLf
        strText = strText + "Type: "   + Cstr(objMUCondition.Type) + vbCrLf
    Next
' Output a list of membership rules
MsgBox strText

For more examples of the VBScript codes, refer to Administering Managed Units.
