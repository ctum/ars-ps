contact Class

The contact class provides a formal description of a Contact object. In addition to the standard ADSI Contact class attributes, Active Roles defines the following attributes, which can be accessed using the Active Roles ADSI Provider.

Note:

For more information about the Contact class, please refer to the MSDN Library.

edsaAdminGroup

edsaAllExchangeTasks

edsaDeleteEmail

edsaEstablishEmail

edsaExchangeTasksAvailable

edsaIsMsExchangePresent

edsaMsExchMixedMode

edsaRemoveAllMsExchAttributes

For detailed description of the attributes above, see Exchange Related Attributes.
