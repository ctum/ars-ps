Managing Access Templates

This section contains the VBScript code samples that demonstrate how to manage Access Template (AT) with the Active Roles ADSI Provider. You can use these examples to perform the following tasks on Access Templates:

  • Create AT

  • Add Permission Entries

  • Apply AT

  • Create AT Container

  • Find ATs Linked to a Directory Object

To use the scripts, copy the appropriate script code, paste it into Notepad or another text editor, and then save the file with a .vbs file extension. You can find these examples as separate .vbs files in the folder '...\SDK\Samples\ADSI Provider\Access Template'.
