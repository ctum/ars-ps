IEDMManagedUnit

The IEDMManagedUnit interface allows you to administer the Manage Unit objects.

Properties

The IEDMManagedUnit interface has the following properties.

Property

Description

MemberShipRules (Deprecated)

Sets and returns an array of membership rules for Managed Unit.

MemberShipRuleCollection

Returns an object used to represent the IEDMMembershipRuleCollection interface.
