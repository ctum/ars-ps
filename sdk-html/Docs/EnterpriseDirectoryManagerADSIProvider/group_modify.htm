Modifying Groups

File name: SetDescription.vbs

This code snippet illustrates how to modify properties of a group.

Set groupObject=GetObject("EDMS://CN=SalesStuff,DC=mycompany,DC=com")
groupObject.Put "description", "A test group"
groupObject.SetInfo
