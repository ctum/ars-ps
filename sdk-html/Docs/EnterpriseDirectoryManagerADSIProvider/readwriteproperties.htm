Reading and Writing the Object Properties

The Active Roles ADSI provider supports a number of interfaces whose properties and methods allow you to read/write properties of the object you have bound to (see Binding to Directory Objects).

All directory objects support the IADs interface. The object's properties can be accessed or modified by their LDAP display names using  the Get/GetEx or Put/PutEx method of the IADs interface. In Automation applications, you can use the dot operator to access the so called basic IADs properties of directory objects such as the object's AdsPath, name, etc.

To access information about objects, you can also use other interfaces specific to the object class in question. For example, user objects support the IADsUser interface.

For more information about how to read and write the object properties, refer to Accessing Properties of ADSI Objects and Modifying Properties of ADSI Objects, respectively.
