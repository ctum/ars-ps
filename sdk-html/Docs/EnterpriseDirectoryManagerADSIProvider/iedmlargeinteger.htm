IEDMLargeInteger

Active Roles defines the Automation object AelitaEDM.EDMLargeInteger that supports the IEDMLargeInteger interface. This interface defines methods that help you create and process the directory object attributes conforming to the LargeInteger syntax, such as the user object's attributes pwdLastSet, accountExpires, etc. Using this interface you can easily transform a large integer type value to the Generalized-Time type string and vice versa.

To use this interface, you must first create an instance of the object AelitaEDM.EDMLargeInteger with the function CreateObject, as shown in the following example: 

set objLInteger = CreateObject("AelitaEDM.EDMLargeInteger")

Properties

The IEDMLargeInteger interface has the following properties:

Property

Description

HighPart

Gets and sets the upper 32 bits of the large integer.

LowPart

Gets and sets the lower 32 bits of the large integer.

Methods

The IEDMLargeInteger interface defines the following methods:

Method

Description

Set

Sets the LargeInteger type value that you want to process.

SetString

Sets the LargeInteger type value that you want to process.

SetOctetString

Sets the LargeInteger type value that you want to process.

GetIADsLargeInteger

Returns a reference to an object of the IADsLargeInteger type.

SetDate

Sets the LargeInteger type value that you want to process.

GetDate

Returns a Generalized-Time type string representing the date and time.

GetString

Returns a string representing the large integer value.

GetOctetString

Returns an array of bytes representing the binary value defined in the current instance of the AelitaEDM.EDMLargeInteger object.

Example Code (VBScript)

The following code snippet illustrates how to use the object AelitaEDM.EDMLargeInteger to get the date that the specified user account will expire, and the date and time that the user password was modified.

'Set DN of the user object in question
strObjDN = "CN=John S. Dow,OU=Sales,DC=mycompany,DC=com"
ss = ""
Set obj = GetObject("EDMS://" + strObjDN)
obj.GetInfoEx Array("pwdLastSet","accountExpires"),0
ss = ss + "User: " + obj.Get("distinguishedName") + vbNewLine + vbNewLine
 'Create an instance of the object AelitaEDM.EDMLargeInteger
Set objLInteger = CreateObject("AelitaEDM.EDMLargeInteger")
 'Process the attribute pwdLastSet
objLInteger.Set obj.Get("pwdLastSet")
ss = ss + "The user password was modified on : " + CStr(objLInteger.GetDate) + vbNewLine
'Process the attribute accountExpires
objLInteger.Set obj.Get("accountExpires")
ss = ss + "The user account expires on : " + CStr(objLInteger.GetDate) + vbNewLine
MsgBox ss
