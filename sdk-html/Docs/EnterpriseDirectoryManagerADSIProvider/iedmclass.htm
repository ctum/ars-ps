IEDMClass

The IEDMClass interface is designed for managing schema class objects that provide class definitions for any ADSI object or Active Roles-specific object.

Properties

The IEDMClass interface has the following properties.

Property

Description

SchemaObjectADsPath

Returns the ADs path to the classSchema object in the schema container.

SchemaObjectCN

Returns the common name of the classSchema object.

Example Code (VBScript)

This code snippet illustrates how to get the ADs path for the classSchema object.

Dim Sch
' Bind to the classSchema object (for Group)
Set Sch=GetObject("EDMS://group,Schema")
MsgBox Sch.SchemaObjectADsPath + vbCRLf + Sch.SchemaObjectCN
' Bind to the classSchema object (for Policy Object)
Set Sch=GetObject("EDMS://edsPolicyObject,Schema")
MsgBox Sch.SchemaObjectADsPath + vbCRLf + Sch.SchemaObjectCN

Remarks

To get a reference to an IEDMClass object, use the following binding string:

"("EDMS://<class name>,Schema")"

For example, to manage the classSchema object for groups, your script must include the following line: 

Set ClObject=GetObject("EDMS://group,Schema")
