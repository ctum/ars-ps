IEDMPOLink

The IEDMAPOLink interface is designed to manage the Policy Object Link objects (the objectClass attribute is edsPolicyObjectLink).

Properties

The IEDMATLink interface has the following properties.

Property

Description

IsBlockingLink

Specifies whether the policy inheritance for a securable object (a directory object to which Policy Object has been applied) is blocked.

LinkDisabled

Specifies whether the Policy Object link is disabled.

LinkFlags

Sets and returns the edsaLinkFlags property of the Policy Object link.

PolicyObjectADsPath

Sets and returns the distinguished name of Policy Object used to enforce the administrative policy.

PolicyObjectGuid

Sets and returns the GUID of Policy Object used to enforce the administrative policy.

PolicyObjectGuidAsString

Sets and returns the GUID (as a string) of Policy Object used to enforce the administrative policy.

SecurableObjectADsPath

Sets and returns the distinguished name of the securable object.

SecurableObjectGuid

Sets and returns the GUID of the securable object.

SecurableObjectGuidAsString

Sets and returns the GUID (as a string) of the securable object.

SecurableObjectName

Returns samAccountName of the securable object.

Example Code (VBScript)

This code snippet illustrates how to enumerate the Policy Object links for a specified Policy Object.

' Set the distinguished name for Policy Object
strDN="cn=MyPO,CN=Administration,CN=Policies,CN=Configuration"
Set objPO = GetObject("EDMS://" & strDN)
ss = "The Policy Object links for this PO:" & vbCRLF & vbCRLF
i=0
 ' Enumerate the Policy Object links for Policy Object
For each l in objPO.Links
        i=i+1
        ss = ss + "Policy Object: " +l.PolicyObjectAdsPath + vbCRLF
        ss = ss + "Securable object: " + l.SecurableObjectAdsPath + vbCRLF
        ss = ss + "Link flags: " + CStr(l.LinkFlags) + vbCRLF
        ss = ss + "Is link disabled? " + Cstr(l.LinkDisabled) + vbCRLF
        ss = ss + "===============================================" + vbCRLF
        
Next    
MsgBox ss

For more examples of the VBScript codes, refer to Managing Policy Objects.
