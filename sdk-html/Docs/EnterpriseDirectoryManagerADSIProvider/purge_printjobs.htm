Purging the Printer Queue

File name: PrinterPurge.vbs

This code snippet illustrates how to purge the queue for the printer in question.

option explicit
Const strPrinterDN = "EDMS://cn=HPLaserJet2100,CN=Printers,cn=mycomputer$,cn=mycompany,CN=Network"
Dim PrinterObject
Set PrinterObject=GetObject(strPrinterDN)
PrinterObject.Put "edsaPurgePrinter", True
PrinterObject.SetInfo

This script removes all the print jobs from the printer HPLaserJet2100 installed on the computer mycomputer in the domain mycompany.com.
